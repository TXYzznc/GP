# 技能图标提示词生成器 - 智能推理版

## 核心设计理念

此技能的核心是**推理能力**而非模板匹配。它通过分析技能的以下维度来生成提示词：

1. **职业属性** - 不同职业有不同的视觉语言
2. **效果机制** - 技能的伤害类型、控制效果、持续性等
3. **视觉关键词** - 从技能名称和描述中提取的视觉元素
4. **技能类型** - 被动/普攻/技能/大招的不同表现方式
5. **背景环境** - 根据效果推理最合适的背景主题

## 职业分析框架

### 职业特征识别

**射手类职业**（后羿、弓箭手等）
- 视觉特征：直线、箭矢、远程、动感
- 色彩倾向：金黄、橙红、棕色
- 效果特征：灼烧、穿透、速度感
- 背景倾向：日落、战场、天空

**法师类职业**（嫦娥、冰法等）
- 视觉特征：圆形、法术、范围、优雅
- 色彩倾向：冰蓝、紫色、白色
- 效果特征：寒霜、冻结、融化、范围伤害
- 背景倾向：月夜、冰雪、星空

**召唤师类职业**（狂怒之心等）
- 视觉特征：力量、能量、多元素、混合
- 色彩倾向：深紫、血红、金黄
- 效果特征：增强、愤怒、召唤、多效果
- 背景倾向：能量场、神秘、混合元素

**岩石/地系职业**（岩石巨人等）
- 视觉特征：厚重、坚固、地面、块状
- 色彩倾向：灰棕、土黄、深灰
- 效果特征：石化、碎石、地震、防御
- 背景倾向：岩石地形、地裂、尘埃

**水系/海洋职业**（海神等）
- 视觉特征：流动、波浪、深邃、优雅
- 色彩倾向：深蓝、青色、白色、紫蓝
- 效果特征：冻结、淹没、波浪、控制
- 背景倾向：海洋、水流、深海、波浪

### 推理方法

当遇到新职业时，分析以下特征：
1. **名称含义** - 从职业名称推断主要元素（岩石→石头、海神→水）
2. **效果描述** - 提取关键词（灼烧、冻结、石化等）
3. **机制特性** - 是否有范围、持续、控制等特性
4. **视觉类比** - 与已知职业的相似性和差异性

## 效果特征分析框架

### 效果类型映射

**伤害类效果**
- 灼烧：火焰、热浪、烟雾 → 橙红、金黄
- 冻结：冰晶、寒气、冻结纹理 → 冰蓝、白色
- 石化：岩石、裂纹、沉重感 → 灰色、棕色
- 淹没：水流、波浪、深邃 → 深蓝、青色

**控制类效果**
- 眩晕：星星、旋转、混乱 → 黄色、白色
- 困禁：荆棘、锁链、束缚 → 棕色、黑色
- 减速：冰冷、粘稠、沉重 → 蓝色、灰色

**增强类效果**
- 力量增强：能量波、光晕、肌肉纹理 → 金黄、红色
- 防御增强：护盾、光墙、坚固感 → 银色、蓝色
- 速度增强：风、闪电、动感 → 白色、黄色

### 推理方法

1. **关键词提取** - 从技能描述中找出效果词（灼烧、冻结等）
2. **视觉转化** - 将效果词转化为视觉元素（灼烧→火焰粒子）
3. **色彩匹配** - 根据效果选择对应的色彩系统
4. **粒子效果** - 推断合适的粒子类型（火焰、冰晶、能量等）

## 技能类型分析框架

### 被动技能
- **特征**：持续效果、环境影响、稳定感
- **构图**：中心对称、稳定、光晕环绕
- **动感**：低动感、持久感
- **背景**：环境融合、氛围烘托

### 普通攻击
- **特征**：基础攻击、简洁清晰、快速反应
- **构图**：清晰主体、动感方向
- **动感**：中等动感、直接感
- **背景**：战场环境、简洁背景

### 技能
- **特征**：中等规模、特殊效果、能量释放
- **构图**：中心爆发、能量扩散
- **动感**：高动感、爆发感
- **背景**：效果环境、能量场

### 大招
- **特征**：大规模、强烈视觉冲击、终极能量
- **构图**：中心爆发、全屏影响
- **动感**：极高动感、灾难感
- **背景**：灾难环境、极端场景

## 背景推理框架

### 背景主题推理

根据以下因素推理背景：

1. **效果类型** → 背景环境
   - 灼烧 → 火焰、热浪、烟雾环境
   - 冻结 → 冰雪、寒气、冻结环境
   - 石化 → 岩石、地裂、尘埃环境
   - 水系 → 海洋、波浪、水流环境

2. **职业特性** → 背景氛围
   - 射手 → 日落、战场、天空
   - 法师 → 月夜、星空、神秘
   - 岩石 → 地形、山脉、地下
   - 海洋 → 海岸、深海、水域

3. **技能类型** → 背景规模
   - 被动 → 环境融合、细节丰富
   - 普攻 → 简洁背景、战场感
   - 技能 → 中等规模、能量场
   - 大招 → 大规模、灾难感

### 背景元素组合

背景 = 主环境 + 效果元素 + 光效 + 氛围

例如：
- 灼烧大招 = 灾难天空 + 火焰 + 热浪 + 爆炸感
- 冻结技能 = 冰风暴 + 冰晶 + 寒气 + 冷静感
- 石化被动 = 岩石地形 + 裂纹 + 尘埃 + 沉重感

## 提示词生成流程

### 第一步：信息收集与分析

收集以下信息：
1. 技能名称 - 提取视觉关键词
2. 技能类型 - 被动/普攻/技能/大招
3. 职业属性 - 识别职业类型
4. 效果描述 - 提取效果关键词
5. 特殊机制 - 范围、持续、控制等

### 第二步：多维度推理

进行以下推理：

**职业推理**
- 识别职业类型（射手/法师/岩石/海洋等）
- 推断职业的视觉语言和色彩系统
- 确定职业的典型背景倾向

**效果推理**
- 提取主要效果（灼烧、冻结、石化等）
- 推断视觉表现（火焰、冰晶、裂纹等）
- 确定色彩系统和粒子效果

**背景推理**
- 根据效果推理主环境
- 根据职业推理氛围
- 根据技能类型推理规模
- 组合背景元素

**构图推理**
- 根据技能类型选择构图方式
- 确定主体位置和动感
- 决定光效方向

### 第三步：提示词组织

组织提示词结构：
```
[分辨率] [风格] [职业特征] [主体描述]
Color palette: [色彩代码]
[光效描述]
Background: [背景描述]
[质感和细节]
Professional game art style, similar to Clash of Gods, [主题]
```

## 关键推理能力

### 1. 职业推理
- 从职业名称推断主要元素
- 识别职业的视觉语言
- 推断职业的色彩系统
- 确定职业的背景倾向

### 2. 效果推理
- 从效果描述提取关键词
- 转化为视觉元素
- 推断粒子效果
- 确定色彩匹配

### 3. 背景推理
- 根据多个因素组合背景
- 推理背景规模和复杂度
- 确定背景元素的层次
- 融合效果和职业特性

### 4. 构图推理
- 根据技能类型选择构图
- 确定动感程度
- 推断光效方向
- 决定视觉焦点

## 最佳实践

### ✅ 应该做

- **深度分析** - 不要只看表面，分析技能的本质特性
- **推理而非匹配** - 根据原理推理，而不是查表
- **灵活组合** - 根据具体情况灵活组合背景元素
- **考虑职业** - 不同职业有不同的视觉语言
- **考虑效果** - 效果决定了视觉表现
- **考虑规模** - 技能类型决定了背景规模
- **保持一致性** - 同一职业的技能应有视觉连贯性

### ❌ 应该避免

- **硬编码模板** - 不要依赖预设模板
- **忽视职业差异** - 不同职业需要不同的表现
- **过度复杂** - 保持简洁但详细
- **色彩冲突** - 背景应衬托主体
- **不合理的背景** - 背景应与效果相符
- **忽视技能类型** - 被动和大招的表现方式完全不同

## 工作流程总结

1. **收集信息** → 技能名称、类型、职业、效果描述
2. **职业推理** → 识别职业特性和视觉语言
3. **效果推理** → 提取效果关键词，转化为视觉元素
4. **背景推理** → 根据多个因素推理背景主题
5. **构图推理** → 根据技能类型确定构图和动感
6. **提示词组织** → 按照结构组织最终提示词
7. **验证一致性** → 确保所有元素相互协调

## 常见问题

**Q: 如何处理完全陌生的职业？**
A: 从职业名称和效果描述推理。例如"冰霜巨人"→ 冰系 + 巨人 = 冰冷 + 厚重，推理出冰蓝色 + 灰棕色的组合。

**Q: 如何处理多效果技能？**
A: 识别主要效果和次要效果。主要效果决定主视觉，次要效果作为辅助元素。

**Q: 背景应该有多复杂？**
A: 根据技能类型。被动技能背景简洁，大招背景复杂。

**Q: 如何确保生成的提示词质量？**
A: 通过深度推理而非模板匹配。每个提示词都应该是根据具体情况推理出来的。

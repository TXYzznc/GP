# MCP 工作区

这是 txt-to-xlsx-converter 技能的统一工作目录。

## 目录说明

所有通过 txt-to-xlsx-converter 技能生成的 TXT 和 XLSX 文件都会保存在这个文件夹中。

## 文件组织建议

你可以在此文件夹内创建子文件夹来组织不同类型的文件：

```
MCP工作区/
├── 配置表/          # 游戏配置表文件
├── 临时文件/        # 临时生成的文件
├── 测试数据/        # 测试用的数据文件
└── ...             # 其他自定义分类
```

## 使用说明

1. 所有 TXT 配置表会首先生成到这个目录
2. 转换后的 XLSX 文件也会输出到这个目录
3. 可以根据需要创建子文件夹进行分类管理
4. 定期清理不需要的临时文件

## 注意事项

- 请勿手动删除此文件夹，它是技能正常工作所必需的
- 如果文件夹被删除，技能会自动重新创建
- 建议定期备份重要的配置表文件

# 预制体图标生成器 - 新功能说明

## 新增功能

### 1. 支持多种输入方式 ✨

现在可以通过两种方式选择要生成图标的预制体：

#### 方式A：文件夹批量生成（原有功能）
- 选择包含预制体的文件夹
- 自动扫描文件夹中的所有预制体
- 适合批量处理大量预制体

#### 方式B：直接拖入预制体（新功能）⭐
- 点击"添加"按钮添加预制体槽位
- 直接将预制体拖入列表
- 支持添加单个或多个预制体
- 可以混合使用两种方式

**使用场景**：
- 只需要为几个特定预制体生成图标
- 预制体分散在不同文件夹中
- 需要精确控制哪些预制体生成图标

### 2. 实时预览功能 ✨

在生成图标前，可以实时预览截图效果！

#### 使用方法
1. 将预制体拖入"预览预制体"字段
2. 自动显示预览图
3. 调整参数（相机角度、光照等）
4. 点击"刷新预览"查看效果
5. 满意后再批量生成

**预览特性**：
- 实时渲染，所见即所得
- 显示透明背景的棋盘格
- 显示预览尺寸信息
- 支持所有参数调整

## 详细使用指南

### 输入设置

#### 预制体文件夹（可选）
```
用途：批量处理文件夹中的所有预制体
操作：拖入文件夹资源
```

#### 预制体列表（可选）
```
用途：精确选择要处理的预制体
操作：
1. 点击"添加"按钮
2. 将预制体拖入列表
3. 点击"×"删除不需要的项
4. 点击"清空"清除所有项
```

**提示**：
- 可以同时使用文件夹和列表
- 重复的预制体会自动去重
- 至少需要选择一种输入方式

---

### 预览功能

#### 预览预制体
```
用途：实时查看截图效果
操作：
1. 拖入要预览的预制体
2. 自动生成预览图
3. 调整参数后点击"刷新预览"
```

#### 预览显示
- **棋盘格背景**：透明背景时显示，方便查看透明度
- **尺寸信息**：显示预览图的实际尺寸
- **自适应大小**：根据窗口宽度自动调整

#### 场景1：快速预览单个预制体
```
1. 拖入预制体到"预览预制体"

3. 调整相机角度、光照等参数
```

#### 场景3：混合使用
```
1. 选择主要的预制体文件夹
2. 添加其他文件夹中的特定预制体到列表
3. 一次性生成所有图标
```

## 常见问题

### Q: 预览图和最终生成的图标一样吗？
A: 预览图使用256x256固定尺寸，最终图标使用"图标尺寸"设置。其他参数（相机、光照）完全一致。

### Q: 可以同时使用文件夹和列表吗？
A: 可以！两种方式的预制体会合并处理，重复的会自动去重。

### Q: 预览功能会影响性能吗？
A: 预览只在以下情况触发：
- 拖入新的预览预制体
- 点击"刷新预览"按钮
- 不会持续占用资源

### Q: 预制体列表有数量限制吗？
A: 没有硬性限制，但建议：
- 少量预制体（<10个）：使用列表
- 大量预制体（>10个）：使用文件夹

### Q: 如何快速添加多个预制体？
A: 
1. 点击多次"添加"按钮创建多个槽位
2. 依次拖入预制体
3. 或者使用文件夹方式批量处理

## 技术细节

### 预览渲染
- 使用 `PreviewRenderUtility` 进行离线渲染
- 固定256x256尺寸（优化性能）
- 与最终生成使用相同的渲染管线

### 内存管理
- 预览纹理在切换时自动清理
- 面板关闭时释放所有资源
- 不会造成内存泄漏

### 性能优化
- 预览按需生成，不持续更新
- 列表滚动优化，支持大量预制体
- 批量生成时使用进度条，可随时取消

## 快捷键和技巧

```
1. 在Project窗口选中多个预制体
2. 点击多次"添加"按钮
3. 依次拖入选中的预制体
```

# 外轮廓描边 - 5分钟快速配置

## 🚀 快速开始（5个步骤）

### ⚡ 步骤 1：创建 Layer（30秒）

```
Edit → Project Settings → Tags and Layers

在 Layers 列表中添加：
┌─────────────────────┐
│ User Layer 6: Enemy │
│ User Layer 7: Ally  │
│ User Layer 8: Interactive │
└─────────────────────┘
```

---

### ⚡ 步骤 2：配置 URP Renderer（1分钟）

```
1. 找到 URP Renderer Asset
   Project 窗口搜索: t:UniversalRendererData
   或者: Edit → Project Settings → Graphics → 点击 URP Asset → 查看 Renderer List

2. 选中 Renderer Asset (例如: URP_Renderer.asset)

3. Inspector 底部点击: Add Renderer Feature

4. 选择: Outline Render Feature

5. 分配 Shader:
   ┌──────────────────────────────────────────┐
   │ Draw Occupied Shader:                    │
   │   → Outline/DrawOccupied                 │
   │                                          │
   │ Outline Detection Shader:                │
   │   → Outline/OutlineDetection             │
   │                                          │
   │ Composite Shader:                        │
   │   → Outline/OutlineComposite             │
   └──────────────────────────────────────────┘
```

**如何找到 Shader：**
- 点击 Shader 字段右侧的圆圈图标
- 在弹出窗口搜索 "Outline"
- 或直接从 Project 窗口拖拽 Shader 文件

---

### ⚡ 步骤 3：创建 OutlineManager（30秒）

```
1. Hierarchy 右键 → Create Empty
2. 重命名为: OutlineManager
3. Add Component → Outline Manager
4. 保持默认配置（已自动创建 3 个 Profile）
```

**默认配置预览：**
```
Profile 1: Enemy (红色轮廓)
Profile 2: Ally (绿色轮廓)
Profile 3: Interactive (黄色轮廓)
```

---

### ⚡ 步骤 4：标记测试物体（30秒）

```
选择场景中的物体 → Inspector 顶部 Layer 下拉菜单

敌人模型    → Layer: Enemy
友军模型    → Layer: Ally
可交互物体  → Layer: Interactive
```

**快捷操作：**
- 选中多个物体可以批量设置 Layer
- 修改父物体 Layer 时会提示是否同时修改子物体

---

### ⚡ 步骤 5：测试运行（10秒）

```
1. 点击 Play 按钮
2. 观察效果:
   ✅ Enemy 物体显示红色轮廓
   ✅ Ally 物体显示绿色轮廓
   ✅ Interactive 物体显示黄色轮廓
```

## ⚠️ 常见错误速查

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 看不到轮廓 | Render Feature 未添加 | 检查 URP Renderer Asset |
| 轮廓全是白色 | Shader 未分配 | 重新分配 3 个 Shader |
| 只有部分物体有轮廓 | Layer 设置错误 | 检查物体 Layer |
| 编译错误 | OutlineProfile.cs 未修复 | 使用修复后的版本 |
| 性能差 | 参数设置过高 | 降低 Resolution Scale |

## 🎮 测试快捷键（可选）

创建测试脚本 `OutlineTest.cs`:

```csharp
using UnityEngine;

public class OutlineTest : MonoBehaviour
{
    void Update()
    {
        var manager = OutlineManager.Instance;
        
        // 按 1/2/3 切换轮廓
        if (Input.GetKeyDown(KeyCode.Alpha1))
            manager.SetProfileEnabled("Enemy", !manager.outlineProfiles[0].enabled);
        
        if (Input.GetKeyDown(KeyCode.Alpha2))
            manager.SetProfileEnabled("Ally", !manager.outlineProfiles[1].enabled);
        
        if (Input.GetKeyDown(KeyCode.Alpha3))
            manager.SetProfileEnabled("Interactive", !manager.outlineProfiles[2].enabled);
    }
}
```

将此脚本添加到 OutlineManager 物体上。

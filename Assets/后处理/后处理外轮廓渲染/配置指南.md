## 🔧 前置准备

### 确认项目使用 URP
1. 打开 `Edit → Project Settings → Graphics`
2. 确认 `Scriptable Render Pipeline Settings` 已分配 URP Asset
3. 通常路径：`Assets/URP.asset` 或 `Assets/Settings/URP-HighQuality.asset`

### 文件清单
确保以下文件存在：

**脚本文件：**
- ✅ `OptimizedMultiLayerOutline.cs`
- ✅ `OutlineManager.cs`
- ✅ `OutlineProfile.cs`
- ✅ `OutlineRenderFeature.cs`
- ✅ `OutlineRenderPass.cs`
- ✅ `PostEffectsBase.cs`
- ✅ `OutlineObjectTracker.cs`
- ✅ `OutlinePerformanceMonitor.cs`

**Shader 文件：**
- ✅ `DrawOccupied.shader`
- ✅ `OutlineDetection.shader`
- ✅ `OutlineComposite.shader`
- ✅ `OutlineShader.shader` (Built-in RP 备用)
- ✅ `BlendShader.shader` (Built-in RP 备用)

## 🏷️ Layer 设置

### 步骤 1：创建 Layer

1. 打开 `Edit → Project Settings → Tags and Layers`
2. 在 `Layers` 部分，找到空闲的 Layer 槽位
3. 创建以下 Layer（建议使用 Layer 6-8）：

```
User Layer 6: Enemy
User Layer 7: Ally  
User Layer 8: Interactive
```

### 步骤 2：验证 Layer

在 Unity 编辑器中，选择任意物体，在 Inspector 顶部的 Layer 下拉菜单中应该能看到：
- Enemy
- Ally
- Interactive

## 🎨 URP Renderer 配置

### 步骤 1：找到 URP Renderer Asset

**方法 A：通过 Graphics Settings**
1. `Edit → Project Settings → Graphics`
2. 点击 `Scriptable Render Pipeline Settings` 中的 URP Asset
3. 在 Inspector 中找到 `Renderer List`
4. 点击 Renderer（通常是 `URP_Renderer` 或 `ForwardRenderer`）

**方法 B：直接搜索**
1. 在 Project 窗口搜索 `t:UniversalRendererData`
2. 通常在 `Assets/` 或 `Assets/Settings/` 目录下

### 步骤 2：添加 Outline Render Feature

1. 选中 URP Renderer Asset（例如 `URP_Renderer.asset`）
2. 在 Inspector 底部点击 `Add Renderer Feature`
3. 选择 `Outline Render Feature`

### 步骤 3：配置 Shader 引用

在新添加的 `Outline Render Feature` 中：

1. **Draw Occupied Shader**
   - 点击圆圈图标或拖拽
   - 选择 `Outline/DrawOccupied`
   - 路径：`Assets/后处理/后处理外轮廓渲染/Shader/DrawOccupied.shader`

2. **Outline Detection Shader**
   - 选择 `Outline/OutlineDetection`
   - 路径：`Assets/后处理/后处理外轮廓渲染/Shader/OutlineDetection.shader`

3. **Composite Shader**
   - 选择 `Outline/OutlineComposite`
   - 路径：`Assets/后处理/后处理外轮廓渲染/Shader/OutlineComposite.shader`

### 步骤 4：保存配置

- 点击 `Ctrl+S` 或 `File → Save Project`
- 确保 Renderer Asset 显示为已修改（有星号标记）

## 🎬 场景配置

### 步骤 1：创建 OutlineManager

1. 在 Hierarchy 中右键 → `Create Empty`
2. 重命名为 `OutlineManager`
3. 添加组件：`Add Component → Outline Manager`

### 步骤 2：配置 OutlineManager

在 `OutlineManager` 组件的 Inspector 中：

**轮廓配置（Outline Profiles）：**

系统会自动创建 3 个默认 Profile，你可以调整：

**Profile 1: Enemy（敌人）**
- Profile Name: `Enemy`
- Target Layer: `Enemy`
- Outline Color: 红色 `(255, 0, 0, 204)`
- Outline Width: `4`
- Iterations: `3`
- Max Distance: `50`
- Use Distance Culling: ✅
- Use Texture Cache: ✅
- Cache Refresh Rate: `0.1`
- Enabled: ✅

**Profile 2: Ally（友军）**
- Profile Name: `Ally`
- Target Layer: `Ally`
- Outline Color: 绿色 `(0, 255, 0, 204)`
- Outline Width: `4`
- Iterations: `3`
- Max Distance: `50`
- Use Distance Culling: ✅
- Use Texture Cache: ✅
- Cache Refresh Rate: `0.1`
- Enabled: ✅

**Profile 3: Interactive（可交互）**
- Profile Name: `Interactive`
- Target Layer: `Interactive`
- Outline Color: 黄色 `(255, 255, 0, 204)`
- Outline Width: `3`
- Iterations: `2`
- Max Distance: `30`
- Use Distance Culling: ✅
- Use Texture Cache: ✅
- Cache Refresh Rate: `0.15`
- Enabled: ✅

**性能设置（Performance Settings）：**
- Resolution Scale: `1.0` (PC) / `0.5` (移动端)
- Max Profiles Per Frame: `3`
- Use Deferred Rendering: ❌ (关闭以实时显示所有轮廓)

**调试（Debug）：**
- Show Debug Info: ❌ (需要时开启查看日志)

### 步骤 3：标记场景物体

选择需要显示轮廓的物体：

**敌人物体：**
1. 选中敌人模型
2. 在 Inspector 顶部，Layer 下拉菜单选择 `Enemy`

**友军物体：**
1. 选中友军模型
2. Layer 选择 `Ally`

**可交互物体：**
1. 选中可交互物体（宝箱、门等）
2. Layer 选择 `Interactive`

**注意：** 如果物体有子物体，会弹出对话框询问是否同时修改子物体的 Layer，根据需要选择。

## ✅ 测试验证

### 快速测试步骤

1. **进入 Play 模式**
   - 点击 Play 按钮

2. **检查轮廓显示**
   - Enemy Layer 的物体应该显示红色轮廓
   - Ally Layer 的物体应该显示绿色轮廓
   - Interactive Layer 的物体应该显示黄色轮廓

3. **测试距离剔除**
   - 移动相机远离物体
   - 超过 Max Distance 后轮廓应该消失

4. **查看调试信息**（可选）
   - 在 OutlineManager 中启用 `Show Debug Info`
   - 查看 Console 窗口的日志输出

### 性能测试

1. 打开 `Window → Analysis → Profiler`
2. 进入 Play 模式
3. 查看 `Rendering` 部分
4. 找到 `OutlineRenderPass` 的耗时

**性能参考：**
- PC (1080p): < 2ms
- 移动端 (720p, 0.5 scale): < 1ms

## 🐛 常见问题

### 问题 1：看不到轮廓效果

**可能原因：**
- ❌ Render Feature 未添加或未启用
- ❌ Shader 未正确分配
- ❌ 物体 Layer 设置错误
- ❌ OutlineManager 未添加到场景

**解决方法：**
1. 检查 URP Renderer Asset 中是否有 `Outline Render Feature`
2. 确认 3 个 Shader 都已分配
3. 确认物体 Layer 与 Profile 的 Target Layer 匹配
4. 确认场景中有 OutlineManager 组件

### 问题 2：轮廓太粗或太细

**解决方法：**
- 调整 `Outline Width` 参数（1-10）
- 调整 `Iterations` 参数（1-5）

### 问题 3：轮廓有锯齿

**解决方法：**
- 增加 `Iterations` 值（2 → 3 → 4）
- 提高 `Resolution Scale`（0.5 → 0.75 → 1.0）
- 启用项目的抗锯齿（Edit → Project Settings → Quality → Anti Aliasing）

### 问题 4：性能问题/帧率下降

**解决方法：**
- 降低 `Resolution Scale`（1.0 → 0.75 → 0.5）
- 减少 `Iterations`（3 → 2）
- 减少 `Outline Width`（4 → 3 → 2）
- 启用 `Use Deferred Rendering`
- 减少 `Max Profiles Per Frame`（3 → 2 → 1）
- 减小 `Max Distance`（50 → 30 → 20）

### 问题 5：编译错误

**错误：RTHandle 相关**
- 确保已应用之前的 `OutlineProfile.cs` 修复
- `cachedTexture` 用于 Built-in RP
- `cachedTextureHandle` 用于 URP

**错误：Shader 编译失败**
- 确认 URP 包已安装（Package Manager → Universal RP）
- 版本建议：URP 12.0+ (Unity 2021.3+)

### 问题 6：某些物体没有轮廓

**可能原因：**
- 物体没有 Renderer 组件（MeshRenderer/SkinnedMeshRenderer）
- 物体被禁用或不在相机视锥内
- 物体距离超过 Max Distance

**解决方法：**
- 确认物体有 Renderer 组件
- 启用 `Show Debug Info` 查看可见物体数量
- 增加 Max Distance 或禁用 Use Distance Culling

## 🎮 运行时控制（代码示例）

如果需要通过代码动态控制轮廓：

```csharp
using UnityEngine;

public class OutlineController : MonoBehaviour
{
    private OutlineManager manager;
    
    void Start()
    {
        manager = OutlineManager.Instance;
    }
    
    void Update()
    {
        // 按键 1：切换敌人轮廓
        if (Input.GetKeyDown(KeyCode.Alpha1))
        {
            manager.SetProfileEnabled("Enemy", !manager.outlineProfiles[0].enabled);
        }
        
        // 按键 2：切换友军轮廓
        if (Input.GetKeyDown(KeyCode.Alpha2))
        {
            manager.SetProfileEnabled("Ally", !manager.outlineProfiles[1].enabled);
        }
        
        // 按键 3：切换可交互物体轮廓
        if (Input.GetKeyDown(KeyCode.Alpha3))
        {
            manager.SetProfileEnabled("Interactive", !manager.outlineProfiles[2].enabled);
        }
        
        // 按键 C：修改敌人轮廓颜色为紫色
        if (Input.GetKeyDown(KeyCode.C))
        {
            manager.SetProfileColor("Enemy", new Color(1f, 0f, 1f, 0.8f));
        }
    }
}
```

# 卡牌效果通用框架设计方案

## 现状分析

当前12个卡牌效果脚本中，存在大量重复代码：
- 都是 Init + Execute 的简单接口
- 都要获取所有棋子、遍历、筛选、应用效果、播放特效
- 只有**目标选择逻辑**和**效果应用逻辑**差异大

### 代码重复度统计

| 效果 | 目标选择 | 效果类型 | 特殊性 |
|------|--------|--------|-------|
| FlameStorm | 敌方全体 | 伤害+Buff | ❌ 通用 |
| GroupHeal | 友方全体 | 治疗 | ❌ 通用 |
| ThunderStrike | 敌方最近 | 伤害 | ❌ 通用 |
| TimeRewind | 友方最近 | 治疗 | ❌ 通用 |
| LifeDrain | 敌方全体→友方最低HP | 伤害+治疗 | ⚠️ 联动 |
| Resurrection | 友方最近死亡 | 治疗+改状态 | ⚠️ 状态依赖 |
| FrostNova | 敌方范围 | Buff | ❌ 通用 |
| ShadowAssault | 敌方最近 | 伤害系数+Buff | ⚠️ 需要扩展 |
| BerserkCardEffect | 友方最近 | Buff | ❌ 通用 |
| WarCryCardEffect | 友方全体 | Buff | ❌ 通用 |
| HolyShieldCardEffect | 友方全体 | Buff | ❌ 通用 |
| ChaosCurseCardEffect | 敌方全体 | Buff | ❌ 通用 |

## 解决方案

### 架构设计

采用**策略模式 + 组合模式**：

```
ICardEffect（接口）
    ├─ GenericCardEffect（通用处理器）
    │   ├─ TargetSelector 策略
    │   │   ├─ AllAlliesSelector
    │   │   ├─ AllEnemiesSelector  
    │   │   ├─ ClosestAllySelector
    │   │   ├─ ClosestEnemySelector
    │   │   ├─ LowestHpAllySelector
    │   │   └─ CustomSelector
    │   │
    │   └─ EffectApplier 策略链
    │       ├─ DamageApplier
    │       ├─ HealApplier
    │       ├─ BuffApplier
    │       ├─ StateChangeApplier
    │       └─ CompositeApplier（组合多个效果）
    │
    └─ 特殊效果脚本（仅保留真正特殊的）
        ├─ LifeDrainCardEffect（伤害转治疗的联动逻辑）
        ├─ ResurrectionCardEffect（状态判断复杂）
        └─ ... 其他特殊情况
```

### 关键类设计

#### 1. 目标选择器接口

```csharp

public interface ICardTargetSelector

    /// </summary>

    /// <param name="cardData">卡牌数据</param>
    /// <param name="targetPosition">释放位置</param>

    List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition);
```

#### 2. 效果应用器接口

```csharp

    /// 对目标应用效果
```

#### 3. 通用效果处理器

```csharp
public class GenericCardEffect : ICardEffect
{
    private CardData m_CardData;
    private ICardTargetSelector m_TargetSelector;
    private List<ICardEffectApplier> m_EffectAppliers;

    public void Init(CardData cardData, ICardTargetSelector selector, params ICardEffectApplier[] appliers)
    {
        m_CardData = cardData;
        m_TargetSelector = selector;
        m_EffectAppliers = new List<ICardEffectApplier>(appliers);
    }

    public void Execute(Vector3 targetPosition)
    {
        if (m_CardData == null || m_TargetSelector == null || m_EffectAppliers.Count == 0)
            return;

        var allChess = BattleChessManager.Instance?.GetAllChessEntities();
        if (allChess == null || allChess.Count == 0) return;

        // 1. 选择目标
        var targets = m_TargetSelector.SelectTargets(allChess, m_CardData, targetPosition);
        if (targets == null || targets.Count == 0) return;

        // 2. 应用所有效果
        foreach (var applier in m_EffectAppliers)
        {
            applier.ApplyEffect(targets, m_CardData);
        }

        // 3. 播放特效
        CardEffectHelper.PlayEffect(m_CardData.TableRow.EffectId, targetPosition);
    }
}
```

```csharp
/// <summary>
/// 选择敌方全体
/// </summary>
public class AllEnemiesSelector : ICardTargetSelector
{
    public List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition)
    {
        var targets = new List<ChessEntity>();
        foreach (var chess in allChess)
        {
            if (chess != null && chess.Camp == (int)CampType.Enemy)
                targets.Add(chess);
        }
        return targets;
    }
}

/// <summary>
/// 选择友方全体
/// </summary>
public class AllAlliesSelector : ICardTargetSelector
{
    public List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition)
    {
        var targets = new List<ChessEntity>();
        foreach (var chess in allChess)

/// <summary>
/// 选择范围内最近的敌人
/// </summary>
public class ClosestEnemySelector : ICardTargetSelector
{
    public List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition)
    {
        float radius = cardData.AreaRadius;
        ChessEntity closest = null;
        float closestDistance = float.MaxValue;

        foreach (var chess in allChess)
        {
            if (chess != null && chess.Camp == (int)CampType.Enemy)
            {
                float distance = Vector3.Distance(chess.transform.position, targetPosition);
                if (distance <= radius && distance < closestDistance)
                {
                    closestDistance = distance;
                    closest = chess;
                }
            }
        }

        return closest != null ? new List<ChessEntity> { closest } : new List<ChessEntity>();
    }
}

/// <summary>
/// 选择HP最低的友方
/// </summary>
public class LowestHpAllySelector : ICardTargetSelector
{
    public List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition)
    {
        ChessEntity lowestHp = null;
        double minHp = double.MaxValue;

        foreach (var chess in allChess)
        {
            if (chess != null && chess.Camp == (int)CampType.Player && chess.Attribute != null)
            {
                if (chess.Attribute.CurrentHp < minHp)
                {
                    minHp = chess.Attribute.CurrentHp;
                    lowestHp = chess;
                }
            }
        }

        return lowestHp != null ? new List<ChessEntity> { lowestHp } : new List<ChessEntity>();
    }
}
```

#### 效果应用器实现

```csharp
/// <summary>
/// 伤害应用器
/// </summary>
public class DamageApplier : ICardEffectApplier
{
    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        float damage = cardData.TableRow.BaseDamage;
        int damageType = cardData.TableRow.DamageType;

        foreach (var target in targets)
        {
            if (target != null)
                CardEffectHelper.DealDamage(target, damage, damageType);
        }
    }
}

/// <summary>
/// 治疗应用器
/// </summary>
public class HealApplier : ICardEffectApplier
{
    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        float healAmount = cardData.GetParam("healAmount", 150f);

        foreach (var target in targets)
        {
            if (target != null)
                CardEffectHelper.HealTarget(target, healAmount);
        }
    }
}

/// <summary>
/// Buff应用器
/// </summary>
public class BuffApplier : ICardEffectApplier
{
    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        foreach (var target in targets)
        {
            if (target != null)
            {
                foreach (int buffId in cardData.HitBuffIds)
                {
                    CardEffectHelper.ApplyBuff(target, buffId);
                }
            }
        }
    }
}

/// <summary>
/// 带伤害系数的伤害应用器
/// 伤害 = BaseDamage + DamageCoeff × CasterAtk
/// </summary>
public class DamageWithCoefficientApplier : ICardEffectApplier
{
    private ChessEntity m_CasterChess; // 可选，施法者棋子，用于读取攻击力

    public DamageWithCoefficientApplier(ChessEntity casterChess = null)
    {
        m_CasterChess = casterChess;
    }

    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        // TODO: 根据实际需求确定如何获取施法者攻击力
        float casterAtk = m_CasterChess?.Attribute?.Attack ?? 0f;
        float damage = cardData.TableRow.BaseDamage + cardData.TableRow.DamageCoeff * casterAtk;
        int damageType = cardData.TableRow.DamageType;

        foreach (var target in targets)
        {
            if (target != null)
                CardEffectHelper.DealDamage(target, damage, damageType);
        }
    }
}

/// <summary>
/// 状态改变应用器
/// </summary>
public class StateChangeApplier : ICardEffectApplier
{
    private ChessState m_TargetState;

    public StateChangeApplier(ChessState targetState)
    {
        m_TargetState = targetState;
    }

    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        foreach (var target in targets)
        {
            if (target != null)
                target.ChangeState(m_TargetState);
        }
    }
}
```

**新增一个 Buff 应用器变体（使用 InstantBuffIds）：**
```csharp
public class InstantBuffApplier : ICardEffectApplier
{
    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        foreach (var target in targets)
        {
            if (target != null)
            {
                foreach (int buffId in cardData.InstantBuffIds)
                {
                    CardEffectHelper.ApplyBuff(target, buffId);
                }
            }
        }
    }
}
```

### 第三阶段：保留特殊效果
需要独立脚本的2个效果（逻辑复杂，无法用框架直接覆盖）：

1. **LifeDrainCardEffect (ID=1006)** ⚠️ 联动逻辑
   - 对敌方全体造成伤害 + 累计总伤害
   - 对友方中HP最低的单位按比例治疗（治疗量 = 总伤害 × 治疗比例）
   - **原因**：伤害和治疗有依赖关系，无法用独立的 Applier 处理
   - **建议**：保留现有脚本或创建 `ChainedEffectApplier` 基类支持

2. **ResurrectionCardEffect (ID=1012)** ⚠️ 状态依赖
   - 复活一个阵亡的友方单位（需要筛选死亡状态）
   - 恢复HP + 改变状态（Dead → Idle）
   - **原因**：目标选择需要判断 `ChessState.Dead`，这是通用选择器无法处理的复杂条件
   - **建议**：创建 `DeadAlliesInRadiusSelector` 以支持此类场景，或保留脚本

## 常见问题

### Q: 什么时候应该用框架，什么时候写独立脚本？
**A: 判断标准：**
- ✅ 框架 - 可以用「目标选择器」+「效果应用器」的组合表达
- ❌ 独立脚本 - 多个效果间有依赖关系、或需要复杂的条件判断、或需要自定义流程

### Q: 新增卡牌效果如何快速实现？
**A:** 
```csharp
// 只需配置一行（无需写脚本）
var effect = new GenericCardEffect();
effect.Init(cardData,
    new AllEnemiesSelector(),
    new DamageApplier(),
    new BuffApplier());
```

### Q: 是否可以进一步支持 Excel 配置化？
**A:** 可以，但超出本方案范围。需要在 CardTable 中添加以下字段：
- `TargetSelectorType` - 目标选择器类型（0=全敌方, 1=全友方, ...）
- `EffectApplierTypes` - 效果应用器类型列表（用逗号分隔）
- 然后由工厂类根据配置动态组装

### Q: 目标选择器中的圆形范围是硬编码的 AreaRadius，后续如何扩展支持其他形状？
**A:** 可扩展为：
```csharp
public interface IAreaFilter
{
    bool IsInArea(ChessEntity chess, Vector3 centerPos, CardData cardData);
}
// 则 SelectTargets 改为：
public List<ChessEntity> SelectTargets(...) 
{
    var targets = new List<ChessEntity>();
    foreach (var chess in allChess)
    {
        if (ShouldInclude(chess) && m_AreaFilter.IsInArea(chess, targetPosition, cardData))
            targets.Add(chess);
    }
}
```

## 风险与缓解

| 风险 | 影响 | 缓解方案 |
|------|------|--------|
| 框架过度设计 | 增加复杂度 | ✅ 当前设计很轻量，仅2个接口 |
| 特殊效果难以扩展 | 未来维护困难 | ✅ 设计了足够的扩展点（可添加新选择器/应用器） |
| 老代码兼容性 | 迁移工作量 | ✅ 旧脚本可逐个删除，无需同时修改 |
| 性能（大战斗） | 遍历效率 | ⚠️ 当前实现：O(n) 遍历。大规模可用空间分割优化 |

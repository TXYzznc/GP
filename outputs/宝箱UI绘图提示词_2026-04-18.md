# 宝箱 UI 绘图提示词（底座+盖子分离）

## 模型结构说明

宝箱由两个独立模型组成：
- **盖子（Lid）**：拱形顶部，从正面看是半圆形，从上面看是圆形或椭圆形
- **底座（Base）**：方形主体，包含把手、装饰脚、侧面细节

每个模型需要单独生成，确保能从多个角度（正面、侧面、上面）正确显示。

**特征**：破旧、可疑、不起眼

## 盖子和底座一致性检查清单

✅ **必须保证一致的要素：**
- 材质类型（木质、金属、宝石等）
- 颜色和色调
- 装饰元素（雕刻、宝石、边框等）
- 魔法光效的颜色和强度
- 整体风格和质感
- 破旧程度或华丽程度

✅ **盖子特有特征：**
- 拱形设计（从正面看半圆形，从上面看圆形）
- 边缘金属边框
- 与底座相接处的铰链痕迹

✅ **底座特有特征：**
- 方形主体
- 正面中央的大开口（内部空间）
- U形把手（上方两侧）
- 四个角落的金属脚
- 侧面细节

## 使用说明

1. **分别生成** - 盖子和底座需要分别输入提示词到 AI 绘图工具
2. **复制提示词** - 选择对应宝箱的盖子和底座提示词
3. **AI 绘图工具** - 使用 Midjourney、Stable Diffusion 等工具生成
4. **多角度检查** - 确保从正面、侧面、上面都能正确显示
5. **调整参数** - 根据需要调整风格、颜色、细节
6. **导出资源** - 生成后分别导出盖子和底座为 PNG
7. **保存位置** - 盖子保存到 `Assets/AAAGame/Resources/UI/TreasureBoxes/Lid/`
8. **保存位置** - 底座保存到 `Assets/AAAGame/Resources/UI/TreasureBoxes/Base/`

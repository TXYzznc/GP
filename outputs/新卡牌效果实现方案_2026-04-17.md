## 架构理解总结

### 核心框架
```
GenericCardEffect（通用效果框架）
├─ ICardTargetSelector（选择器）→ 确定目标
└─ ICardEffectApplier[]（应用器）→ 应用效果

执行流程：
选择目标 → 应用效果 → 播放特效
```

### 现有选择器
| 选择器 | 功能 |
|--------|------|
| `AllEnemiesSelector` | 全体敌方 |
| `AllAlliesSelector` | 全体友方 |
| `ClosestEnemySelector` | 最近敌人（单体） |
| `ClosestAllySelector` | 最近友方（单体） |
| `LowestHpAllySelector` | 最低血量友方（单体） |
| `EnemiesInRadiusSelector` | 范围内敌方 |

### 现有应用器
| 应用器 | 功能 | 参数来源 |
|--------|------|---------|
| `DamageApplier` | 基础伤害 | CardData.TableRow.BaseDamage |
| `HealApplier` | 治疗 | CardData.GetParam("healAmount") |
| `BuffApplier` | 命中应用Buff | CardData.HitBuffIds |
| `InstantBuffApplier` | 立即应用Buff | CardData.InstantBuffIds |
| `DamageWithCoefficientApplier` | 伤害系数伤害 | BaseDamage + Coefficient × 攻击力 |

## 需要新增的组件

### 1. 新增选择器（1个）
```csharp
/// <summary>
/// 选择范围内所有友方
/// </summary>
public class AlliesInRadiusSelector : ICardTargetSelector
{
    public List<ChessEntity> SelectTargets(List<ChessEntity> allChess, CardData cardData, Vector3 targetPosition)
    {
        float radius = cardData.AreaRadius;
        var targets = new List<ChessEntity>();

        foreach (var chess in allChess)
        {
            if (chess != null && chess.Camp == (int)CampType.Player)
            {
                float distance = Vector3.Distance(chess.transform.position, targetPosition);
                if (distance <= radius)
                    targets.Add(chess);
            }
        }

        return targets;
    }
}
```

## 问题和建议

### 关于 ParamsConfig
目前 1014-1025 的新卡牌在 CardTable.txt 中已配置了 ParamsConfig，例如：
- 1016 救赎之光: `{"healAmount":150,"dispelDebuffs":true}`
- 1017 血源强化: `{"bloodSourceBoost":0.3}`

**建议**: 
- 这些参数主要用于Buff系统，不直接被通用框架使用
- 如果Buff需要特殊参数，可通过 CardData.GetParam() 获取

### 关于复杂效果
目前 1026 和 1027 无法用通用框架实现，需要：
- 创建自定义脚本
- 在 CardEffectExecutor 的 default 分支中注册
- 实现完整的 ICardEffect 接口

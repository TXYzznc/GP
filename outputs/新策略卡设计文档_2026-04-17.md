## 设计思路

现有12张卡牌（1001-1012）已覆盖基础机制。新增12张卡牌补充以下维度：

| 维度 | 原有卡牌 | 新增卡牌 |
|------|---------|---------|
| **持续伤害** | 无 | 剧毒之雾、灾厄之雷（灼烧） |
| **降低敌方能力** | 无 | 虚弱诅咒（降攻）、沉默之符（禁技） |
| **反伤/护甲** | 护盾（神圣庇护） | 反伤之盾、圣域防线（护甲强化） |
| **吸血机制** | 生命汲取 | 吸血之刃（物理吸血） |
| **驱散负面** | 无 | 救赎之光（驱散+治疗） |
| **控制目标焦点** | 无 | 嘲讽之声 |
| **伤害转移** | 无 | 伤害转移 |
| **综合强化** | 无 | 生死轮回（治疗+反伤） |

## 12张新卡牌详细设计

### 1. 剧毒之雾 (ID: 1013)
- **灵力消耗**: 30
- **目标类型**: 4（敌方全体）
- **伤害类型**: 0（无伤害，施加Buff）
- **特效**: 30113（释放） / 30207（受击）
- **稀有度**: 2（稀有）
- **效果**:
  - 敌方全体中毒 4 回合
  - 每回合掉血 20（需配置 Buff 5010）
  - 策略用途：持续伤害，消耗战利器

```csharp
// Assets/AAAGame/Scripts/Game/Card/Effect/PoisonMistCardEffect.cs
public class PoisonMistCardEffect : ICardEffect
{
    public override void Execute(CardData cardData, List<ChessEntity> targets)
    {
        foreach (var target in targets)
        {
            target.ApplyBuff(BuffType.Poison, /* 参数 */);
        }
    }
}

// 类似方式创建: WeaknessCardEffect, HolyDefenseLineCardEffect 等
```

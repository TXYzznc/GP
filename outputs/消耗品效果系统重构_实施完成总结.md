### 1. 参照 Buff 系统架构

## 🚀 后续扩展建议

### 新增效果只需 3 步
1. 在 Effects 文件夹创建新类，继承 ItemEffectBase
2. 在 ItemEffectFactory.RegisterAll() 中注册
3. 在配置表添加对应效果配置行

### 示例：新增"获得随机卡牌"效果
```csharp
// 1. 创建 RandomCardEffect.cs
public class RandomCardEffect : ItemEffectBase
{
    public override bool Execute(ItemEffectContext context)
    {
        var playerData = context.GetPlayerData();
        var cardId = GetRandomCard();
        playerData.OwnedStrategyCardIds.Add(cardId);
        return true;
    }
}

// 2. 注册
Register("RandomCard", () => new RandomCardEffect());

// 3. 配置表
{"type":"RandomCard","pool":[1001,1002,1003]}
```

毕设项目策划案
《诸神对决 (Clash of Gods)》游戏策划案
1. 游戏概述 (Game Overview)
•游戏名称： 诸神对决
•游戏类型： 策略卡牌 / 自走棋 / 搜打撤 (Extraction) / Roguelite
•美术风格： 三渲二 (2.5D)，偏向日式二次元或美漫风格，强调技能特效与神话史诗感。
•核心体验： 在危机四伏的地图中探索资源，维持棋子状态，通过策略卡牌辅助自动战斗，最终击败BOSS带回战利品。
•平台： PC / 移动端 (跨平台数据互通)
2. 世界观与剧情 (World & Narrative)
2.1 背景故事
•深渊降临： 世界被不可名状的深渊污染，神话变为现实，但大多是扭曲的“污神”。
•觉醒者： 人类中觉醒灵力者，可通过“媒介卡牌”召唤神话生物。
•圣水技术： 中华神系（灵宝天尊为首）研发出圣水，可净化污神并将其力量封印入卡牌。
•现状： 污神无法被彻底消灭，只能被击败后净化或暂时封印。玩家作为失忆的觉醒者，加入对抗深渊的组织。
2.2 开局流程 (新手引导)
1.苏醒： 玩家在街道醒来，失忆，遭遇黑影袭击。
2.救援： 神秘少女（引导NPC）出现，演示战斗（教程1：棋子放置与自动攻击）。
3.基地检测： 玩家面临核心选择（难度选择）：
￮选项A：调动力量（困难模式/高收益）： 判定为“无限级”。灵力直升10级，解锁第二卡槽，经验获取UP，但精英怪刷新率大幅提升。
￮选项B：压下力量（普通模式/标准体验）： 判定为“E级”。标准成长路线，循序渐进。
4.初次任务： 领取基础包（中华神话），通过传送阵进入第一个地图，进行实战教学（教程2：策略卡使用、探索与撤离）。
3. 核心玩法循环 (Core Loop)
游戏采用 “基地经营 -> 副本探索(搜打撤) -> 资源转化” 的循环模式。
1.准备阶段 (基地)： 配置卡组（棋子+策略），升级科技，购买/补充圣水，选择召唤师职业。
2.探索阶段 (副本)：
￮进入地图，迷雾探索。
￮搜集资源（灵力、材料、金币）。
￮遭遇敌人（巡逻/触发），进行战斗。
￮触发随机事件（强化/交易）。
3.战斗阶段 (局内)： 自走棋战斗 + 实时策略卡释放。
4.结算阶段 (撤离)：
￮成功： 击败地图BOSS -> 携带所有资源撤离。
￮失败： 棋子全灭 -> 损失大部分资源，保留少量保底奖励。
4. 详细系统设计 (System Design)
4.1 战斗系统 (PVE核心)
•棋子机制 (Units)：
￮自动战斗： 棋子放置后自动攻击/释放技能。
￮生命继承： 战斗结束后，棋子当前HP保留至下一场战斗。若棋子死亡，本局无法再出战（除非有复活道具/祭坛）。
￮局内进化： 棋子通过战斗获得经验或拾取道具，可在局内变身（例如：石猴 -> 齐天大圣 -> 斗战胜佛）。
•策略卡 (Spells)：
￮能量消耗： 玩家拥有随时间恢复的“灵力值”。
￮卡牌刷新： 策略卡池每隔一段时间刷新（或手动刷新），如：火球术、群体治疗、攻速Buff、地形改变。
￮作用： 弥补自走棋不可控的缺陷，关键时刻扭转战局。
•召唤师 (Player Class)：
￮狂战： 提升棋子攻速/吸血，自身可下场进行简单普攻。
￮术士： 擅长Debuff（中毒/诅咒），策略卡效果增强。
￮混沌： 随机性强，可能造成双倍伤害也可能误伤，高风险高回报。
￮德鲁伊： 强化回复能力，召唤自然生物充当炮灰，适合持久战。
￮进阶： 全局养成，通过等级+资源+任务解锁（如：狂战 -> 鲜血战神）。
4.2 探索与地图 (Exploration)
•地图机制：
￮战争迷雾： 需要玩家移动开视野。
￮特殊事件雷达： 稀有事件在一定范围内会在小地图标红。
•敌人AI：
￮巡逻型： 有可视范围（扇形/圆形），玩家进入范围 -> 警觉条涨满 -> 追击 -> 接触触发战斗。
￮被动型： 中立生物或宝箱怪，交互后触发战斗。
•事件系统 (Roguelite元素)：
￮宝箱： 获得局内货币、装备、材料。
￮商人： 消耗局内货币购买临时Buff或补给品。
￮强化术士： 牺牲部分HP换取棋子属性提升。
￮圣泉： 恢复棋子生命值（稀有）。
•BOSS战：
￮必须主动触发。
￮BOSS战前需评估棋子状态（血量/Buff），策略性地使用回复道具。
4.3 养成与资源 (Progression)
•全局变量：
￮卡牌背包： 收集的神话角色。
￮科技树： 提升基础属性、解锁商店货架、增加背包格数、提升圣水携带上限。
￮圣水： 关键资源。战斗前携带，用于净化（捕获）特定怪物或作为强力Buff媒介。
•角色体系：
￮中华： 悟空（输出/进阶多）、盘古（坦克/开天辟地）、灵宝天尊（法师/控制）。
￮北欧： 奥丁（召唤/Buff）、洛基（刺客/分身）。
￮日本： 须佐之男（战士）、天照（全屏AOE）。
￮获取方式： 基础包免费，进阶/稀有角色通过“净化污神”（碎片合成）或活动获取。
5. 联机与技术架构 (Technical & Multiplayer)
5.1 数据存储
•混合存档机制：
￮本地存档： 记录单局内的临时状态（防止断网丢失进度）。
￮云存档： 结算时上传，存储全局数据（等级、卡牌、科技）。
￮校验： 登录时校验本地与云端时间戳，防止作弊。
5.2 联机模式 (PVP)
•局域网/模拟匹配：
￮玩家上传自己的防守阵容（镜像）。
￮进攻方挑战防守方的镜像数据（非实时PVP，降低网络延迟要求）。
￮或者局域网内实时对战（TCP/UDP直连）。
6. 界面与交互 (UI/UX)
•主界面 (基地)： 3D场景，点击不同NPC/建筑进入功能（科技树是实验室，商店是自动贩卖机，出击是传送门）。
•战斗界面：
￮上方： 敌人血条、BOSS状态。
￮中间： 3D战场（棋盘）。
￮下方： 策略卡手牌、灵力槽、召唤师技能。
￮左侧： 棋子状态监控（血量/Buff）。
•探索界面： 俯视视角，点击地面移动，敌人有警示圈。
7. 商业化与扩展 (Monetization - 简述)
•不卖数值： 核心付费点在于皮肤（立绘+模型特效）、棋盘样式、召唤师外观。
•DLC/扩展包： 新的神话体系（如希腊神话、埃及神话）作为大型DLC更新。
•月卡/通行证： 提供更多的圣水每日配额、更多的云存档位、外观奖励。

补充：角色设计示例 - 孙悟空 (中华)
•定位： 刺客 / 持续输出
•基础形态：灵明石猴
￮技能：顽石重击 - 对单体造成物理伤害。
•一阶进化：齐天大圣 (需局内等级3)
￮技能：分身术 - 召唤一个继承50%属性的分身。
￮被动：铜头铁臂 - 减免一定物理伤害。
•二阶进化 (分支A)：斗战胜佛 (需特殊道具“佛光”或科技解锁)
￮技能：定海神针 - 全屏直线AOE，带眩晕。
￮特性：对“污神”类敌人伤害加成。
•二阶进化 (分支B)：妖王悟空 (需特殊道具“魔气”)
￮技能：法天象地 - 体型变大，攻击变为范围溅射，但攻速降低。

## 图3.2 系统总体用例图

**参与者：** 玩家

**系统边界：** Clash of Gods 游戏系统

### 探索系统

| 用例 | 说明 |
|------|------|
| 地图探索 | 玩家在地图中自由探索 |
| 触发战斗 | 玩家与敌人发生战斗 |
| 净化敌人 | 玩家净化已击败的敌人 |
| 尝试脱战 | 玩家尝试逃离战斗 |

### 战斗系统

| 用例 | 说明 |
|------|------|
| 部署棋子 | 玩家在战斗前部署棋子 |
| 自动战斗 | 战斗过程自动进行 |
| 使用策略卡 | 玩家在战斗中使用策略卡 |
| 查看战斗结算 | 查看战斗结果 |

| 用例 | 说明 |
|------|------|
| 管理背包 | 玩家管理背包中的物品 |
| 装备物品 | 玩家装备物品到角色 |
| 使用消耗品 | 玩家使用消耗类物品 |
| 查看物品详情 | 查看物品的详细信息 |

### 策略卡系统

| 用例 | 说明 |
|------|------|
| 查看手牌 | 玩家查看当前手牌 |
| 使用卡牌 | 玩家使用策略卡 |
| 查看卡牌效果 | 查看卡牌的效果描述 |
| 获得卡牌 | 玩家获得新卡牌 |

### 系统自动执行

| 用例 | 说明 |
|------|------|
| 敌人AI巡逻 | 敌人在地图中自动巡逻 |
| 视野检测 | 系统检测玩家与敌人的视野范围 |
| 污染值增长 | 地图污染值自动增长 |
| 奖励发放 | 系统自动发放战斗奖励 |
| 战斗结算 | 系统自动进行战斗结算 |
| 场景过渡 | 系统自动处理场景切换 |
| 精英敌人广播 | 精英敌人出现时系统广播 |

### 关系

- **玩家** → 地图探索、触发战斗、尝试脱战、部署棋子、使用策略卡、管理背包、装备物品、查看物品详情、查看手牌、使用卡牌
- 净化敌人 --<<包含>>--> 获得卡牌

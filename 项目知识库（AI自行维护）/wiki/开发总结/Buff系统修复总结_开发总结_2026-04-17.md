## Duration 使用规则（统一标准）

### 规则

| Duration 值 | 行为 | 场景 |
|-----------|------|------|
| **> 0** | Buff 倒计时，时间到后自动移除 | 临时技能效果 |
| **= 0** | Buff 永久存在，不自动移除 | 被动增益、事件驱动的 Buff |
| **< 0** | 同 = 0（永久） | 不建议使用 |

### 使用原则

✅ **正确做法**：
```csharp
// 技能只负责添加 Buff，移除由 BuffManager 自动处理
buffManager.AddBuff(4, caster, attr);  // Duration 由 BuffTable 控制
```

❌ **错误做法**：
```csharp
// 技能代码中手动计时和移除（已修复）
m_DurationRemain -= dt;
if (m_DurationRemain <= 0)
    buffManager.RemoveBuff(4);  // ❌ 不应该这样做
```

## HouyiPassive 特殊情况

**ID 5（日月长弓 Buff）的情况**：
- 这个 Buff 是被动技能增益
- Duration = 0（永久，直到昼夜改变）
- HouyiPassive 中的 RemoveBuff() 调用**保留**，因为它是基于昼夜系统的状态变化，而不是时间

```csharp
// 这是合理的，因为不是基于时间，而是基于系统事件
if (m_IsDay && !m_DayBuffApplied)
    buffManager.AddBuff(5, ...);    // 白天添加
else if (!m_IsDay && m_DayBuffApplied)
    buffManager.RemoveBuff(5);      // 夜晚移除
```

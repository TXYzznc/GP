## 二、系统架构

### ISlotContainer 层次

```
ISlotContainer（接口）
    ↓
SlotContainerBase（抽象基类）
    ├─ InventorySlotContainerImpl（背包）
    ├─ WarehouseSlotContainerImpl（仓库）
    ├─ FastBarSlotContainerImpl（快捷栏）
    └─ EquipSlotContainerImpl（装备栏框架）
```

### 操作规则矩阵

| 源 → 目标 | 允许 | 清空源 | 交换 |
|---------|------|--------|------|
| 背包→背包 | ✓ | MoveItem 处理 | ✓ |
| 背包→仓库 | ✓ | targetEmpty \|\| sameItem | ✗ |
| 背包→快捷栏 | ✓ | targetEmpty \|\| sameItem | ✗ |
| 仓库→背包 | ✓ | targetEmpty \|\| sameItem | ✗ |
| 仓库→仓库 | ✓ | SwapSlots 处理 | ✓ |
| 仓库→快捷栏 | **✗** | — | — |
| 快捷栏→背包 | ✓ | targetEmpty \|\| sameItem | ✗ |
| 快捷栏→仓库 | **✗** | — | — |
| 快捷栏→快捷栏 | ✓ | SwapSlots 处理 | ✓ |

## 三、负重显示功能

**文件**: `InventoryUI.cs`

```csharp
protected void RefreshAll()
{
    RefreshInventory();
    RefreshHotbar();
    RefreshEquipSlots();
    RefreshWeightState();  // 新增
}

private void RefreshWeightState()
{
    float currentWeight = m_InventoryManager.CalculateCurrentWeight();
    varWeightStateText.text = $"负重: {currentWeight:F0}";
}
```

**更新场景**：
- 打开背包（OnOpen → RefreshAll）
- 物品拖拽操作（源/目标格子变化触发事件）
- 整理背包（SortInventory 后）
- 快捷栏/仓库变化

## 二、ColorTable 配置（稀有度颜色系统）

### 配置表设计

**字段**:
| 字段 | 类型 | 说明 |
|------|------|------|
| Id | int | 颜色唯一标识（对应稀有度值） |
| ColorName | string | 颜色名称 |
| ColorHex | string | 十六进制色值（#RRGGBB） |
| Description | string | 颜色描述/用途 |

**数据**: 8 种颜色（绿/橙/红/紫/白/黑/黄/灰）

### RarityColorHelper 重构

**文件**: `Assets/AAAGame/Scripts/UI/Components/RarityColorHelper.cs`

**改进**：从硬编码颜色数组改为从 ColorTable 配置表读取：

```csharp
// 获取数据表
var colorTable = GF.DataTable.GetDataTable<ColorTable>();
var colorRow = colorTable.GetDataRow(rarity);
string colorHex = colorRow.ColorHex;
```

**技术特性**：
- Dictionary 缓存已解析的颜色，避免重复查表
- 支持 `#RRGGBB` 或 `RRGGBB` 格式，转换为 Unity Color (0-1 范围)
- 提供 `ClearCache()` 方法用于配置表重新加载时清空缓存

**注意事项**：
- ColorTable 必须在游戏启动时加载，否则 GetDataTable 返回 null
- 修改 ColorTable 配置后，需调用 `RarityColorHelper.ClearCache()` 清空缓存
- 十六进制色值必须是有效的 6 位十六进制数，否则输出错误日志并返回白色

## 三、背包 UI 交互功能

### 3.7 物品详情面板

**文件**: `Assets/AAAGame/Scripts/UI/Item/ItemDetailPanel.cs`

**功能**：
- 显示物品详细信息：名称、图标、稀有度、重量、数量、耐久度、描述
- 从 ItemTable 读取配置，使用 RarityColorHelper 获取颜色，异步加载图标
- 支持无耐久度物品的自动隐藏

### 3.8 物品上下文菜单

**文件**: `Assets/AAAGame/Scripts/UI/Item/ItemContextMenu.cs`

**功能**：右键点击物品显示上下文菜单，根据物品类型动态显示操作：

| 物品类型（ItemTable.Type） | 可用操作 |
|--------------------------|---------|
| 消耗品（Type=0） | 使用、拆分、丢弃 |
| 装备（Type=1） | 装备、丢弃 |
| 宝物（Type=2） | 丢弃 |
| 任务道具（Type=3） | 无操作 |

**注意**: 拆分功能为 TODO，需后续实现拆分对话框。

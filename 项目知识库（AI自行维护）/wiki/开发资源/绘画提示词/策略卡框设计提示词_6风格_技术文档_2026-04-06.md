## 📋 使用说明

### 提示词应用流程
1. **选择风格** — 根据卡牌主题选择对应风格
2. **复制提示词** — 完整复制上述提示词到 AI 绘图工具
3. **调整参数** — 根据实际需求微调颜色、装饰元素
4. **生成素材** — 使用 Midjourney / Stable Diffusion 等工具生成
5. **Unity 集成** — 导入 PNG，配置 9-slice sprite 进行适配

### 关键参数说明

| 参数 | 说明 | 调整建议 |
|------|------|---------|
| **Layout** | 上下分割比例 | 保持 2/3 + 1/3 不变 |
| **Border** | 边框样式 | 已设为"较细"，可改为"无边框"（删除 border 相关描述） |
| **Material** | 材质感 | 根据风格调整（木质/金属/布料等） |
| **Color palette** | 色彩方案 | 可根据游戏主题调整具体色值 |
| **Decorative elements** | 装饰元素 | 核心风格标识，建议保留 |

### 无边框版本调整
如需完全无边框，删除以下行：
```
Border: thin [color] line with [pattern], no heavy outer border, internal decorative elements only
```

替换为：
```
Border: no visible border, decorative elements integrated into panel design
```

## API 速查

### BuffTestTool（核心）
```csharp
// 应用
ApplyBuffToTarget(buffId, target)           // 单个
ApplyBuffs(buffIds[], target)                // 批量
ApplyBuffs(ids, target, caster)              // 指定施法者

// 移除
RemoveBuffFromTarget(buffId, target)         // 移除指定
ClearAllBuffs(target)                        // 清空全部

// 查询
GetBuff(buffId, target)                      // 获取实例
GetTargetBuffs(target)                       // 获取列表
GetAllAvailableBuffs()                       // 所有可用
```

### BuffPresetManager（预设）
```csharp
AddPreset(name, ids[])                       // 创建预设
ApplyPreset(name, target)                    // 应用预设
LoadPreset(name)                             // 加载预设
GetAllPresets()                              // 获取全部
DeletePreset(name)                           // 删除预设
```

### BuffEffectVerifier（验证）
```csharp
GetBuffDetails(target)                       // Buff 列表
GetTargetAttributes(target)                  // 属性信息
GetControlStates(target)                     // 控制状态
GetBuffAndDebuffCount(target)                // 增益减益数
VerifyBuffApplied(buffId, target)            // 验证应用
GenerateTestReport(target)                   // 生成报告
```

### BuffTestUIManager（UI）
```csharp
SetTarget(target)                            // 设置目标
ShowUI()                                     // 显示面板
HideUI()                                     // 隐藏面板
ToggleUI()                                   // 切换显示
```

## 常见场景代码

### 场景 1：验证单个 Buff
```csharp
BuffTestTool.Instance.ApplyBuffToTarget(10101, target);
var result = BuffEffectVerifier.Instance.VerifyBuffApplied(10101, target);
Debug.Log($"应用成功: {result.IsApplied}, 堆叠: {result.StackCount}");
```

### 场景 2：测试 Buff 组合
```csharp
int[] combo = { 10101, 10102, 10105 };
BuffTestTool.Instance.ApplyBuffs(combo, target);
var details = BuffEffectVerifier.Instance.GetBuffDetails(target);
Debug.Log($"已应用 {details.Count} 个 Buff");
```

### 场景 3：检查属性修改
```csharp
BuffTestTool.Instance.ApplyBuffToTarget(10101, target);
var attr = BuffEffectVerifier.Instance.GetTargetAttributes(target);
Debug.Log($"攻击力: {attr.AtkDamage}");
```

### 场景 4：检查控制状态
```csharp
BuffTestTool.Instance.ApplyBuffToTarget(10104, target); // 眩晕
var controls = BuffEffectVerifier.Instance.GetControlStates(target);
if (controls.Contains("眩晕")) Debug.Log("目标已眩晕");
```

### 场景 5：清空并重置
```csharp
BuffTestTool.Instance.ClearAllBuffs(target);
var report = BuffEffectVerifier.Instance.GenerateTestReport(target);
Debug.Log(report);
```

## 常见问题速解

| 问题 | 原因 | 解决 |
|------|------|------|
| UI 不显示 | Canvas 未创建 | 检查 BuffTestUIManager 是否成功创建 |
| 找不到目标 | 没有 ChessEntity | 确保棋子实体存在并在场景中 |
| Buff 无法应用 | BuffTable 未加载 | 查询 GetAllAvailableBuffs() 确认 ID 正确 |
| 属性不变化 | Buff 未正确生效 | 使用 VerifyBuffApplied() 检查 |
| 预设为空 | 预设初始化失败 | 检查 Buff ID 是否有效 |

## 自动化测试模板

```csharp
public void TestBuffApplying()
{
    var target = FindObjectOfType<ChessEntity>().gameObject;
    
    // 1. 应用
    BuffTestTool.Instance.ApplyBuffToTarget(10101, target);
    
    // 2. 验证
    var result = BuffEffectVerifier.Instance.VerifyBuffApplied(10101, target);
    Assert.IsTrue(result.IsApplied);
    
    // 3. 检查
    var attr = BuffEffectVerifier.Instance.GetTargetAttributes(target);
    Assert.Greater(attr.AtkDamage, 0);
    
    // 4. 清理
    BuffTestTool.Instance.ClearAllBuffs(target);
}
```

## 完整工作流示例

```csharp
// 1. 初始化
var testTool = BuffTestTool.Instance;
var preset = BuffPresetManager.Instance;
var verify = BuffEffectVerifier.Instance;

// 2. 获取目标
var target = FindObjectOfType<ChessEntity>().gameObject;

// 3. 应用预设
preset.ApplyPreset("伤害组合", target);

// 4. 验证
var details = verify.GetBuffDetails(target);
Debug.Log($"已应用 {details.Count} 个 Buff");

// 5. 检查效果
var controls = verify.GetControlStates(target);
var attr = verify.GetTargetAttributes(target);
Debug.Log($"攻击: {attr.AtkDamage}, 控制状态: {controls.Count}");

// 6. 生成报告
var report = verify.GenerateTestReport(target);
Debug.Log(report);

// 7. 清理
testTool.ClearAllBuffs(target);
```

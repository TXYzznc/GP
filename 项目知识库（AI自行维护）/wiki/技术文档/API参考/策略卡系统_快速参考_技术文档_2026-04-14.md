# 策略卡系统 — 快速参考卡

## 🚀 常见操作

### 添加新卡牌
```
1. 编辑 Assets/AAAGame/DataTable/CardTable.txt
   → 末尾添加一行新卡配置 (ID: 1001-1012)
   
2. GameFramework → DataTable → Generate
   → 生成 CardTable.cs
   
3. 重启战斗
   → 随机8张卡中可能出现新卡
```

### 调整卡牌排列外观
```
在 Prefab 或 Inspector 中调整 CardSlotContainer 参数：

FanRadius = 800       // 圆弧半径（越大越展开）
MaxFanAngle = 50      // 展开角度（越大越广）
CircleCenterOffsetY = -100  // 圆心位置（负值下移）

参数实时刷新，无需重启！
```

### 修改动画时长
```csharp
// CardSlotContainer 中的参数
m_EnterDuration = 0.35f      // 进场动画 (秒)
m_CardDealInterval = 0.1f    // 发牌间隔 (秒)
m_RearrangeDuration = 0.25f  // 补位动画 (秒)
```

### 执行卡牌效果
```csharp
// 关键方法链：
拖拽释放 → OnEndDrag() 
  → IsPositionInBattleArea() 
  → ExecuteCardEffect() 
  → CardEffectExecutor.ExecuteCardEffect()
  → 销毁卡牌 UI
```

│  └─ SetData(cardData) → container.AddCardAsync()     │
│  ↓                                                    │

│  └─ 延迟(index * 0.1s) → 进场动画 → 最后卡时重排    │
├─────────────────────────────────────────────────────┤
│ 用户交互（进行中）                                   │
│  ├─ 点击：SelectCard() → 显示详情UI                 │

│  └─ 拖拽：OnBeginDrag → OnDrag → OnEndDrag          │
├─────────────────────────────────────────────────────┤
│ 卡牌使用                                              │
│  ├─ 释放在战场：ExecuteCardEffect()                  │
│  ├─ 创建 Executor → 执行效果                        │
│  ├─ 播放闪光 + 销毁动画                              │
│  ├─ CardManager.RemoveCard()                         │
│  └─ 触发 OnCardRemoved 事件                         │
│  ↓                                                    │
│ CombatUI.OnCardRemoved()                             │
│  ├─ 销毁卡牌 UI GameObject                           │
│  └─ container.RemoveCard() → 重排其他卡             │
├─────────────────────────────────────────────────────┤
│ 战斗结束：OnCombatLeave                              │
│  ↓                                                    │
│ CardManager.Clear()                                  │

│  └─ 杀死所有动画，重置状态                          │

## 📋 CardTable 字段速查

### 基础信息
```
Id          → 卡牌ID (1001-1012)
Name        → 卡牌名称
Desc        → 卡牌描述
IconId      → 图标资源ID (12xx系列)
StoryText   → 故事文本
Rarity      → 稀有度 (1=普通 2=稀有 3=史诗 4=传说)
```

### 消耗与目标
```
SpiritCost  → 灵力消耗 (25-80)
TargetType  → 1=自身 2=友方单体 3=友方全体 4=敌方单体 5=敌方全体 6=全场
CastRange   → 施法范围 (0=无限制)
AreaRadius  → AOE半径 (0=单体)
```

### 伤害系统
```
DamageType  → 0=无 1=物理 2=魔法 3=真实
DamageCoeff → 伤害系数 (0-2.5)
BaseDamage  → 基础固定伤害 (0-200)
```

### 效果系统
```
InstantBuffs    → 释放时Buff (格式: "5001:5,5002:3")
HitBuffs        → 命中时Buff
ParamsConfig    → 参数配置JSON ({"key":value})
EffectId        → 技能特效ID (301xx)
HitEffectId     → 受击特效ID (302xx)
EffectSpawnHeight → 特效高度偏移
```

## ⚠️ 常见陷阱

### ❌ 状态未重置
```csharp
// 错误：多次战斗时卡牌位置错乱
// 原因：上一战斗的状态未清理

// 正确做法
OnCombatLeave() {
    container.ClearState()      // 清理容器
    CardManager.Instance.Clear() // 清理管理器
}
```

### ❌ 动画未清理
```csharp
// 错误：频繁拖拽导致DOTween堆积
// 原因：之前的Tween未Kill

// 正确做法
m_RearrangeTween?.Kill()  // 先杀死旧动画
m_RearrangeTween = sequence  // 再设置新动画
```

### ❌ 异步未等待
```csharp
// 错误：卡牌还在播放进场动画就开始重排
AddCardAsync(card)  // 忘了 await

// 正确做法
await container.AddCardAsync(card)  // 等待完成
```

### ❌ 事件未取消订阅
```csharp
// 错误：战斗结束时事件仍然监听
CardManager.Instance.OnCardRemoved += OnCardRemoved
// OnCombatLeave 时忘记取消订阅

// 正确做法
CardManager.Instance.OnCardRemoved -= OnCardRemoved
```

## 📞 常见问题答案

**Q: 如何改变卡牌数量？**
A: 修改 `CardManager.InitializeForCombat()` 中的 `.Take(8)` 为需要的数量

**Q: 如何调整进场动画速度？**
A: 修改 `m_EnterDuration = 0.35f` 或 `m_CardDealInterval = 0.1f`

**Q: 如何禁用拖拽功能？**
A: 在 `CardSlotItem.OnBeginDrag()` 开头添加 `return`

**Q: 卡牌为什么显示不全？**
A: 检查 `FanRadius` 和 `MaxFanAngle`，或 Prefab 的 Canvas 尺寸

**Q: 如何自定义卡牌效果？**
A: 继承 `ICardEffect` 在 `Effects/` 目录创建新效果脚本

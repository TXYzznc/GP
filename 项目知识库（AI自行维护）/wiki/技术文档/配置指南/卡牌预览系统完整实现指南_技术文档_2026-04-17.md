## 🎯 实现原理

### 1️⃣ 范围预览 (CardRangePreview)

**工作流程：**
```
显示预览(位置, 半径)
    ↓
设置预览数据
    ↓
启动淡入动画 (0 → 1, 0.2s)
    ↓
启动脉冲循环 (1.0 → 1.05 → 1.0, 0.6s)
    ↓
OnDrawGizmos() 每帧绘制
    └─ 黄色半透明圆形
    └─ 中心 = 目标位置
    └─ 半径 = 卡牌配置的 AreaRadius
```

**核心实现：**
```csharp
// 显示预览
CardRangePreview.Instance.ShowPreview(targetPosition, aoeRadius);

// 隐藏预览
CardRangePreview.Instance.HidePreview();

// Gizmos 绘制（自动每帧调用）
private void OnDrawGizmos()
{
    if (!m_IsShowingPreview) return;
    
    Gizmos.color = new Color(1f, 1f, 0f, 0.5f);  // 黄色半透明
    DrawCircle(centerPos, radius * pulseScale, 32);  // 32段圆形
}
```

### 2️⃣ 目标识别 (ICardEffect)

**目标类型分类：**

| TargetType | 值 | 说明 | 识别方式 |
|-----------|-----|------|---------|
| **自身** | 1 | 施放者自己 | - |
| **友方单体** | 2 | 需要用户选择 | 点击选择 |
| **友方全体** | 3 | 所有友方棋子 | 遍历 Camp == Player |
| **敌方单体** | 4 | 需要用户选择 | 点击选择 |
| **敌方全体** | 5 | 所有敌方棋子 | 遍历 Camp == Enemy |
| **全场** | 6 | 所有棋子 | 遍历所有 |

**具体实现示例：**

```csharp
// ✅ 全体目标 (HolyShieldCardEffect)
var allChess = BattleChessManager.Instance.GetAllChessEntities();
foreach (var chess in allChess)
{
    if (chess.Camp == (int)CampType.Player)  // 友方过滤
    {
        // 应用效果
    }
}

// ✅ 范围目标 (FrostNovaCardEffect)
float aoeRadius = m_CardData.TableRow.AreaRadius;
foreach (var chess in allChess)
{
    float distance = Vector3.Distance(chess.transform.position, targetPosition);
    if (distance <= aoeRadius)  // 范围过滤
    {
        // 应用效果
    }
}

// ✅ 单体最近敌人 (ShadowAssaultCardEffect)
ChessEntity closestEnemy = null;
float closestDistance = float.MaxValue;
foreach (var chess in allChess)
{
    if (chess.Camp == (int)CampType.Enemy)
    {
        float distance = Vector3.Distance(chess.transform.position, targetPosition);
        if (distance < closestDistance)
        {
            closestDistance = distance;
            closestEnemy = chess;
        }
    }
}
```

## 💻 核心代码详解

### CardRangePreview 类

```csharp
public class CardRangePreview : MonoBehaviour
{
    // 显示范围预览
    public void ShowPreview(Vector3 position, float radius)
    {
        m_CurrentPreviewPosition = position;
        m_CurrentPreviewRadius = radius;
        m_IsShowingPreview = true;
        
        // 淡入：0 → 1
        m_FadeTween = DOTween.To(() => m_CurrentAlpha, x => m_CurrentAlpha = x, 1f, 0.2f);
        
        // 脉冲：1.0 → 1.05 → 1.0 (循环)
        PlayPulseAnimation();
    }
    
    // 隐藏预览
    public void HidePreview()
    {
        m_FadeTween = DOTween.To(() => m_CurrentAlpha, x => m_CurrentAlpha = x, 0f, 0.2f)
            .OnComplete(() => m_IsShowingPreview = false);
    }
    
    // 绘制圆形
    private void DrawCircle(Vector3 center, float radius, int segments)
    {
        float angleStep = 360f / segments;
        Vector3 lastPoint = center + new Vector3(radius, 0, 0);
        
        for (int i = 1; i <= segments; i++)
        {
            float angle = angleStep * i * Mathf.Deg2Rad;
            Vector3 newPoint = center + new Vector3(
                Mathf.Cos(angle) * radius,
                0,
                Mathf.Sin(angle) * radius
            );
            Gizmos.DrawLine(lastPoint, newPoint);
            lastPoint = newPoint;
        }
    }
}
```

```csharp
public class FrostNovaCardEffect : ICardEffect
{
    public void Execute(Vector3 targetPosition)
    {
        var allChess = BattleChessManager.Instance.GetAllChessEntities();
        float aoeRadius = m_CardData.TableRow.AreaRadius;
        
        // 识别范围内的敌人
        foreach (var chess in allChess)
        {
            if (chess.Camp == (int)CampType.Enemy)
            {
                float distance = Vector3.Distance(chess.transform.position, targetPosition);
                if (distance <= aoeRadius)

                }
            }

        // 显示范围预览（执行时显示）
        CardRangePreview.Instance?.ShowPreview(targetPosition, aoeRadius);
        
        // 播放特效
        CardEffectHelper.PlayEffect(m_CardData.TableRow.EffectId, targetPosition);
    }
}
```

### 方案 B：显示目标高亮（需要扩展）

**创建新的预览层（TargetHighlightPreview）：**

```csharp
public class TargetHighlightPreview : MonoBehaviour
{
    public static TargetHighlightPreview Instance { get; private set; }
    
    private List<ChessEntity> m_HighlightedTargets = new List<ChessEntity>();
    
    /// <summary>
    /// 高亮显示目标棋子
    /// </summary>
    public void HighlightTargets(List<ChessEntity> targets)
    {
        ClearHighlight();
        
        foreach (var target in targets)
        {
            if (target != null)
            {
                // 给棋子加上高亮效果（例如改变颜色或添加 Outline）
                var renderer = target.GetComponent<SpriteRenderer>();
                if (renderer != null)
                {
                    renderer.color = Color.yellow;  // 高亮色
                }
                
                m_HighlightedTargets.Add(target);
            }
        }
    }
    
    /// <summary>
    /// 清除所有高亮
    /// </summary>
    public void ClearHighlight()
    {
        foreach (var target in m_HighlightedTargets)
        {
            if (target != null)
            {
                var renderer = target.GetComponent<SpriteRenderer>();
                if (renderer != null)
                {
                    renderer.color = Color.white;  // 恢复原色
                }
            }
        }
        
        m_HighlightedTargets.Clear();
    }
}
```

**在 CardSlotItem.OnDrag() 中使用：**

```csharp
public void OnDrag(PointerEventData eventData)
{
    // 获取预期的目标列表
    var targets = GetTargetsForCard(m_CardData, dragWorldPos);
    
    // 高亮显示目标
    TargetHighlightPreview.Instance?.HighlightTargets(targets);
}

private List<ChessEntity> GetTargetsForCard(CardData cardData, Vector3 position)
{
    var targets = new List<ChessEntity>();
    var allChess = BattleChessManager.Instance.GetAllChessEntities();
    
    switch (cardData.TableRow.TargetType)
    {
        case 3:  // 友方全体
            targets = allChess.FindAll(c => c.Camp == (int)CampType.Player);
            break;
        case 5:  // 敌方全体
            targets = allChess.FindAll(c => c.Camp == (int)CampType.Enemy);
            break;
        case 6:  // 全场
            targets = allChess;
            break;
        default:
            // 范围 AOE
            if (cardData.TableRow.AreaRadius > 0)
            {
                targets = allChess.FindAll(c =>
                    Vector3.Distance(c.transform.position, position) <= cardData.TableRow.AreaRadius);
            }
            break;
    }
    
    return targets;
}
```

```csharp
    Canvas canvas = GetComponentInParent<Canvas>();
    RectTransform canvasRect = canvas.GetComponent<RectTransform>();
    
    // UI 中心 = 世界 (0, 0)
    float xWorld = (uiPos.x - canvasRect.rect.width / 2) * 0.01f;

Vector3 GetWorldPosFromScreenPoint(Vector3 screenPos)

    if (Physics.Raycast(ray, out hit))
    {
        return hit.point;
    }
    
    return Vector3.zero;
}
```

## 🐛 常见问题

### Q: 预览圆形不显示

**检查清单：**
1. 确认 CardRangePreview 被初始化（Awake 有打印日志）
2. 确认调用了 ShowPreview()
3. 确认 Scene 视图启用了 Gizmos 绘制
4. 确认 m_IsShowingPreview = true

### Q: 拖拽时预览不跟随

**原因：** 未在拖拽过程中更新预览位置

**解决：**
```csharp
public void OnDrag(PointerEventData eventData)
{
    Vector3 worldPos = GetWorldPositionFromScreenPoint(eventData.position);
    
    // 每帧更新预览位置
    if (m_CardData.TableRow.AreaRadius > 0)
    {
        CardRangePreview.Instance?.ShowPreview(worldPos, m_CardData.TableRow.AreaRadius);
    }
}
```

### Q: 预览显示的半径不对

**原因：** AreaRadius 配置错误或坐标系转换有问题

**排查：**
1. 在 Execute() 中打印实际使用的 radius：
```csharp
Debug.Log($"Actual radius: {m_CardData.TableRow.AreaRadius}");
```

2. 对比 Gizmos 显示的圆和实际棋子范围

### Q: 多个预览重叠显示

**解决：**
```csharp
// 在显示新预览前隐藏旧的
CardRangePreview.Instance?.HideAllPreviews();
CardRangePreview.Instance?.ShowPreview(newPos, newRadius);
```

## 用户手动配置部分（需用户完成）

### 步骤 1：打开 Prefab 编辑

1. 在 Project 窗口找到 `Assets/AAAGame/Prefabs/UI/CombatUI.prefab`
2. 双击打开 Prefab 编辑模式（或右键 → Open Prefab）

### 步骤 2：找到卡牌容器

在 Hierarchy 中找到 `CombatUI` → `varCardSlots`（或类似的卡牌容器对象名称）

### 步骤 3：移除 LayoutGroup

1. 选中 `varCardSlots`
2. 在 Inspector 右上角找到已挂载的 LayoutGroup 组件（HorizontalLayoutGroup、VerticalLayoutGroup 等）
3. 点击 ⋮ 菜单 → **Remove Component**
4. 确认移除

### 步骤 4：添加 CardSlotContainer 脚本

1. 仍选中 `varCardSlots`
2. 点击 **Add Component**
3. 搜索 `CardSlotContainer`
4. 添加组件

### 步骤 5：配置参数（可选）

在 Inspector 中可以看到 CardSlotContainer 的以下参数（已有合理默认值）：

```
扇形排列
  Fan Radius: 800
  Max Fan Angle: 50
  Circle Center: (0, 0)
  Circle Center Offset Y: -100

动效时长
  Enter Duration: 0.35
  Rearrange Duration: 0.25
  Rearrange Ease: OutCubic
```

**推荐调整**（根据实际卡牌数量）：
- **2-3 张卡**：`Fan Radius = 600`，`Max Fan Angle = 60-80`
- **4-5 张卡**：`Fan Radius = 800`，`Max Fan Angle = 50`（默认）
- **6+ 张卡**：`Fan Radius = 1000`，`Max Fan Angle = 40`

### 步骤 6：保存 Prefab

1. Ctrl+S 或 File → Save
2. 退出 Prefab 编辑模式

### 步骤 7：验证

1. 在 Scene 中运行游戏进入战斗
2. 观察卡牌是否以扇形排列显示
3. 测试各种动效：
   - 进场：卡牌应从下方滑入
   - 悬停：卡牌应上移放大
   - 补位：移除卡后，其他卡应平滑补位
   - 拖拽：拖拽时其他卡应让开空间

## 可能的问题排查

### 问题 1：卡牌没有出现

**原因**：`varCardSlots` 不是 CardSlotContainer 挂载的位置

**解决**：
- 确认 CardSlotContainer 是否真的添加到了正确的对象上
- 检查 Prefab 是否已保存
- 查看 Console 是否有错误日志

### 问题 2：卡牌排列不对称或重叠

**原因**：圆心偏移或半径参数不合适

**解决**：
- 调整 `Circle Center Offset Y`（默认 -100，可试试 -150 或 -50）
- 调整 `Fan Radius`（增大会让卡牌展开更宽）
- 调整 `Max Fan Angle`（增大会让卡牌角度更大）

### 问题 3：动画播放不顺滑

**原因**：缓动函数或时长不适合

**解决**：
- 增大 `Enter Duration`（例如改为 0.5）
- 增大 `Rearrange Duration`（例如改为 0.35）
- 改变 `Rearrange Ease`（试试 OutQuad 或 OutCubic）

### 问题 4：悬停时卡牌位置错误

**原因**：基准坐标未被正确设置

**解决**：
- 检查 Console 日志中是否有 `SetBaseFanTransform` 的调用
- 确认 CardSlotItem 上是否挂载了正确的脚本
- 尝试重新运行游戏

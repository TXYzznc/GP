# 传送阵系统 - 快速参考

> **最后更新**：2026-04-17

## 🚀 快速启动（3 步）

### 1️⃣ 配置 SceneTable

打开 `AAAGameData/DataTables/SceneTable.xlsx`，确保每个可传送的场景都有记录：

```
Id | SceneName | DisplayName | ConditionType | ConditionParam
1  | MainBase  | 主基地      | 0             | 
2  | World1    | 世界 第一章  | 1             |
```

运行 **GameFramework → DataTable → Generate** 生成代码。

---

### 2️⃣ 搭建 UI Prefab（手动）

#### OverworldUI 预制体结构

```
Canvas (OverworldUI)
├─ CloseBtn (Button) → varCloseBtn
├─ MapContainer (GridLayoutGroup)
│  └─ MapItemUI_1 (Prefab Instance)
│     ├─ MapItemUI.cs
│     ├─ MapItemSceneIdHolder (sceneId=1)
│     ├─ Btn (Button) → varBtn
│     ├─ MapName (Text) → varMapName
│     ├─ Mask (Image) → varMask (未解锁时显示)
│     └─ InteractUI (Panel) → varInteractUI (鼠标悬浮时显示)
```

**重点**：
- 添加 `OverworldUI.cs` 脚本到 Canvas
- 设置 `varMapItemUI` 为 MapContainer
- MapItemUI 需要添加 `MapItemSceneIdHolder` 组件

---

### 3️⃣ 创建传送阵交互物

在场景中创建一个空 GameObject：

```
TeleportGate (GameObject)
├─ TeleportGateInteractable.cs
├─ SphereCollider (isTrigger=true, radius=2)
```

完成！靠近传送阵就能交互。

---

## 📝 脚本映射

| 脚本 | 位置 | 职责 |
|------|------|------|
| `TeleportGateInteractable` | `Game/Interact/Items/` | 交互触发点 |
| `OverworldUI` | `UI/` | UI 容器 |
| `MapItemUI` | `UI/Item/` | 单个场景项 |
| `MapItemSceneIdHolder` | `UI/Item/` | SceneId 配置 |

## 🔧 常用操作

### 打开地图 UI

```csharp
GF.UI.OpenUI(UIViews.OverworldUI);
```

### 检查场景是否解锁

```csharp
var sceneData = GF.DataTable.GetDataTable<SceneTable>().GetDataRow(sceneId);
var playerData = PlayerAccountDataManager.Instance.CurrentSaveData;
bool isUnlocked = sceneData.CheckCondition(playerData);
```

### 获取解锁条件提示

```csharp
string msg = sceneData.GetConditionNotMetMessage();
// 输出：需要达到 5 级才能进入此区域
```

## ❌ 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| `MapItemUI: 场景ID xx 不存在于 SceneTable` | SceneTable 中没有该 ID | 检查 SceneTable 配置，运行 DataTableGenerator |
| `OverworldUI: varMapItemUI 未设置` | 没有在 Inspector 中配置 | 在 OverworldUI 预制体中设置 varMapItemUI |
| `MapItemUI 缺少 MapItemSceneIdHolder 组件` | MapItemUI 预制体上没有 SceneId 配置脚本 | 添加 MapItemSceneIdHolder 组件，设置 SceneId |
| 场景始终显示已解锁/未解锁 | PlayerSaveData 没有正确初始化 | 检查 PlayerAccountDataManager 的初始化 |

# 传送阵系统设计

> **最后更新**：2026-04-17  
> **作者**：AI  
> **状态**：实现完成

## 📋 系统概述

传送阵是一个**可交互物品**，玩家靠近时可以交互，打开**地图界面（OverworldUI）**查看并传送到已解锁的场景。

### 核心特性

1. **场景条件检查** - 基于 SceneTable 的条件系统
2. **动态UI更新** - 根据解锁状态显示不同UI
3. **场景加载** - 点击后自动加载对应场景

---

## 🏗️ 系统架构

### 三层结构

```
TeleportGateInteractable（交互触发）
    ↓ OpenUI
OverworldUI（地图容器）
    ↓ Initialize
MapItemUI × N（单个场景项）
    ↓ CheckCondition
SceneTable + PlayerSaveData（数据源）
```

### 数据流

```
玩家靠近 → InteractionDetector 检测 → OnInteract() 触发
    ↓
TeleportGateInteractable.OnInteract()
    ↓
GF.UI.OpenUI(UIViews.OverworldUI)
    ↓
OverworldUI.OnOpen()
    ↓
初始化所有 MapItemUI 子对象（读取 SceneId）
    ↓
MapItemUI.Initialize(sceneId)
    ↓
检查场景是否解锁 + 刷新UI
    ↓
绘制 Mask（未解锁） / InteractUI（鼠标悬浮）
```

```csharp
protected override void OnOpen(object userData)
{
    InitializeMapItems();  // 初始化所有子对象
}

private void InitializeMapItems()
{
    // 获取所有 MapItemUI 子对象
    MapItemUI[] mapItems = varMapItemUI.GetComponentsInChildren<MapItemUI>();
    
    // 为每个地图项初始化 SceneId
    foreach (var mapItem in mapItems)
    {
        var holder = mapItem.GetComponent<MapItemSceneIdHolder>();
        if (holder != null)
        {
            mapItem.Initialize(holder.SceneId);
        }
    }
}
```

### 3. MapItemUI（单个场景项）

**位置**：`Assets/AAAGame/Scripts/UI/Item/MapItemUI.cs`

**职责**：
- 显示单个场景的信息

- 点击时加载场景

**关键方法**：

#### Initialize(sceneId)
```csharp
public void Initialize(int sceneId)
{
    m_SceneId = sceneId;
    
    // 读取场景数据
    m_SceneData = GF.DataTable.GetDataTable<SceneTable>().GetDataRow(sceneId);
    
    // 读取玩家存档数据
    m_PlayerSaveData = PlayerAccountDataManager.Instance.CurrentSaveData;
    
    RefreshUI();
}
```

#### CheckSceneUnlocked()
```csharp
private bool CheckSceneUnlocked()
{
    // 使用 SceneTableExtensions.CheckCondition() 检查是否满足条件
    return m_SceneData.CheckCondition(m_PlayerSaveData);
}
```

#### RefreshUI()
```csharp
private void RefreshUI()
{
    m_IsUnlocked = CheckSceneUnlocked();
    
    // 更新场景名称
    varMapName.text = m_SceneData.DisplayName;
    
    // 显示/隐藏 Mask（未解锁时显示遮罩）
    varMask.gameObject.SetActive(!m_IsUnlocked);
    
    // 设置按钮交互状态
    varBtn.interactable = m_IsUnlocked;
}
```

#### OnPointerEnter/Exit
```csharp
public void OnPointerEnter()
{
    varInteractUI.gameObject.SetActive(true);
    
    if (!m_IsUnlocked)
    {
        // 可显示解锁条件提示
        string msg = m_SceneData.GetConditionNotMetMessage();
    }
}

public void OnPointerExit()
{
    varInteractUI.gameObject.SetActive(false);
}
```

## 🔗 场景解锁条件系统

### 条件检查流程

所有场景解锁条件由 **SceneTable + PlayerSaveData** 驱动：

```csharp
// SceneTableExtensions.CheckCondition() 实现
public static bool CheckCondition(this SceneTable table, PlayerSaveData saveData)
{
    switch (table.GetConditionTypeEnum())
    {
        case SceneConditionType.None:
            return true;  // 无条件
            
        case SceneConditionType.CompleteTutorial:
            return saveData.HasCompletedTutorial;  // 完成教程
            
        case SceneConditionType.CompleteQuest:
            return CheckQuestConditions(saveData);  // 完成指定任务
            
        case SceneConditionType.ReachLevel:
            return saveData.GlobalLevel >= requiredLevel;  // 达到等级
            
        case SceneConditionType.HasItem:
            return CheckItemConditions(saveData);  // 拥有指定物品
            
        case SceneConditionType.UnlockTech:
            return CheckTechConditions(saveData);  // 解锁指定科技
    }
}
```

### 条件类型

| 类型 | 参数格式 | 示例 |
|------|---------|------|
| `None` | - | 无条件 |
| `CompleteTutorial` | - | 完成新手引导 |
| `CompleteQuest` | 任务ID (逗号分隔) | `1,2,3` |
| `ReachLevel` | 等级值 | `5` |
| `HasItem` | 物品ID (逗号分隔) | `101,102` |
| `UnlockTech` | 科技ID (逗号分隔) | `1,2` |

## 📊 数据依赖

### PlayerSaveData 字段

- `HasCompletedTutorial` - 教程完成状态
- `CompletedQuestIds` - 已完成任务列表
- `GlobalLevel` - 玩家等级
- `InventoryItems` - 背包物品
- `UnlockedTechIds` - 已解锁科技列表

### SceneTable 字段

- `SceneId` - 场景 ID
- `SceneName` - 场景名称（Unity 场景名）
- `DisplayName` - 显示名称（本地化 Key）
- `ConditionType` - 条件类型（0-5）
- `ConditionParam` - 条件参数

## 🐛 常见问题

### Q: 如何更新场景解锁状态？

A: 当玩家完成任务/升级等操作时，系统自动更新 `PlayerSaveData`（如 `CompletedQuestIds`）。下次打开 OverworldUI 时会自动重新检查条件。

### Q: 如何手动刷新 MapItemUI？

A: 调用 `RefreshUI()` 方法：
```csharp
mapItemUI.Initialize(sceneId);  // 重新初始化
```

### Q: 地图项如何显示解锁提示？

A: 在 `MapItemUI.OnPointerEnter()` 中显示 `InteractUI` 面板，并调用：
```csharp
string msg = m_SceneData.GetConditionNotMetMessage();
// 在 UI 上显示提示文本
```

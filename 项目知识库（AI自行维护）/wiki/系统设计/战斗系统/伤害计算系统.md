## 设计目标与核心流程

**设计目标**：覆盖物伤、法伤、真伤、护盾、治疗等所有元素；各类型有合理互制关系；数值清晰易调整。

**实现方案**：代码驱动（所有参数硬编码到 `DamageCalculator.cs`），无需复杂配置表，O(1)静态方法，高性能。

**完整计算流程**：
```
基础伤害计算 (攻击力 × 技能系数)
  ↓
应用增伤/减伤Buff (最多+200%)
  ↓
暴击伤害计算 (1.5倍基础 + 爆伤值)
  ↓
防御减伤（护甲/法抗，两段式递减）
  ↓
护盾抵消 (优先级最高)
  ↓
最小伤害穿透 (至少15%最大血量)
  ↓
最终伤害结果
```

## 公式集合（11套）

### 一、基础伤害（公式1-3）

**公式1：物理伤害**
```
基础伤害 = 攻击力 × 技能系数
应用增伤 = 基础伤害 × (1 + 增伤%)
最终 = 应用增伤 × 暴击倍数 × 护甲减伤系数
```

**公式2：魔法伤害**
```
与物理伤害结构相同，基于魔力属性 + 法抗减伤
护甲和法抗独立计算，不互相影响
```

**公式3：真实伤害**
```
不受护甲/法抗减伤（MinDamagePassthrough=100%）
只能通过护盾和生命值直接抵消
暴击对真伤仍有效
```

### 二、防御减伤（公式4-5）

**公式4：护甲减伤/增伤（两段式 + 负数破甲）**

```
正数护甲（减伤）：
  低阈值段（护甲≤100）：减伤% = 护甲 × 3%
  高阈值段（护甲>100）：减伤% = 3% + (超出量 × 1%)，上限85%

负数护甲（破甲增伤）：
  增伤% = -护甲 × 3% × 50% = -护甲 × 1.5%
  上限：最多+30%增伤（远小于85%减伤）
  护甲下限：-200

关键参数（硬编码）：
  ARMOR_LOW_THRESHOLD = 100
  ARMOR_LOW_RATE = 0.03f (3%)
  ARMOR_HIGH_BASE = 0.03f
  ARMOR_HIGH_RATE = 0.01f
  ARMOR_MAX_REDUCTION = 0.85f (最多85%减伤)
  NEGATIVE_ARMOR_RATE = 0.015f (破甲系数=护甲系数×50%)
  MAX_POSITIVE_ARMOR_INCREASE = 0.30f (最多增伤30%)
  MIN_ARMOR_LIMIT = -200f
```

**公式5：法抗减伤/增伤（参数比护甲更弱）**
```
  MAGICRESIST_LOW_THRESHOLD = 80（阈值更低）
  MAGICRESIST_LOW_RATE = 0.035f (3.5%，初期稍强)
  MAGICRESIST_MAX_REDUCTION = 0.80f（上限80%，低于护甲85%）
  NEGATIVE_MAGICRESIST_RATE = 0.018f（破法系数更强，平衡法抗弱的问题）
  破法最大增伤：35%
设计意图：鼓励魔法核心职业输出
```

### 三、暴击（公式6）

```
暴击率上限：75%
触发暴击：伤害 × (1.5 + 爆伤值)（爆伤值范围0.5~1.5，总倍数2.0~3.0）

示例：
- 20%暴击率，0.5爆伤 → 平均伤害 = 100 × (0.8×1.0 + 0.2×2.0) = 120
- 50%暴击率，1.0爆伤 → 平均伤害 = 100 × (0.5×1.0 + 0.5×2.5) = 175
```

### 四、护盾（公式7-8）

**公式7：护盾吸收**
```
护盾优先级最高，最先抵消伤害
damage <= shield → 伤害完全抵消，扣护盾
damage > shield → 护盾归零，扣血 = damage - shield
```

**公式8：护盾生成**
```
ShieldValue = BaseHealth × ShieldRatio × (1 + ShieldBonus)
过量治疗可转化护盾：overHeal × 0.5 = shieldConverted
```

### 五、增伤效果（公式9）

```
叠加模式：所有增伤Buff效果相加，上限+200%
分类：通用增伤/物理增伤/魔法增伤/真伤增伤
单Buff上限：通用+100%，专属+150%
```

### 六、治疗（公式10）

```
治疗 = 治疗力 × 系数 × (1 + 治疗加成)
单次治疗上限：不超过目标最大血量30%
```

### 七、完整物理伤害流程（公式11）

```
Step1: BaseDamage = Attack × SkillCoefficient
Step2: DamageWithBuff = BaseDamage × (1 + DamageIncreaseRatio)
Step3: if(CriticalHit) → × (1.5 + CritPower)
Step4: × CalculateArmorReduction(TargetArmor)
Step5: 护盾抵消
Step6: FinalDamage = max(afterShield, MaxHealth × 0.15)  // 最小15%穿透
```

## 破甲机制详解

### 设计理念（非对称防御）

```
防御强度 > 破甲强度（防御生态更健康）

+100护甲 → 减伤30% → 伤害系数70%
-100破甲 → 增伤15% → 伤害系数115%
（破甲强度 = 护甲强度的50%）
```

### 破甲来源

```
1. 基础护甲（ChessAttribute.Armor）
2. 护甲加成Buff（正数，如+50护甲）
3. 破甲Buff（负数修改，如-30护甲、-100护甲）
4. 减防Debuff（同为负数护甲修改）
护甲硬下限：-200（最多增伤30%）
```

### 破甲强度等级

```
轻度（-30~-50）：增伤0.45%~0.75%
  → 普通技能附加效果，可长期叠加
中度（-80~-100）：增伤1.2%~1.5%
  → 技能核心效果，3回合
重度（-150~-200）：增伤2.25%~3%（触及上限）
  → 大招/终极技能，2回合
```

### 破甲叠加规则

```
推荐：最强覆盖（而非求和）
  已有-50，再施加-100 → 保留-100（更强覆盖）
  
若需要叠加：
  -50 × 4层 = -200总破甲 → 护甲下限-200 → 最大增伤30%
```

### 实现方式（推荐：Buff系统 + 护甲修改器混合方案）

在ChessAttribute中：
```csharp
private float m_BaseArmor;
private float m_ArmorModifier;  // Buff和破甲导致的修改

public float GetFinalArmor()
{
    return Mathf.Max(m_BaseArmor + m_ArmorModifier, -200f);
}

public void ApplyArmorModifier(float modifier)
{
    m_ArmorModifier += modifier;
    DebugEx.LogModule("ChessAttribute", $"护甲修改: {modifier:+0;-0}, 最终: {GetFinalArmor()}");
}
```

在BuffTable中新增破甲Buff：
```
ID=4001 破甲术：BuffType=2(Debuff), EffectType=5(防御修改), EffectValue=-50, Duration=3, MaxStack=3
ID=4002 致命破甲：EffectValue=-100, Duration=2, MaxStack=1
ID=4003 脆皮诅咒：EffectValue=-30, Duration=5, MaxStack=2
（EffectType=5 为新增类型，需在 BuffEffectType 枚举中添加 ArmorModification=5）
```

## DamageCalculator完整代码

```csharp
using UnityEngine;

namespace AAAGame.Combat
{
    /// <summary>
    /// 伤害计算器：统一处理所有伤害计算逻辑
    /// 位置：Assets/AAAGame/Scripts/Game/Combat/DamageCalculator.cs
    /// </summary>
    public static class DamageCalculator
    {
        // ===== 护甲参数 =====
        private const float ARMOR_LOW_THRESHOLD = 100f;
        private const float ARMOR_LOW_RATE = 0.03f;
        private const float ARMOR_HIGH_BASE = 0.03f;
        private const float ARMOR_HIGH_RATE = 0.01f;
        private const float ARMOR_MAX_REDUCTION = 0.85f;

        // ===== 法抗参数 =====
        private const float MAGICRESIST_LOW_THRESHOLD = 80f;
        private const float MAGICRESIST_LOW_RATE = 0.035f;
        private const float MAGICRESIST_HIGH_BASE = 0.025f;
        private const float MAGICRESIST_HIGH_RATE = 0.008f;
        private const float MAGICRESIST_MAX_REDUCTION = 0.80f;

        // ===== 破甲参数 =====
        private const float NEGATIVE_ARMOR_RATE = 0.015f;
        private const float NEGATIVE_MAGICRESIST_RATE = 0.018f;
        private const float ARMOR_MIN_DAMAGE_PASSTHROUGH = 0.15f;
        private const float MAX_POSITIVE_ARMOR_INCREASE = 0.30f;
        private const float MIN_ARMOR_LIMIT = -200f;

        // ===== 暴击参数 =====
        private const float CRIT_RATE_CAP = 0.75f;
        private const float CRIT_BASE_MULTIPLIER = 1.5f;

        public enum DamageType { Physical = 1, Magic = 2, True = 3 }

        #region 基础伤害

        public static double CalculatePhysicalDamage(double attackPower, double skillCoefficient,
            double damageIncreaseRatio = 0, double critRate = 0, double critPower = 0)
        {
            double baseDamage = attackPower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            return ApplyCritical(damageWithBuff, critRate, critPower);
        }

        public static double CalculateMagicDamage(double magicPower, double skillCoefficient,
            double damageIncreaseRatio = 0, double critRate = 0, double critPower = 0)
        {
            double baseDamage = magicPower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            return ApplyCritical(damageWithBuff, critRate, critPower);
        }

        public static double CalculateTrueDamage(double truePower, double skillCoefficient,
            double damageIncreaseRatio = 0, double critRate = 0, double critPower = 0)
        {
            double baseDamage = truePower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            return ApplyCritical(damageWithBuff, critRate, critPower);
        }

        #endregion

        #region 防御减伤

        /// <summary>护甲减伤系数（支持负数护甲=破甲增伤）</summary>
        public static float CalculateArmorReduction(float armor)
        {
            armor = Mathf.Max(armor, MIN_ARMOR_LIMIT);
            if (armor > 0)
            {
                float reduction = armor <= ARMOR_LOW_THRESHOLD
                    ? armor * ARMOR_LOW_RATE
                    : ARMOR_HIGH_BASE + (armor - ARMOR_LOW_THRESHOLD) * ARMOR_HIGH_RATE;
                return 1f - Mathf.Min(reduction, ARMOR_MAX_REDUCTION);
            }
            else if (armor < 0)
            {
                float damageIncrease = -armor * ARMOR_LOW_RATE * NEGATIVE_ARMOR_RATE;
                return 1f + Mathf.Min(damageIncrease, MAX_POSITIVE_ARMOR_INCREASE);
            }
            return 1f;
        }

        /// <summary>法抗减伤系数（支持负数法抗=破法增伤）</summary>
        public static float CalculateMagicResistReduction(float magicResist)
        {
            magicResist = Mathf.Max(magicResist, MIN_ARMOR_LIMIT);
            if (magicResist > 0)
            {
                float reduction = magicResist <= MAGICRESIST_LOW_THRESHOLD
                    ? magicResist * MAGICRESIST_LOW_RATE
                    : MAGICRESIST_HIGH_BASE + (magicResist - MAGICRESIST_LOW_THRESHOLD) * MAGICRESIST_HIGH_RATE;
                return 1f - Mathf.Min(reduction, MAGICRESIST_MAX_REDUCTION);
            }
            else if (magicResist < 0)
            {
                float damageIncrease = -magicResist * MAGICRESIST_LOW_RATE * NEGATIVE_MAGICRESIST_RATE;
                return 1f + Mathf.Min(damageIncrease, 0.35f);
            }
            return 1f;
        }

        #endregion

        #region 完整伤害计算

        /// <summary>完整物理伤害计算（含护甲减伤+护盾+最小穿透）</summary>
        public static double CalculateFinalPhysicalDamage(
            double attackPower, double skillCoefficient, double targetArmor,
            double maxHealth, double damageIncreaseRatio = 0,
            double critRate = 0, double critPower = 0, double shieldValue = 0)
        {
            double baseDamage = attackPower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            double damageAfterCrit = ApplyCritical(damageWithBuff, critRate, critPower);
            double damageAfterArmor = damageAfterCrit * CalculateArmorReduction((float)targetArmor);
            double damageAfterShield = ApplyShield(damageAfterArmor, shieldValue);
            return Mathf.Max((float)damageAfterShield, (float)(maxHealth * ARMOR_MIN_DAMAGE_PASSTHROUGH));
        }

        /// <summary>完整魔法伤害计算</summary>
        public static double CalculateFinalMagicDamage(
            double magicPower, double skillCoefficient, double targetMagicResist,
            double maxHealth, double damageIncreaseRatio = 0,
            double critRate = 0, double critPower = 0, double shieldValue = 0)
        {
            double baseDamage = magicPower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            double damageAfterCrit = ApplyCritical(damageWithBuff, critRate, critPower);
            double damageAfterMR = damageAfterCrit * CalculateMagicResistReduction((float)targetMagicResist);
            double damageAfterShield = ApplyShield(damageAfterMR, shieldValue);
            return Mathf.Max((float)damageAfterShield, (float)(maxHealth * ARMOR_MIN_DAMAGE_PASSTHROUGH));
        }

        /// <summary>完整真实伤害计算（不受防御影响）</summary>
        public static double CalculateFinalTrueDamage(
            double truePower, double skillCoefficient,
            double maxHealth, double damageIncreaseRatio = 0,
            double critRate = 0, double critPower = 0, double shieldValue = 0)
        {
            double baseDamage = truePower * skillCoefficient;
            double damageWithBuff = baseDamage * (1 + Mathf.Clamp((float)damageIncreaseRatio, 0, 2f));
            double damageAfterCrit = ApplyCritical(damageWithBuff, critRate, critPower);
            return ApplyShield(damageAfterCrit, shieldValue);
        }

        #endregion

        #region 辅助方法

        private static double ApplyCritical(double baseDamage, double critRate, double critPower)
        {
            float clampedCritRate = Mathf.Clamp((float)critRate, 0, CRIT_RATE_CAP);
            if (UnityEngine.Random.Range(0f, 1f) < clampedCritRate)
                return baseDamage * (CRIT_BASE_MULTIPLIER + Mathf.Clamp((float)critPower, 0, 1.5f));
            return baseDamage;
        }

        private static double ApplyShield(double damage, double shieldValue)
        {
            if (shieldValue <= 0) return damage;
            return damage <= shieldValue ? 0 : damage - shieldValue;
        }

        public static void LogDamageCalculation(string attackerName, string targetName,
            double baseDamage, double finalDamage, double targetDefense,
            float defenseReduction, DamageType damageType)
        {
            string defenseType = damageType == DamageType.Physical ? "护甲" : "法抗";
            DebugEx.LogModule("DamageCalculator",
                $"{attackerName} → {targetName}: 基础={baseDamage:F1}, " +
                $"{defenseType}={targetDefense:F0}, " +
                $"减伤={(1 - defenseReduction) * 100:F1}%, " +
                $"最终={finalDamage:F1}");
        }

        /// <summary>单元测试（验证公式正确性）</summary>
        public static void RunTests()
        {
            // 测试：+100护甲减伤应该是3%
            float test1 = CalculateArmorReduction(100);
            Debug.Assert(Mathf.Abs(test1 - 0.97f) < 0.01f, $"测试1失败: {test1}");

            // 测试：-100破甲增伤应该是1.5%
            float test2 = CalculateArmorReduction(-100);
            Debug.Assert(Mathf.Abs(test2 - 1.015f) < 0.005f, $"测试2失败: {test2}");

            DebugEx.LogModule("DamageCalculator", "所有单元测试通过");
        }

        #endregion
    }
}
```

## 平衡性建议

### 职业伤害对标（无Buff无防御）
```
物理核心：攻击力120，技能系数1.0 → 基础120
魔法核心：魔力100，技能系数1.2 → 基础120
真伤职业：真伤力80，技能系数1.5 → 基础120
结论：基础伤害持平，输出稳定性不同
```

### 防御压制建议
```
100护甲 → 减伤30%+（让玩家感受到防御存在）
300护甲 → 减伤约70%（高端防御值）
物理防御强 > 魔法防御（鼓励魔法输出）
```

### 增伤上限
```
单一Buff：通用增伤+50%~+100%，专属增伤+75%~+150%
总叠加上限：+200%（防止伤害爆表）
```

### 治疗平衡
```
单次治疗 = 伤害的60%~80%
持续治疗 = 伤害的40%~60%
```

### 实现要点
1. **暴击判定单次**：用同一个`isCrit`变量控制伤害和特效，避免重复判定
2. **护甲计算精度**：用double计算，最后转float
3. **护盾顺序**：伤害→护甲减伤→护盾抵消（护盾在完整伤害计算后才发挥作用）

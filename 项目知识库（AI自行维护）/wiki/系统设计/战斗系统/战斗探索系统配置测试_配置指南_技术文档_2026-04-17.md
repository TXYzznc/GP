## 🎮 第二步：配置 UITable

### 方式：编辑Excel或直接编辑.txt文件

#### 方式A：编辑 UITable.xlsx（推荐）

```
位置: AAAGameData/DataTables/Core/UITable.xlsx

步骤:
1. 打开文件
2. 找到最后一行（应该是 EndCombatUI = 25）
3. 在下一行添加:

   26  偷袭Debuff选择              2  SneakDebuffSelectionUI          True  2  False
   27  先手Buff通知                2  InitiativeBuffNotificationUI    True  2  False
   28  脱战结果界面                2  EscapeResultUI                  True  2  False

字段说明:
- 第1列：ID (26, 27, 28)
- 第2列：UIName (UI名称)
- 第3列：SortOrder (2 = Dialog组中的排序)
- 第4列：UIPrefab (预制体文件名)
- 第5列：PauseCoveredUI (True = 覆盖时暂停下层UI)
- 第6列：UIGroupId (2 = Dialog对话框组)
- 第7列：EscapeClose (False = 不支持ESC关闭)

4. 保存文件
```

#### 方式B：直接编辑 UITable.txt（高级）

```
位置: Assets/AAAGame/DataTable/Core/UITable.txt

打开文件后，在末尾添加三行:

26	偷袭Debuff选择	2	SneakDebuffSelectionUI	True	2	False
27	先手Buff通知	2	InitiativeBuffNotificationUI	True	2	False
28	脱战结果界面	2	EscapeResultUI	True	2	False

注意：用Tab分隔（不是空格）
保存文件
```

### 常见错误信息及解决

```
错误: "BuffTable未加载"
→ 检查BuffTable.xlsx是否已保存
→ 运行DataTable Generator

错误: "未找到合适的先手Buff（IsEnemy=False）"
→ BuffTable中没有IsInitiativeBuff=true且BuffOwner=0的Buff
→ 添加玩家先手Buff配置

错误: "获取偷袭Debuff池: 0个"
→ BuffTable中没有IsSneakDebuff=true的Buff
→ 添加偷袭Debuff配置

错误: "NullReferenceException"
→ BuffTable.cs可能有编译错误
→ 清理Assets/AAAGame/Scripts/DataTable文件夹，重新生成
```

## 🛠️ 调试技巧

### 快速调试Buff配置

```csharp
// 在任何脚本中添加，然后在菜单执行

#if UNITY_EDITOR
[UnityEditor.MenuItem("Debug/Print All Buffs")]
private static void PrintAllBuffs()
{
    var buffTable = GF.DataTable.GetDataTable<BuffTable>();
    if (buffTable == null)
    {
        Debug.Log("BuffTable not loaded");
        return;
    }

    var allBuffs = buffTable.GetAllDataRows();
    Debug.Log($"=== Total Buffs: {allBuffs.Count} ===");

    foreach (var buff in allBuffs)
    {
        Debug.Log($"ID:{buff.Id} | Name:{buff.Name} | " +
            $"Initiative:{buff.IsInitiativeBuff} | " +
            $"Sneak:{buff.IsSneakDebuff} | " +
            $"Owner:{buff.BuffOwner}");
    }
}
#endif
```

## 📞 常见问题速查

| 问题 | 症状 | 解决方案 |
|-----|------|---------|
| BuffTable未加载 | "BuffTable未加载" 日志 | 重新运行DataTable Generator |
| Buff ID返回0 | GetRandomInitiativeBuff()返回0 | 检查BuffTable中是否有符合条件的Buff |
| UI变量为null | "NullReferenceException" | 检查UI预制体中SerializeField是否已绑定 |
| UITable配置缺失 | UIViews中没有新的枚举值 | 添加UITable配置行，重新生成 |
| 脱战失败 | EscapeResultUI无法打开 | 检查EscapeRuleTable配置 |

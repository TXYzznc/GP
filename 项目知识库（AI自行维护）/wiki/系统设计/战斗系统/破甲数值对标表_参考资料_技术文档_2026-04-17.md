### 实战示例

#### 场景1：基础对决

```
攻击方基础伤害: 100
目标护甲配置: 100

无破甲情况：
  伤害 = 100 × 0.97 = 97

应用 -100 破甲后：
  目标护甲 = 100 - 100 = 0
  伤害 = 100 × 1.00 = 100

伤害提升：3点（3%）
```

#### 场景2：高防敌人被破甲

```
攻击方基础伤害: 100
敌方护甲: 200

无破甲：
  伤害 = 100 × 0.94 = 94（减伤6%）

单个 -100 破甲：
  目标护甲 = 200 - 100 = 100
  伤害 = 100 × 0.97 = 97（提升3点）

两个 -100 破甲（最强覆盖）：
  目标护甲 = 200 - 100 = 100（同上，取最强）
  伤害 = 100 × 0.97 = 97

多重 -50 破甲（叠加4层）：
  目标护甲 = 200 - 200 = 0（达到下限）
  伤害 = 100 × 1.00 = 100（最大增伤）
```

#### 场景3：极限破甲

```
目标护甲: 150

应用 -200 破甲（达到下限）：
  目标护甲 = 150 - 200 = -50（限制到 -200 之前）
  
实际计算：
  伤害 = 100 × (1 + 50 × 1.5%) = 100 × 1.075 = 107.5

继续施加更多破甲：
  目标护甲已在 -50，无法更低（考虑叠加机制）
  
最终增伤上限：
  当护甲 ≤ -200 时，增伤 ≥ 3%，触发上限
```

## 不同破甲强度的应用场景

### 轻度破甲（-30 ~ -50）

```
增伤: 0.45% ~ 0.75%

推荐场景：
✓ 普通技能附加效果（每回合都能触发）
✓ 初级破甲Buff
✓ 合作效果（多个弱小破甲叠加）

配置示例：
- 普攻附带 -30 破甲
- 技能叠加 -20 破甲（可5层）
```

增伤: 1.2% ~ 1.5%

## 破甲与护甲加成的互动

### 场景：同时有护甲加成和破甲

```
目标基础护甲: 100

情况1：先加成，后破甲
  Step 1: 100 + 50 (护甲加成) = 150
  Step 2: 150 - 100 (破甲) = 50
  结果: 减伤1.5% → 伤害系数98.5%

情况2：先破甲，后加成
  Step 1: 100 - 100 (破甲) = 0
  Step 2: 0 + 50 (护甲加成) = 50
  结果: 减伤1.5% → 伤害系数98.5%

结论: 顺序不影响结果（可交换）
```

### 叠加破甲的最优策略

```
推荐：最强覆盖（而非求和）

目标护甲: 100
已应用: -50 破甲
现在应用: -100 破甲

方式1（求和）:
  最终破甲 = -50 + (-100) = -150
  目标护甲 = 100 - 150 = -50
  增伤 = 0.75%

方式2（最强覆盖）:
  保留较强的 -100 破甲
  目标护甲 = 100 - 100 = 0
  增伤 = 0%

战斗中：
  第一回合应用 -50 破甲（弱化版）
  第二回合应用 -150 破甲（强化版，覆盖弱化版）
  结果：更灵活的战术选择
```

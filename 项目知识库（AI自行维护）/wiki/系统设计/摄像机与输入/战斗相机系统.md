# 战斗相机系统

合并自：战斗场景相机位置问题（问题诊断）+ 战斗相机优化（设计方案）+ 相机缩放统一性（Bug修复）

## 一、战斗相机缩放统一性修复

**问题**：战斗准备阶段和战斗阶段，滚动鼠标滚轮时相机位置会移动，不符合设计要求（战斗模式下相机应完全固定）。

**修复**（`ThirdPersonCamera.cs`）：

```csharp
private void HandleZoomInput()
{
    // 战斗模式下完全忽略缩放输入
    if (viewMode == CameraViewMode.Combat)
        return;
    // ... 其他逻辑保持不变
}

private void HandleCombatMode()
{
    // 战斗模式：只处理动态FOV（平滑过渡），不处理缩放
    HandleDynamicFOV();
}
```

同时移除未使用的字段 `combatMinDistance`、`combatMaxDistance` 和方法 `UpdateCombatCameraZoom()`，修复 `LateUpdate()` 中重复调用 `HandleDynamicFOV()` 的问题。

## 二、战斗场景相机位置和玩家位置问题

### 问题1：相机没有移动到 CameraAnchor

**原因**：`SpawnBattleArena()` 是异步方法，但 `MoveCameraToCameraAnchor()` 的调用没有放在 `await` 之后，场地还未加载完成就执行了相机移动。

**修复**：将相机移动调用放在 `await DissolveTransitionManager.Instance.TransitionToBattle(arena)` 之后。

### 问题2：玩家位置恢复失败

**错误日志**：`[Warning] [EnemyEntityManager] 无法恢复玩家位置：玩家角色不存在`

**原因**：时序问题——`ExplorationState.OnEnter()` 在玩家角色生成之前被调用。

```
1. ExplorationState.OnEnter() ← 玩家还不存在
2. 尝试 RestorePlayerPosition() ← 失败
3. PlayerCharacterManager 开始生成玩家角色
```

**修复**：
1. 移除 `ExplorationState.OnEnter()` 中的位置恢复代码
2. 在 `InGameState.OnCombatEnd()` 中战斗结束后、切换探索状态前恢复玩家位置

## 三、战斗相机优化（完整设计方案）

### 功能需求

1. **相机平滑移动**：进入战斗准备阶段，相机 0.8 秒平滑移动到 CameraAnchor，与场景加载并行
2. **战斗视角模式**：CombatViewMode——位置固定，不跟随玩家，不支持鼠标旋转，只处理动态 FOV
3. **视角锁定/恢复**：进入战斗时缓存当前视角模式并锁定，离开战斗时解锁并恢复

### ThirdPersonCamera 修改

```csharp
// 新增字段
private bool m_IsViewModeLocked;
private CameraViewMode m_CachedViewMode;
private bool m_IsSmoothMoving;
private Vector3 m_TargetPosition;
private Quaternion m_TargetRotation;
private float m_SmoothMoveTime;
private float m_SmoothMoveElapsed;

// 视角锁定 API
public void SetViewModeLocked(bool locked)
public void CacheCurrentViewMode()
public void RestoreCachedViewMode()

// SetViewMode 中添加锁定检查
public void SetViewMode(CameraViewMode mode)
{
    if (m_IsViewModeLocked && mode != viewMode) return; // 锁定时忽略
    // ...
}

// 平滑移动 API
public void SmoothMoveTo(Vector3 targetPosition, Quaternion targetRotation, float duration)
```

### CombatPreparationState 修改

```csharp
protected override void OnEnter(...)
{
    // 1. 缓存视角模式并切换到战斗视角（锁定）
    SetupCombatCamera();

    // 2. 开始平滑移动相机（与场景加载并行）
    // 3. 生成战斗场地（异步）
}

private void SetupCombatCamera()
{
    var cam = CameraRegistry.ThirdPersonCamera;
    cam.CacheCurrentViewMode();
    cam.SetViewMode(CameraViewMode.Combat);
    cam.SetViewModeLocked(true);
}

protected override void OnLeave(...)
{
    // 离开战斗准备阶段时解锁（但不恢复，在探索状态恢复）
    CameraRegistry.ThirdPersonCamera?.SetViewModeLocked(false);
}
```

### ExplorationState 修改

```csharp
protected override void OnEnter(...)
{
    // 恢复战斗前的视角模式
    CameraRegistry.ThirdPersonCamera?.RestoreCachedViewMode();
    // ...
}
```

### 相机移动实现

```csharp
private void StartCameraSmoothMove()
{
    var arena = BattleArenaManager.Instance?.CurrentArena;
    Transform cameraAnchor = arena?.transform.Find("CameraAnchor");
    var cam = CameraRegistry.ThirdPersonCamera;
    if (cameraAnchor != null && cam != null)
        cam.SmoothMoveTo(cameraAnchor.position, cameraAnchor.rotation, 0.8f);
}
```

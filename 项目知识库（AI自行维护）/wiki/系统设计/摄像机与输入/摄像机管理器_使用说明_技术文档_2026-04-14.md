## 🎯 核心方法

### 1. ActivateCamera(GameObject, int)

激活指定游戏对象上的摄像机（默认第一个）

```csharp
/// <summary>
/// 激活指定游戏对象上的摄像机
/// </summary>
/// <param name="target">目标游戏对象</param>
/// <param name="cameraIndex">摄像机索引（默认为0）</param>
/// <returns>是否成功激活</returns>
public bool ActivateCamera(GameObject target, int cameraIndex = 0)
```

**使用示例：**
```csharp
// 激活玩家身上的第一个摄像机
CameraManager.Instance.ActivateCamera(playerGameObject);

// 激活玩家身上的第二个摄像机
CameraManager.Instance.ActivateCamera(playerGameObject, 1);
```

---

### 2. ActivateCamera(Camera)

直接激活指定的摄像机组件

```csharp
/// <summary>
/// 直接激活指定的摄像机组件
/// </summary>
/// <param name="camera">要激活的摄像机</param>
/// <returns>是否成功激活</returns>
public bool ActivateCamera(Camera camera)
```

**使用示例：**
```csharp
Camera myCamera = GameObject.Find("MainCamera").GetComponent<Camera>();
CameraManager.Instance.ActivateCamera(myCamera);
```

---

### 3. DeactivateCurrentCamera()

禁用当前激活的摄像机

```csharp
/// <summary>
/// 禁用当前激活的摄像机
/// </summary>
public void DeactivateCurrentCamera()
```

**使用示例：**
```csharp
// 禁用当前摄像机（例如进入过场动画）
CameraManager.Instance.DeactivateCurrentCamera();
```

---

### 4. GetCameras(GameObject)

获取指定对象上的所有摄像机

```csharp
/// <summary>
/// 获取指定对象上的所有摄像机
/// </summary>
/// <param name="target">目标对象</param>
/// <returns>摄像机数组</returns>
public Camera[] GetCameras(GameObject target)
```

**使用示例：**
```csharp
Camera[] cameras = CameraManager.Instance.GetCameras(playerGameObject);
Debug.Log($"玩家身上有 {cameras.Length} 个摄像机");
```

```

   └─> PlayerCharacterManager.SpawnPlayerCharacterFromSave()
       └─> SpawnCharacter()
           ├─> 加载预制体
           ├─> 实例化角色
```

## ⚙️ 配置要求

### 角色预制体配置

确保你的召唤师预制体上有摄像机组件：

```
召唤师预制体
├── Model (模型)
├── PlayerController (脚本)
└── Camera (摄像机) ← 必须有这个
    ├── Position: (0, 2, -5)  // 示例位置
    ├── Rotation: (10, 0, 0)   // 示例旋转
    └── Field of View: 60
```

**推荐配置：**
- **位置**：相对于角色的偏移（例如：后上方）
- **旋转**：稍微向下看角色
- **Tag**：MainCamera（可选）
- **Audio Listener**：确保只有一个激活的 Audio Listener

## 🐛 常见问题

### Q1: 角色生成后摄像机没有激活？

**检查清单：**
1. ✅ 角色预制体上是否有 Camera 组件？
2. ✅ Camera 组件是否在子对象上？（会自动查找）
3. ✅ 查看 Console 日志，是否有警告信息？

**调试代码：**
```csharp
Camera[] cameras = CameraManager.Instance.GetCameras(playerGameObject);
Debug.Log($"找到 {cameras.Length} 个摄像机");
foreach (var cam in cameras)
{
    Debug.Log($"摄像机: {cam.name}, 启用: {cam.enabled}");
}
```

---

### Q2: 有多个 Audio Listener 警告？

**原因：** 场景中有多个激活的 Audio Listener 组件。

**解决方案：**
```csharp
// 在激活摄像机后，禁用其他 Audio Listener
AudioListener[] listeners = FindObjectsOfType<AudioListener>();
foreach (var listener in listeners)
{
    if (listener.gameObject != CameraManager.Instance.ActiveCamera.gameObject)
    {
        listener.enabled = false;
    }
}
```

或者在 `CameraManager` 中添加自动处理：

```csharp
public bool ActivateCamera(GameObject target, int cameraIndex = 0)
{
    // ... 现有代码 ...
    
    // 自动处理 Audio Listener
    AudioListener[] allListeners = FindObjectsOfType<AudioListener>();
    foreach (var listener in allListeners)
    {
        listener.enabled = (listener.gameObject == targetCamera.gameObject);
    }
    
    return true;
}
```

---

### Q3: 如何平滑切换摄像机？

**实现思路：**

```csharp
public class CameraManager : MonoBehaviour
{
    /// <summary>
    /// 平滑切换到目标摄像机
    /// </summary>
    public IEnumerator SmoothSwitchCamera(Camera targetCamera, float duration = 1f)
    {
        if (ActiveCamera == null || targetCamera == null)
        {
            ActivateCamera(targetCamera);
            yield break;
        }
        
        Camera oldCamera = ActiveCamera;
        Vector3 startPos = oldCamera.transform.position;
        Quaternion startRot = oldCamera.transform.rotation;
        
        Vector3 endPos = targetCamera.transform.position;
        Quaternion endRot = targetCamera.transform.rotation;
        
        float elapsed = 0f;
        while (elapsed < duration)
        {
            elapsed += Time.deltaTime;
            float t = elapsed / duration;
            
            oldCamera.transform.position = Vector3.Lerp(startPos, endPos, t);
            oldCamera.transform.rotation = Quaternion.Slerp(startRot, endRot, t);
            
            yield return null;
        }
        
        // 切换到目标摄像机
        ActivateCamera(targetCamera);
    }
}
```

**使用：**
```csharp
StartCoroutine(CameraManager.Instance.SmoothSwitchCamera(newCamera, 2f));
```

## 📊 完整示例

### 完整的玩家摄像机设置

```csharp
public class PlayerCameraSetup : MonoBehaviour
{
    private void Start()
    {
        // 等待角色生成
        StartCoroutine(WaitForPlayerAndSetupCamera());
    }
    
    private IEnumerator WaitForPlayerAndSetupCamera()
    {
        // 等待角色生成
        while (!PlayerCharacterManager.Instance.IsCharacterSpawned)
        {
            yield return null;
        }
        
        GameObject player = PlayerCharacterManager.Instance.CurrentPlayerCharacter;
        
        // 激活摄像机
        bool success = CameraManager.Instance.ActivateCamera(player);
        
        if (success)
        {
            Debug.Log("玩家摄像机设置完成");
        }
        else
        {
            Debug.LogError("玩家摄像机设置失败");
        }
    }
}
```

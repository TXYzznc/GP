# 角色控制与摄像机

合并自：角色控制与摄像机_使用说明（固定视角版）+ 第三人称控制器_使用说明（自由轨道版）

---

> ⚠️ **矛盾标注**：两份文档描述了两种不同的摄像机方案。
> - **固定视角版**（`ThirdPersonCameraRig`）：鼠标控制角色旋转，摄像机跟随旋转，适合 ARPG/MOBA
> - **自由轨道版**（`ThirdPersonCamera`）：鼠标控制摄像机轨道，角色自动转向移动方向，适合动作冒险/开放世界
> 
> 固定视角版标注 `ThirdPersonCameraRig` 已被标记为 `[Obsolete]`，当前项目**实际使用 `ThirdPersonCamera`（自由轨道版）**。固定视角部分保留仅供参考。

## 当前使用方案：ThirdPersonCamera（自由轨道版）

### 核心特性

- 摄像机独立轨道控制（鼠标控制摄像机旋转，角色不旋转）
- 移动方向基于摄像机视角
- 角色模型平滑转向移动方向
- 智能防遮挡系统（SphereCast）
- 滚轮缩放
- 动态 FOV（快跑时增加视野）

### 操作说明

| 操作 | 效果 |
|------|------|
| 鼠标移动 | 控制摄像机旋转（角色不旋转） |
| W/A/S/D | 基于摄像机方向移动，角色自动转向 |
| 双击 W | 进入慢跑状态 |
| Shift（慢跑时） | 切换到快跑状态 |
| 滚轮 | 缩放摄像机距离 |
| Tab | 切换鼠标锁定状态 |

### ThirdPersonCamera 主要参数

```
轨道控制:
- orbitSensitivityX / Y: 水平/垂直旋转灵敏度（默认 3 / 2）
- orbitDamping: 旋转阻尼时间（默认 0.1）
- minPitch / maxPitch: 俯仰角限制（-40 ~ 80）

跟随设置:
- followSmoothTime: 跟随平滑时间（默认 0.1）
- targetOffset: 目标偏移（默认 0, 1.5, 0）

缩放设置:
- defaultDistance: 默认距离（默认 5）
- minDistance / maxDistance: 距离限制（2 ~ 10）
- zoomSpeed: 缩放速度（默认 2）

动态 FOV:
- defaultFOV: 默认视野（默认 60）
- sprintFOV: 冲刺视野（默认 70）
```

### PlayerController 主要参数（重构版）

```
移动设置:
- walkSpeed: 行走速度（默认 2）
- slowRunSpeed: 慢跑速度（默认 4）
- fastRunSpeed: 快跑速度（默认 8）
- acceleration / deceleration: 加速度/减速度（默认 10/8）

旋转设置:
- turnSmoothTime: 转身平滑时间（默认 0.1）

组件引用:
- cameraRig: ThirdPersonCamera 引用
```

### 系统组件

1. **ThirdPersonCamera.cs** — 轨道摄像机控制（新版）
2. **PlayerController.cs** — 基于摄像机方向的移动 + 平滑转身
3. **PlayerInputManager.cs** — 统一输入，含 `ScrollDelta` 属性（滚轮缩放）
4. **PlayerCharacterManager.cs** — 角色生成时自动设置 ThirdPersonCamera 引用

## 旧方案参考：ThirdPersonCameraRig（固定视角版，已废弃）

`ThirdPersonCameraRig` 已标记为 `[Obsolete]`，新功能不要使用。以下内容仅供参考。

### 固定视角核心特性

- 鼠标移动控制角色旋转（而非摄像机）
- 摄像机固定偏移跟随，同步角色旋转
- 移动基于角色朝向
- Tab 键切换鼠标锁定

### ThirdPersonCameraRig 配置参数

```
跟随设置:
- Camera Offset: 摄像机相对角色的偏移（默认 (0, 5, -8)）
- Follow Smooth Speed: 位置跟随平滑速度（默认 10）
- Rotation Smooth Speed: 旋转跟随平滑速度（默认 10）

摄像机朝向:
- Look At Offset: 摄像机看向目标的偏移（默认 (0, 1, 0)）

碰撞设置:
- Enable Collision: 是否启用碰撞检测（默认 true）
- Collision Buffer: 碰撞缓冲距离（默认 0.2m）
```

## 常见问题排查

| 症状 | 解决方案 |
|------|---------|
| 鼠标移动但角色不旋转 | 在固定视角模式下才有此行为；当前方案鼠标控制摄像机 |
| 按 W 但角色不向前移动 | 检查角色朝向，当前方案 W 是基于摄像机方向向前 |
| 摄像机不跟随 | 检查 ThirdPersonCamera 的 target 是否正确设置 |
| 摄像机穿墙 | 检查 occlusionLayers 设置 |
| 进入游戏后鼠标没有隐藏 | 检查 PlayerInputManager 的 Lock Cursor 选项 |

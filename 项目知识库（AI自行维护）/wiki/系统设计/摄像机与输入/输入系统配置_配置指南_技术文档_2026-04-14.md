## 🏗️ 架构说明

### 1. **IPlayerInputSource** - 输入源接口
```csharp
public interface IPlayerInputSource
{
    Vector2 GetMove();           // 获取移动输入 (WASD/方向键)
    bool GetSkillDown(int slot); // 获取技能按下 (J/K/L)
    bool GetSkillHeld(int slot); // 获取技能持续按住
}
```

**作用：** 定义统一的输入接口，方便切换不同的输入源（键盘、手柄、触摸屏等）

---

### 2. **KeyboardInputSource** - 键盘输入实现
```csharp
public class KeyboardInputSource : IPlayerInputSource
{
    public Vector2 GetMove()
        => new Vector2(Input.GetAxisRaw("Horizontal"), Input.GetAxisRaw("Vertical")).normalized;

    public bool GetSkillDown(int slot)
    {
        return slot switch
        {
            1 => Input.GetKeyDown(KeyCode.J),  // 技能1
            2 => Input.GetKeyDown(KeyCode.K),  // 技能2
            3 => Input.GetKeyDown(KeyCode.L),  // 技能3
            _ => false
        };
    }
}
```

**当前键位映射：**
- **移动**：WASD / 方向键（通过 Unity Input Manager 的 Horizontal/Vertical 轴）
- **技能1**：J 键
- **技能2**：K 键
- **技能3**：L 键

---

### 3. **PlayerInputManager** - 输入管理器（单例）
```csharp
public class PlayerInputManager : MonoBehaviour
{
    public static PlayerInputManager Instance { get; private set; }
    
    public Vector2 Move { get; private set; }        // 当前移动输入
    public bool SkillDown(int slot) => ...           // 查询技能是否按下
    public void SetEnable(bool v) => ...             // 启用/禁用输入
    
    private IPlayerInputSource source;               // 当前输入源
}
```

**特点：**
- ✅ 单例模式，全局唯一
- ✅ DontDestroyOnLoad，跨场景持久化
- ✅ 支持启用/禁用输入
- ✅ 可动态切换输入源

## ⚙️ 配置步骤

### 步骤 1：创建 PlayerInputManager 游戏对象

在**任意场景**（建议在启动场景）中创建：

1. 在 Hierarchy 中右键 → Create Empty
2. 重命名为 `PlayerInputManager`
3. 添加 `PlayerInputManager` 组件
4. **不需要**手动设置 DontDestroyOnLoad（脚本已处理）

**位置建议：**
```
Hierarchy
├── GameRoot (或其他管理器父对象)
│   ├── PlayerInputManager  ← 在这里
│   ├── PlayerAccountDataManager
│   └── PlayerCharacterManager
```

---

### 步骤 2：配置 Unity Input Manager

确保 Unity 的 Input Manager 已正确配置：

**打开方式：** Edit → Project Settings → Input Manager

**必需的轴配置：**

#### Horizontal 轴
```
Name: Horizontal
Negative Button: left (或 a)
Positive Button: right (或 d)
Alt Negative Button: a
Alt Positive Button: d
Gravity: 3
Dead: 0.001
Sensitivity: 3
Type: Key or Mouse Button
```

#### Vertical 轴
```
Name: Vertical
Negative Button: down (或 s)
Positive Button: up (或 w)
Alt Negative Button: s
Alt Positive Button: w
Gravity: 3
Dead: 0.001
Sensitivity: 3
Type: Key or Mouse Button
```

**注意：** 如果你的项目已经配置了这些轴，无需修改。

---

### 步骤 3：验证输入系统

#### 方法 1：在编辑器中测试

1. 运行游戏
2. 打开 Console 查看日志：
   ```
   PlayerInputManager 已启用
   ```
3. 按 WASD 键，观察角色是否移动
4. 按 J/K/L 键，观察技能是否触发

#### 方法 2：添加调试代码

在 `PlayerInputManager.Update()` 中临时添加：

```csharp
private void Update()
{
    // ... 现有代码 ...
    
    // 调试输出
    if (Move.sqrMagnitude > 0.01f)
    {
        Debug.Log($"移动输入: {Move}");
    }

    {
        if (skillDown[slot])
        {
            Debug.Log($"技能 {slot} 按下");
        }
```

## 🎯 使用示例

### 在 PlayerController 中使用

```csharp
private void HandleMovement()
{
    // 从输入管理器获取输入
    Vector2 input = Vector2.zero;
    if (PlayerInputManager.Instance != null)
        input = PlayerInputManager.Instance.Move;

    Vector3 moveDirection = new Vector3(input.x, 0f, input.y);
    
    if (moveDirection.sqrMagnitude >= 0.01f)
    {
        // 移动角色
        characterController.Move(moveDirection * moveSpeed * Time.deltaTime);
    }
}
```

## 🔧 扩展功能

### 1. 添加手柄支持

创建 `JoystickInputSource.cs`：

```csharp
public class JoystickInputSource : IPlayerInputSource
{
    public Vector2 GetMove()
    {
        float h = Input.GetAxis("Horizontal");
        float v = Input.GetAxis("Vertical");
        return new Vector2(h, v);
    }

    public bool GetSkillDown(int slot)
    {
        return slot switch
        {
            1 => Input.GetButtonDown("Fire1"),
            2 => Input.GetButtonDown("Fire2"),
            3 => Input.GetButtonDown("Fire3"),
            _ => false
        };
    }

    public bool GetSkillHeld(int slot)
    {
        return slot switch
        {
            1 => Input.GetButton("Fire1"),
            2 => Input.GetButton("Fire2"),
            3 => Input.GetButton("Fire3"),
            _ => false
        };
    }
}
```

在 `PlayerInputManager.Awake()` 中切换：

```csharp
private void Awake()
{
    // ... 现有代码 ...
    
    // 根据平台选择输入源
#if UNITY_STANDALONE || UNITY_EDITOR
    source = new KeyboardInputSource();
#elif UNITY_ANDROID || UNITY_IOS
    source = new TouchInputSource();
#endif
}
```

## 🐛 常见问题

### Q1: 角色不响应输入？

**检查清单：**
1. ✅ PlayerInputManager 游戏对象是否存在？
2. ✅ PlayerInputManager.Instance 是否为 null？
3. ✅ enableInput 是否为 true？
4. ✅ PlayerController 是否已启用？
5. ✅ Unity Input Manager 是否正确配置？

**调试代码：**
```csharp
void Update()
{
    if (PlayerInputManager.Instance == null)
    {
        Debug.LogError("PlayerInputManager.Instance 为 null！");
        return;
    }
    
    Vector2 input = PlayerInputManager.Instance.Move;
    Debug.Log($"输入值: {input}");
}
```

---

### Q2: 输入延迟或不流畅？

**解决方案：**
1. 检查 Time.timeScale 是否为 1
2. 确保 PlayerInputManager 在 Update() 中更新
3. 使用 `GetAxisRaw` 而不是 `GetAxis`（已使用）
4. 检查帧率是否稳定

---

### Q3: 如何支持键位自定义？

**实现思路：**

1. 创建键位配置类：
```csharp
[System.Serializable]
public class KeyBindings
{
    public KeyCode skill1 = KeyCode.J;
    public KeyCode skill2 = KeyCode.K;
    public KeyCode skill3 = KeyCode.L;
}
```

2. 修改 KeyboardInputSource：
```csharp
public class KeyboardInputSource : IPlayerInputSource
{
    private KeyBindings bindings;

    public KeyboardInputSource(KeyBindings bindings = null)
    {
        this.bindings = bindings ?? new KeyBindings();
    }

    public bool GetSkillDown(int slot)
    {
        return slot switch
        {
            1 => Input.GetKeyDown(bindings.skill1),
            2 => Input.GetKeyDown(bindings.skill2),
            3 => Input.GetKeyDown(bindings.skill3),
            _ => false
        };
    }
}
```

## 📊 完整工作流程

```
1. 游戏启动
   └─> PlayerInputManager.Awake()
       └─> 创建 KeyboardInputSource
       └─> 设置为 DontDestroyOnLoad

2. 每帧更新
   └─> PlayerInputManager.Update()
       └─> source.GetMove() → 获取移动输入
       └─> source.GetSkillDown() → 获取技能输入

3. PlayerController 使用输入
   └─> PlayerController.Update()
       └─> HandleMovement()
           └─> PlayerInputManager.Instance.Move
           └─> 移动角色

4. 技能系统使用输入
   └─> PlayerSkillManager.Update()
       └─> PlayerInputManager.Instance.SkillDown(1)
       └─> 释放技能
```

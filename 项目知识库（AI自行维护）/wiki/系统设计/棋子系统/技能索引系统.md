# 技能索引系统

合并自：技能索引更新指南（使用说明）+ 技能索引钩子（完成总结）

---

## 系统概述

技能索引系统通过一个手动触发的钩子（`update-skill-index.kiro.hook`）扫描 `.kiro/skills/` 目录下所有 SKILL.md，提取技能信息，更新 `skill-auto-indexer.kiro.hook` 中的触发词映射，确保 AI 智能推荐始终保持最新。

**选择手动触发而非自动触发的原因**：避免 SKILL.md → 更新钩子 → 触发钩子 → 再次修改 → 循环的死循环风险。

---

## 何时需要更新索引

- 在 `.kiro/skills/` 下创建了新技能或添加了 SKILL.md
- 修改了 SKILL.md 的 name / description / keywords
- 删除了某个技能

---

## 如何触发更新

### 方法1：Kiro UI（推荐）

Agent Hooks 面板 → 找到"更新 SKILL 索引" → 点击触发按钮（▶）

### 方法2：命令面板

`Ctrl+Shift+P` → 输入 `Trigger Hook` → 选择"更新 SKILL 索引"

### 方法3：对话请求

```
请更新技能索引
```

---

## 更新流程

```
用户触发钩子
    ↓
扫描 .kiro/skills/**/SKILL.md
    ↓
提取 frontmatter（name / description / keywords）
    ↓
生成触发词映射
    ↓
更新 skill-auto-indexer.kiro.hook（保留 GitNexus 部分不变）
    ↓
输出更新报告
```

---

## SKILL.md 格式要求

```markdown
---
name: 技能名称（小写+连字符）
description: 简短描述，包含关键词
keywords: 关键词1, 关键词2, 关键词3
license: MIT
---

# 技能标题

详细文档...
```

- `name`：必需，唯一标识符
- `description`：必需，用于生成关键词
- `keywords`：可选，无该字段时从 description 自动提取

## 系统关系图

```
.kiro/skills/**/SKILL.md（技能定义）
         ↓ 扫描
update-skill-index.kiro.hook（手动触发）
         ↓ 更新
skill-auto-indexer.kiro.hook（自动触发）
         ↓ 推荐
用户提交消息 → 智能推荐相关技能
```

## 常见问题

**Q：更新需要多长时间？**
A：通常 10-30 秒，取决于技能数量。

**Q：更新后需要重启 Kiro 吗？**
A：不需要，钩子配置自动重新加载。

**Q：更新失败怎么办？**
1. 检查 SKILL.md 的 frontmatter 格式是否正确
2. 检查是否有 `.kiro/hooks/` 目录的写入权限
3. 查看 Agent 错误日志

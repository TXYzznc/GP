## 📋 系统概述

棋子战斗系统是自走棋玩法的核心，负责管理棋子的AI决策、技能释放、伤害计算和命中检测。

### 核心特性
- **AI完全控制攻击**：AI负责目标搜索和攻击决策
- **技能系统解耦**：技能逻辑独立，易于扩展
- **多种命中检测**：支持瞬发、近战、投射物、AOE、射线
- **数据驱动配置**：所有参数通过配置表管理

## 🏗️ 系统架构

### 职责划分

| 组件 | 职责 | 关键方法 |
|:---|:---|:---|
| **ChessEntity** | 棋子实体，持有所有组件 | `Initialize()`, `Tick()` |
| **ChessAI** | 目标搜索、攻击决策、触发攻击 | `Tick()`, `Attack()` |
| **CombatController** | 动画事件回调、伤害应用 | `OnAttackExecute()`, `OnSkillExecute()` |
| **IChessSkill** | 技能逻辑实现 | `ExecuteSkill()`, `CalculateDamage()` |
| **HitDetector** | 命中检测 | `DetectHit()` |

### 工作流程

```
ChessEntity.Tick()
    ↓
AI.Tick()
    ├─ 寻找目标
    ├─ 检查冷却
    └─ 触发攻击 → AI.Attack(target)
        ↓
    播放攻击动画
        ↓
    动画事件 → AnimEvent_AttackHit
        ↓
    CombatController.OnAttackExecute()
        ├─ 获取缓存的目标和伤害
        ├─ 选择命中检测器
        └─ 应用伤害和效果
```

## 🎯 命中检测系统

### 检测类型

| 类型 | 值 | 说明 | 适用场景 |
|:---|:---|:---|:---|
| **Instant** | 0 | 瞬发命中，直接对锁定目标造成伤害 | 自动战斗、单体技能 |
| **Melee** | 1 | 近战碰撞，通过武器Collider检测 | 近战武器挥砍 |
| **Projectile** | 2 | 投射物，子弹/箭矢通过碰撞检测 | 弓箭射击、远程攻击 |
| **AOE** | 3 | 范围检测，OverlapSphere | AOE技能、爆炸 |
| **Raycast** | 4 | 射线检测，Raycast | 激光、穿透攻击 |

### 配置方式

**配置位置**：`SummonChessTable.xlsx`

```
字段：
- AttackHitType: 普攻命中检测类型（0-4）
- ProjectilePrefabId: 投射物预制体ID（类型=2时使用）
```

**示例**：
```
后羿：
  AttackHitType = 0 (Instant)  # 瞬发命中
  
近战棋子：
  AttackHitType = 1 (Melee)    # 近战碰撞
  
弓箭手：
  AttackHitType = 2 (Projectile)  # 投射物
  ProjectilePrefabId = 3001       # 箭矢预制体
```

## 🔧 技能系统架构

### 设计原则

**重构目标**：将技能执行的控制权从 Controller 下放到技能实现类

**优势**：
- 高扩展性：新技能只需实现接口
- 高可读性：每个技能的逻辑独立清晰
- 高灵活性：技能可以完全自定义执行流程
- 职责分离：Controller 只负责事件转发

### 架构对比

**重构前**（问题）：
```csharp
ChessCombatController
├── OnAttackExecute()  // 100+ 行
│   ├── 查找目标
│   ├── 计算伤害
│   ├── 构建 HitContext
│   ├── 播放特效
│   └── 调用 NormalAttack.Execute()
├── OnSkill1Execute()  // 重复逻辑
└── OnSkill2Execute()  // 重复逻辑 + 特殊判断
```

**重构后**（目标）：
```csharp
ChessCombatController (薄层事件转发)
├── OnAttackExecute()
│   └── m_Entity.NormalAttack.ExecuteAttack(m_Entity)
├── OnSkill1Execute()
│   └── m_Entity.Skill1.ExecuteSkill(m_Entity)
└── OnSkill2Execute()
    └── m_Entity.Skill2.ExecuteSkill(m_Entity)

IChessSkill (完整控制技能流程)
├── ExecuteSkill(caster)
│   ├── 查找目标
│   ├── 计算伤害
│   ├── 构建 HitContext
│   ├── 播放特效
│   └── 应用伤害
```

### 技能接口

```csharp
public interface IChessSkill
{
    // 执行技能（完整流程）
    void ExecuteSkill(ChessEntity caster);
    
    // 计算伤害
    int CalculateDamage(ChessEntity caster, ChessEntity target);
    
    // 获取技能目标
    List<ChessEntity> GetTargets(ChessEntity caster);
    
    // 技能冷却
    float GetCooldown();
    bool IsReady();
}
```

## 🤖 AI战斗系统

### AI类型

| AI类型 | 说明 | 行为特点 |
|:---|:---|:---|
| **RangedAI** | 远程AI | 保持距离，自动攻击 |
| **MeleeAI** | 近战AI | 追击目标，近身攻击 |
| **SupportAI** | 辅助AI | 优先治疗队友 |

### AI决策流程

```csharp
AI.Tick()
{
    // 1. 寻找目标
    target = FindBestTarget();
    if (target == null) return;
    
    // 2. 检查冷却
    if (!IsAttackReady()) return;
    
    // 3. 检查距离
    if (!IsInAttackRange(target))
    {
        MoveTowards(target);
        return;
    }
    
    // 4. 触发攻击
    Attack(target);
}
```

### 目标选择策略

```csharp
// 优先级排序
1. 距离最近的敌人
2. 血量最低的敌人
3. 威胁度最高的敌人（Boss > 精英 > 普通）
```

## 💥 伤害和效果应用

### 伤害计算流程

```
1. 基础伤害 = 攻击力 × 技能倍率
2. 暴击判定 = Random(0,1) < 暴击率
3. 最终伤害 = 基础伤害 × (暴击 ? 暴击倍率 : 1.0) - 防御力
4. 应用伤害 → target.TakeDamage(finalDamage)
```

### HitContext 结构

```csharp
public class HitContext
{
    public ChessEntity Attacker;      // 攻击者
    public ChessEntity Target;        // 目标
    public int Damage;                // 伤害值
    public bool IsCritical;           // 是否暴击
    public Vector3 HitPoint;          // 命中点
    public Vector3 HitNormal;         // 命中法线
    public List<IBuff> Buffs;         // 附加Buff
}
```

## 🔍 关键实现细节

### 1. 攻击缓存机制

```csharp
// AI触发攻击时缓存数据
public void Attack(ChessEntity target)
{
    m_CachedTarget = target;
    m_CachedDamage = CalculateDamage(target);
    PlayAttackAnimation();
}

// 动画事件回调时使用缓存
public void OnAttackExecute()
{
    if (m_CachedTarget != null)
    {
        ApplyDamage(m_CachedTarget, m_CachedDamage);
        m_CachedTarget = null;
    }
}
```

### 2. 命中检测器选择

```csharp
IHitDetector GetHitDetector(AttackHitType type)
{
    switch (type)
    {
        case AttackHitType.Instant:
            return new InstantHitDetector();
        case AttackHitType.Melee:
            return new MeleeHitDetector();
        case AttackHitType.Projectile:
            return new ProjectileHitDetector();
        case AttackHitType.AOE:
            return new AOEHitDetector();
        case AttackHitType.Raycast:
            return new RaycastHitDetector();
    }
}
```

### 3. 技能冷却管理

```csharp
private float m_LastAttackTime;
private float m_AttackCooldown;

public bool IsAttackReady()
{
    return Time.time >= m_LastAttackTime + m_AttackCooldown;
}

public void SetAttackCooldown()
{
    m_LastAttackTime = Time.time;
}
```

3. 配置动画控制器

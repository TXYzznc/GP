## 第三步：创建棋子预制体

### 3.1 基础预制体结构

创建棋子预制体：`Prefabs/SummonChess/Chess_Houyi.prefab`

**预制体结构：**
```
Chess_Houyi (GameObject)
├── Model (子物体 - 3D模型)
│   └── Mesh Renderer
├── Collider (Capsule Collider)
└── 可选：特效、动画控制器等
```

**注意：**
- 不需要手动添加 `ChessEntity` 组件，系统会自动添加
- 不需要手动添加 `ChessAttribute` 组件，系统会自动添加
- 预制体的 Transform 位置应该在 (0, 0, 0)

### 3.2 配置资源ID

在 `ResourceIds.cs` 中添加棋子预制体的资源ID：

```csharp
// 自走棋棋子预制体
```

将预制体添加到Addressables系统：
1. 选中预制体
2. 勾选 `Addressable`
3. 设置地址为资源ID对应的路径

## 第四步：编写测试脚本

创建测试脚本：`Assets/AAAGame/Scripts/Test/ChessSystemTest.cs`

```csharp
using UnityEngine;
using AAAGame;
using Cysharp.Threading.Tasks;

public class ChessSystemTest : MonoBehaviour
{
    [Header("测试配置")]
    [SerializeField] private int m_TestChessId = 1001;
    [SerializeField] private Vector3 m_SpawnPosition = new Vector3(0, 0, 0);

    private void Start()
    {
        Debug.Log("========== 棋子系统测试开始 ==========");
    }

    private void Update()
    {
        // 按键测试
        if (Input.GetKeyDown(KeyCode.Alpha1))
        {
            TestSpawnChess();
        }

        if (Input.GetKeyDown(KeyCode.Alpha2))
        {
            TestUnlockChess();
        }

        if (Input.GetKeyDown(KeyCode.Alpha3))
        {
            TestPrintSystemStatus();
        }

        if (Input.GetKeyDown(KeyCode.Alpha4))
        {
            TestDestroyAllChess();
        }
    }

    /// <summary>
    /// 测试1：生成棋子
    /// </summary>
    [ContextMenu("测试1：生成棋子")]
    private async void TestSpawnChess()
    {
        Debug.Log($"[测试] 开始生成棋子，ID={m_TestChessId}");

        var entity = await SummonChessManager.Instance.SpawnChessAsync(
            m_TestChessId,
            m_SpawnPosition,
            0  // 玩家阵营
        );

        if (entity != null)
        {
            Debug.Log($"[测试] ✓ 棋子生成成功：{entity.Config.Name}");
            Debug.Log($"  - 生命值：{entity.Attribute.CurrentHp}/{entity.Attribute.MaxHp}");
            Debug.Log($"  - 法力值：{entity.Attribute.CurrentMp}/{entity.Attribute.MaxMp}");
            Debug.Log($"  - 攻击力：{entity.Attribute.AtkDamage}");
            Debug.Log($"  - 攻击速度：{entity.Attribute.AtkSpeed}");
            Debug.Log($"  - 移动速度：{entity.Attribute.MoveSpeed}");
        }
        else
        {
            Debug.LogError("[测试] ✗ 棋子生成失败");
        }
    }

    /// <summary>
    /// 测试2：解锁棋子
    /// </summary>
    [ContextMenu("测试2：解锁棋子")]
    private void TestUnlockChess()
    {
        Debug.Log($"[测试] 开始解锁棋子，ID={m_TestChessId}");

        bool success = ChessUnlockManager.Instance.UnlockChess(m_TestChessId);

        if (success)
        {
            Debug.Log($"[测试] ✓ 棋子解锁成功");
        }
        else
        {
            Debug.Log($"[测试] - 棋子已解锁");
        }

        Debug.Log($"[测试] 当前已解锁棋子数量：{ChessUnlockManager.Instance.GetUnlockedCount()}");
    }

    /// <summary>
    /// 测试3：打印系统状态
    /// </summary>
    [ContextMenu("测试3：打印系统状态")]
    private void TestPrintSystemStatus()
    {
        Debug.Log("========== 系统状态 ==========");
        Debug.Log(ChessDataManager.Instance.GetDebugInfo());
        Debug.Log(ChessFactory.GetDebugInfo());
        Debug.Log(ChessUnlockManager.Instance.GetDebugInfo());
        Debug.Log(SummonChessManager.Instance.GetDebugInfo());
        Debug.Log("==============================");

        // 打印所有已解锁的棋子
        ChessUnlockManager.Instance.PrintUnlockedChess();
    }

    /// <summary>
    /// 测试4：销毁所有棋子
    /// </summary>
    [ContextMenu("测试4：销毁所有棋子")]
    private void TestDestroyAllChess()
    {
        var allChess = SummonChessManager.Instance.GetAllChess();
        Debug.Log($"[测试] 开始销毁所有棋子，当前数量：{allChess.Count}");

        // 复制列表，避免在遍历时修改
        var chessList = new System.Collections.Generic.List<ChessEntity>(allChess);

        foreach (var chess in chessList)
        {
            SummonChessManager.Instance.DestroyChess(chess);
        }

        Debug.Log($"[测试] ✓ 所有棋子已销毁");
    }

    /// <summary>
    /// 测试5：测试属性修改
    /// </summary>
    [ContextMenu("测试5：测试属性修改")]
    private void TestAttributeModify()
    {
        var allChess = SummonChessManager.Instance.GetAllChess();
        if (allChess.Count == 0)
        {
            Debug.LogWarning("[测试] 场景中没有棋子");
            return;
        }

        var chess = allChess[0];
        Debug.Log($"[测试] 测试棋子：{chess.Config.Name}");

        // 测试生命值修改
        Debug.Log($"  - 当前生命值：{chess.Attribute.CurrentHp}");
        chess.Attribute.ModifyHp(-100);
        Debug.Log($"  - 扣除100生命值后：{chess.Attribute.CurrentHp}");

        // 测试法力值修改
        Debug.Log($"  - 当前法力值：{chess.Attribute.CurrentMp}");
        chess.Attribute.ModifyMp(50);
        Debug.Log($"  - 增加50法力值后：{chess.Attribute.CurrentMp}");

        // 测试伤害计算
        double physicalDamage = chess.Attribute.CalculatePhysicalDamage(100);
        Debug.Log($"  - 100点物理伤害实际造成：{physicalDamage}");

        double magicDamage = chess.Attribute.CalculateMagicDamage(100);
        Debug.Log($"  - 100点魔法伤害实际造成：{magicDamage}");
    }

    /// <summary>
    /// 测试6：测试查询接口
    /// </summary>
    [ContextMenu("测试6：测试查询接口")]
    private void TestQueryInterface()
    {
        Debug.Log("========== 查询接口测试 ==========");

        // 查询所有棋子
        var allChess = SummonChessManager.Instance.GetAllChess();
        Debug.Log($"所有棋子数量：{allChess.Count}");

        // 查询玩家阵营棋子
        var playerChess = SummonChessManager.Instance.GetChessByCamp(0);
        Debug.Log($"玩家阵营棋子数量：{playerChess.Count}");

        // 查询范围内棋子
        var nearbyChess = SummonChessManager.Instance.GetChessInRange(Vector3.zero, 10f);
        Debug.Log($"原点10米范围内棋子数量：{nearbyChess.Count}");

        Debug.Log("==================================");
    }
}
```

## 第五步：测试流程

### 5.1 基础功能测试

1. **启动游戏**
   - 打开测试场景
   - 点击 Play
   - 观察控制台输出，确认系统初始化成功

2. **测试棋子生成**
   - 按键盘 `1` 键
   - 观察场景中是否生成棋子
   - 检查控制台日志，确认属性正确

3. **测试解锁系统**
   - 按键盘 `2` 键
   - 观察控制台输出，确认解锁成功
   - 重复按 `2` 键，确认去重功能正常

4. **测试系统状态**
   - 按键盘 `3` 键
   - 查看控制台输出的系统状态信息

5. **测试销毁功能**
   - 先按 `1` 键生成几个棋子
   - 按键盘 `4` 键
   - 观察场景中棋子是否被销毁

### 5.2 存档系统测试

1. **创建新存档**
   - 运行游戏，创建新存档
   - 解锁几个棋子（按 `2` 键）
   - 保存存档

2. **加载存档**
   - 退出游戏
   - 重新启动游戏
   - 加载之前的存档
   - 按 `3` 键查看系统状态
   - 确认解锁的棋子数据正确加载

### 5.3 属性系统测试

1. 生成一个棋子
2. 在Inspector中选中棋子
3. 右键点击 `ChessSystemTest` 组件
4. 选择 `测试5：测试属性修改`
5. 观察控制台输出，确认属性修改和伤害计算正确

### 5.4 AI系统测试（占位）

## 第六步：常见问题排查

### 问题1：配置表加载失败

**症状：** 控制台报错 "SummonChessTable 未加载"

**解决方案：**
1. 确认配置表已创建并导入
2. 检查配置表名称是否为 `SummonChessTable`
3. 确认配置表已添加到DataTable系统

### 问题2：预制体加载失败

**症状：** 控制台报错 "prefab not found"

**解决方案：**
1. 确认预制体已创建
2. 检查资源ID是否正确配置
3. 确认预制体已添加到Addressables
4. 检查Addressables地址是否正确

### 问题3：棋子生成后立即消失

**症状：** 棋子生成后马上被销毁

**解决方案：**
1. 检查棋子初始生命值是否大于0
2. 确认没有其他脚本在销毁棋子
3. 检查OnHpChanged事件监听是否正确

### 问题4：存档数据未保存

**症状：** 重新加载存档后，解锁数据丢失

**解决方案：**
1. 确认 `PlayerAccountDataManager.SaveCurrentSave()` 被调用
2. 检查 `UnlockedChessIds` 字段是否正确序列化
3. 查看存档文件内容，确认数据已写入

```csharp
private async void TestBatchSpawn()
{
```

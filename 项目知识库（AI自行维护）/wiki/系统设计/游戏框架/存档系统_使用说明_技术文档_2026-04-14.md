## 📚 新API使用指南

### 核心API变化对照表

| 旧API | 新API | 说明 |
|-------|-------|------|
| `CurrentAccountData` | `CurrentSaveData` | 获取当前存档数据 |
| `CreateNewAccount(name, summonerId, saveIndex)` | `CreateNewSave(saveName, summonerId)` | 创建新存档 |
| `LoadAccountData(saveIndex)` | `LoadSave(saveId)` | 加载存档 |
| `SaveAccountData(accountData)` | `SaveCurrentSave()` | 保存当前存档 |
| `HasSaveData(saveIndex)` | - | 不再需要（通过列表判断） |
| `GetSaveInfo(saveIndex)` | `GetSaveBriefInfo(saveId)` | 获取存档信息 |
| `GetAllSaveInfos()` | `GetAllSaveBriefInfos()` | 获取所有存档 |
| `DeleteSaveData(saveIndex)` | `DeleteSave(saveId)` | 删除存档 |

## 🎮 使用示例

### 1. 游戏启动时设置账号ID

```csharp
void OnGameStart()
{
    // 设置当前账号ID（未来由登录系统提供，现在固定为 "000001"）
    PlayerAccountDataManager.Instance.SetCurrentAccountId("000001");
    
    // 尝试自动加载最近的存档
    if (PlayerAccountDataManager.Instance.AutoLoadLastSave())
    {
        Debug.Log("自动加载最近存档成功");
        // 进入游戏
        EnterGame();
    }
    else
    {
        Debug.Log("没有存档，显示主菜单");
        // 显示主菜单或存档列表
        ShowMainMenu();
    }
}
```

### 2. 创建新存档（NewGameUI 中）

```csharp
// 已在 NewGameUI.cs 中实现
private void OnCharacterCreationComplete()
{
    var selectedSummoner = GetCurrentSummoner();
    
    // 创建新存档
    var saveData = PlayerAccountDataManager.Instance.CreateNewSave(
        playerName,              // 存档名称
        selectedSummoner.Id      // 召唤师ID
    );
    
    if (saveData != null)
    {
        Debug.Log($"存档创建成功: {saveData.SaveId}");
        // 进入游戏
    }
}
```

### 3. 显示存档列表

```csharp
void ShowSaveList()
{
    // 获取所有存档（按最近使用排序）
    var saveInfos = PlayerAccountDataManager.Instance.GetAllSaveBriefInfos();
    
    foreach (var info in saveInfos)
    {
        Debug.Log($"存档: {info.SaveName}");
        Debug.Log($"  等级: {info.GlobalLevel}");
        Debug.Log($"  SaveId: {info.SaveId}");
        
        // 在UI上显示
        CreateSaveSlotUI(info);
    }
}
```

### 4. 加载指定存档

```csharp
void OnClickLoadSave(string saveId)
{
    var saveData = PlayerAccountDataManager.Instance.LoadSave(saveId);
    
    if (saveData != null)
    {
        Debug.Log($"加载存档成功: {saveData.SaveName}");
        // 进入游戏
        EnterGame();
    }
}
```

### 5. 删除存档

```csharp
void OnClickDeleteSave(string saveId)
{
    // 显示确认对话框后
    PlayerAccountDataManager.Instance.DeleteSave(saveId);
    
    // 刷新存档列表
    RefreshSaveList();
}
```

### 6. 游戏中保存

```csharp
void OnGameSave()
{
    // 快速保存当前存档
    PlayerAccountDataManager.Instance.QuickSave();
    
    Debug.Log("游戏已保存");
}
```

### 7. 访问当前存档数据

```csharp
void ShowPlayerInfo()
{
    var currentSave = PlayerAccountDataManager.Instance.CurrentSaveData;
    
    if (currentSave != null)
    {
        Debug.Log($"存档名称: {currentSave.SaveName}");
        Debug.Log($"玩家等级: {currentSave.GlobalLevel}");
        Debug.Log($"金币: {currentSave.Gold}");
        Debug.Log($"钻石: {currentSave.Diamond}");
    }
}
```

## 🔄 存档栈机制

### 工作原理

1. **新建存档**：自动添加到栈顶（列表索引0）
2. **加载存档**：自动移动到栈顶
3. **显示列表**：按栈顺序显示（最近使用的在最前面）

### 示例流程

```
初始状态：
SaveIdStack: []

创建存档A：
SaveIdStack: [A]

创建存档B：
SaveIdStack: [B, A]

创建存档C：
SaveIdStack: [C, B, A]

加载存档A：
SaveIdStack: [A, C, B]  // A被移到最前面

加载存档B：
SaveIdStack: [B, A, C]  // B被移到最前面
```

## 💡 注意事项

### 1. 账号ID管理

当前账号ID固定为 "000001"，未来需要：
- 实现登录系统
- 登录成功后调用 `SetCurrentAccountId(accountId)`
- 支持多账号切换

### 2. 存档数量

- 旧系统：固定3个存档槽位
- 新系统：无限制（由磁盘空间决定）

### 3. 数据兼容性

旧的存档数据（PlayerAccount_0.json 等）无法直接迁移到新系统。如果需要迁移：

```csharp
// 迁移示例（需要自己实现）

{
    for (int i = 0; i < 3; i++)
    {
        string oldPath = Path.Combine(SaveRootPath, $"PlayerAccount_{i}.json");
        if (File.Exists(oldPath))
        {
            // 读取旧数据
            string json = File.ReadAllText(oldPath);
            var oldData = JsonUtility.FromJson<PlayerAccountData>(json);
            
            // 转换为新数据
            var newData = ConvertToNewSaveData(oldData);
            
            // 保存为新格式
            // ...
        }
    }
}
```

## 📞 常见问题

### Q: 如何限制存档数量？

```csharp
void OnCreateNewSaveClick()
{
    var saveInfos = PlayerAccountDataManager.Instance.GetAllSaveBriefInfos();
    
    if (saveInfos.Count >= 10)  // 限制最多10个存档
    {
        GF.UI.ShowToast("存档数量已达上限", UIExtension.ToastStyle.Red);
        return;
    }
    
    // 创建新存档
    GF.UI.OpenUIForm("NewGameUI");
}
```

### Q: 如何重命名存档？

```csharp
void RenameSave(string saveId, string newName)
{
    var saveData = PlayerAccountDataManager.Instance.LoadSave(saveId);
    if (saveData != null)
    {
        saveData.SaveName = newName;
        PlayerAccountDataManager.Instance.SaveCurrentSave();
    }
}
```

### Q: 如何备份存档？

存档文件位于：
```
Windows: C:\Users\[用户名]\AppData\LocalLow\[公司名]\[产品名]\PlayerSaves\000001\
Android: /storage/emulated/0/Android/data/[包名]/files/PlayerSaves/000001/
```

直接复制整个 `000001` 文件夹即可备份。

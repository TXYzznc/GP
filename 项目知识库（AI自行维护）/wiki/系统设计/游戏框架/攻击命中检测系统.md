# 攻击命中检测系统

> **最后更新**: 2026-04-17
> **状态**: 已验证有效
> **系统**: 攻击命中检测

---

## 一、系统概述

实现完整的棋子攻击命中检测系统，支持5种不同的攻击类型，将动画事件与实际命中检测解耦。

**核心设计原则：**
- 动画帧事件只决定"攻击执行"时机，不决定"命中"时机
- 命中检测由各自的检测系统负责
- 所有伤害最终通过 `TakeDamage()` 统一入口

---

## 二、支持的攻击类型

| 类型值 | 枚举名 | 说明 | 适用场景 |
|--------|--------|------|----------|
| 0 | Instant | 瞬发命中 | 自动战斗普攻、锁定类技能 |
| 1 | Melee | 近战碰撞 | 挥砍、刺击等近战攻击 |
| 2 | Projectile | 投射物 | 弓箭、火球、飞刀等远程攻击 |
| 3 | AOE | 范围检测 | AOE技能、爆炸、范围治疗 |
| 4 | Raycast | 射线检测 | 激光、狙击、穿透攻击 |

## 四、配置表字段

### SummonChessTable（棋子表）新增字段

| 字段名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| AttackHitType | int | 0 | 普攻命中类型 |
| ProjectilePrefabId | int | 0 | 投射物预制体ID（AttackHitType=2时使用） |

**配置示例：**

| Id | Name | AttackHitType | ProjectilePrefabId |
|----|------|---------------|--------------------|
| 1001 | 剑士 | 1 | 0 |
| 2001 | 弓箭手 | 2 | 101 |
| 2002 | 法师 | 2 | 102 |
| 3001 | 稻草人 | 0 | 0 |

### SummonChessSkillTable（技能表）命中类型推断规则

技能表没有显式的 `HitType` 字段，系统根据以下规则自动推断：

| 条件 | 推断类型 |
|------|----------|
| `ProjectileSpeed > 0` | Projectile（投射物） |
| `AreaRadius > 0` | AOE（范围） |
| `SkillType == 2`（普攻效果）且 `ProjectileSpeed == 0` | 根据棋子 AIType 决定 |
| 其他情况 | Instant（瞬发） |

## 五、预制体配置

### 近战棋子（WeaponCollider 配置）

1. 打开棋子预制体，找到武器骨骼节点（如 `Weapon` 或 `RightHand`）
2. 创建子物体 `WeaponCollider`
3. 添加碰撞器（推荐 BoxCollider 或 CapsuleCollider），设为 **Trigger** 模式
4. 添加 `ChessWeaponCollider` 组件
5. 碰撞器默认禁用，由 `MeleeHitDetector` 自动控制开关

**预制体结构：**
```
Chess_Warrior
└── Model
    └── Armature
        └── RightHand
            └── WeaponCollider
                ├── BoxCollider (isTrigger=true)
                └── ChessWeaponCollider
```

### 远程棋子（投射物预制体配置）

1. 创建投射物预制体（如 `Arrow.prefab`）
2. 添加碰撞器，设为 **Trigger** 模式
3. 添加 `ChessProjectile` 组件并配置参数
4. 保存到 `Resources/Prefabs/Combat/` 目录

**ChessProjectile 参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| Speed | float | 10 | 飞行速度 |
| MaxLifetime | float | 5 | 最大存活时间（秒） |
| DestroyOnHit | bool | true | 命中后是否销毁 |
| IsPiercing | bool | false | 是否穿透（可命中多个目标） |

**投射物预制体命名：** `Resources/Prefabs/Combat/Projectile_{Id}.prefab`

**投射物预制体ID对照：**

| ProjectilePrefabId | 说明 |
|--------------------|------|
| 0 | DefaultProjectile（默认） |
| 101 | 普通箭矢 |
| 102 | 火球 |
| 103 | 冰锥 |
| 104 | 飞刀 |

## 六、动画配置

### Animator Controller 参数

| 参数名 | 类型 | 说明 |
|--------|------|------|
| AttackSpeed | Float | 攻击动画播放速度倍率 |
| IsMoving | Bool | 是否移动中 |
| Attack | Trigger | 触发攻击动画 |
| Skill1 | Trigger | 触发技能1动画 |
| Skill2 | Trigger | 触发技能2动画 |
| Death | Trigger | 触发死亡动画 |

### Animation Event 配置

**普攻动画事件：**

| 帧位置 | 方法名 | 说明 |
|--------|--------|------|
| 出手帧 | AnimEvent_AttackExecute | 触发攻击执行 |
| 挥砍结束帧 | AnimEvent_MeleeAttackEnd | 关闭武器碰撞（仅近战） |
| 最后一帧 | AnimEvent_AnimationComplete("Attack") | 标记动画结束 |

**技能动画事件：**

| 帧位置 | 方法名 | 说明 |
|--------|--------|------|
| 释放帧 | AnimEvent_Skill1Execute | 触发技能1执行 |
| 最后一帧 | AnimEvent_AnimationComplete("Skill1") | 标记动画结束 |

**重要：** `ChessAnimationEventReceiver` 组件必须挂载在 Animator 所在的 GameObject 上，否则 Animation Event 无法触发。

## 九、常见问题

| 问题 | 可能原因 |
|------|----------|
| 近战攻击没有命中效果 | 没有添加 ChessWeaponCollider / 碰撞器不是Trigger / 缺少动画事件 / Layer配置不正确 |
| 投射物没有生成 | ProjectilePrefabId 配置错误 / 预制体路径不正确 / 预制体没有 ChessProjectile 组件 |
| AOE 技能没有命中目标 | AreaRadius 配置为0 / 目标Layer不在检测范围 / 目标阵营配置错误 |
| 动画事件没有触发 | 没有挂载 ChessAnimationEventReceiver / 方法名拼写错误 / 动画Clip没有正确配置事件 |

## 一、配置表概述

物品系统使用 4 个配置表管理所有物品相关数据。解析类由 DataTableGenerator 自动生成，位于 `Assets/AAAGame/Scripts/DataTable/`。

---

## 二、配置表详解

### 1. ItemTable.txt — 物品配置表

**路径**: `Assets/AAAGame/DataTable/ItemTable.txt`

| 字段名 | 类型 | 说明 |
|:-------|:-----|:-----|
| Id | int | 物品ID（1xxxx消耗品，2xxxx任务道具，3xxxx宝物，4xxxx装备） |
| Name | string | 物品名称 |
| Type | int | 类型（1=消耗品，2=任务道具，3=宝物，4=装备） |
| Quality | int | 品质（1=普通，2=优秀，3=稀有，4=史诗，5=传说） |
| Description | string | 物品描述 |
| IconPath | string | 缩略图路径 |
| DetailIconPath | string | 详细图路径（为空则使用 IconPath） |
| CanStack | int | 是否可堆叠（0/1） |
| MaxStackCount | int | 最大堆叠数量 |
| CanUse | int | 是否可使用 |
| UseEffectId | int | 使用效果 ID（关联 SpecialEffectTable） |
| CanEquip | int | 是否可装备 |
| EquipType | int | 装备类型（0=无，1=武器，2=防具，3=饰品，4=宝物） |
| SpecialEffectId | int | 特殊效果 ID（关联 SpecialEffectTable） |
| AffixPoolIds | string | 词条池 ID 列表（逗号分隔，关联 AffixTable） |
| AffixCount | string | 词条数量范围（格式：最小-最大，如 "1-3"） |
| SynergyIds | string | 羁绊 ID 列表（逗号分隔，关联 SynergyTable） |
| BaseAttributes | string | 基础属性（JSON 格式，如 `{"Attack":50}`） |
| SellPrice | int | 售价 |

**示例**:
```
10001  钱包      1  2  使用后随机获得100-500金币  Items/Consumable/Wallet  ...  1  99  1  1001  0  0  0      ...  {}  50
30001  八尺琼勾玉  3  5  传说中的三神器之一  Items/Treasure/Magatama  ...  0  1  0  0  1  4  3001  2001,2002,2003  1-3  3001  {}  1000
40001  石中剑    4  4  传说中的圣剑  Items/Equipment/Excalibur  ...  0  1  0  0  1  1  4001  ...  {"Attack":50,"CritRate":10}  500
```

### 3. AffixTable.txt — 词条配置表

## 四、配置表使用流程

### 编辑与生成

1. 直接编辑 `.txt` 文件（Tab 分隔字段），或编辑对应 `.xlsx`
2. 运行 DataTableGenerator：`GameFramework → DataTable → Generate`
3. 生成的解析类：`ItemTable.cs`、`SpecialEffectTable.cs`、`AffixTable.cs`、`SynergyTable.cs`

> **不要手动修改解析类**，它们由工具自动生成，手动修改会被覆盖。

### 加载配置表

```csharp
// 在 ProcedureLaunch 或启动流程中
GF.DataTable.LoadDataTable<ItemTable>("ItemTable");
GF.DataTable.LoadDataTable<SpecialEffectTable>("SpecialEffectTable");
GF.DataTable.LoadDataTable<AffixTable>("AffixTable");
GF.DataTable.LoadDataTable<SynergyTable>("SynergyTable");
```

### 初始化 ItemManager

```csharp
// 配置表加载完成后调用
ItemManager.Instance.LoadItemTable();
ItemManager.Instance.LoadSpecialEffectTable();
ItemManager.Instance.LoadAffixTable();
ItemManager.Instance.LoadSynergyTable();
```

## 五、PlayerAccountData 迁移指南

### 问题描述

`PlayerAccountData.cs` 中存在旧的 `ItemData` 类定义，与新系统 `AAAGame.Item.ItemData` 冲突。

### 修改方案

**Step 1: 删除旧的 ItemData 类（第 428-433 行）**

```csharp
// 删除以下代码
[Serializable]
public class ItemData
{
    public int ItemId;
    public int Count;
    public long ObtainTime;
}
```

**Step 2: 修改背包相关方法（第 238-260 行）**

```csharp
// 修改后：使用 InventoryItemSaveData 替换旧 ItemData
public List<InventoryItemSaveData> GetInventoryItems()
{
    if (string.IsNullOrEmpty(InventoryItems))
        return new List<InventoryItemSaveData>();
    try
    {
        return JsonUtility.FromJson<ListWrapper<InventoryItemSaveData>>(InventoryItems).Items;
    }
    catch { return new List<InventoryItemSaveData>(); }
}

public void SetInventoryItems(List<InventoryItemSaveData> items)
{
    InventoryItems = JsonUtility.ToJson(new ListWrapper<InventoryItemSaveData> { Items = items });
}
```

**Step 3: 添加新的存档数据类**

```csharp
[Serializable]
public class InventoryItemSaveData
{
    public int ItemId;
    public int Count;
    public long ObtainTime;
    public int UniqueId;       // 用于区分同 ID 的不同实例（宝物等）
    public string ExtraData;   // 额外数据（JSON格式，存储宝物词条等）
}
```

### 存档数据流程

```
存档加载时：
InventoryItemSaveData → InventoryManager.AddItem(itemId, count)

存档保存时：
InventoryManager.GetAllSlots() → InventoryItemSaveData
```

**完整示例**：

```csharp
// 加载背包数据
private void LoadInventoryData(PlayerSaveData saveData)
{
    var inventoryItems = saveData.GetInventoryItems();
    foreach (var itemData in inventoryItems)
        InventoryManager.Instance.AddItem(itemData.ItemId, itemData.Count);
}

// 保存背包数据
private void SaveInventoryData(PlayerSaveData saveData)
{
    var saveDataList = new List<InventoryItemSaveData>();
    var slots = InventoryManager.Instance.GetAllSlots();
    foreach (var slot in slots)
    {
        if (!slot.IsEmpty)
        {
            saveDataList.Add(new InventoryItemSaveData
            {
                ItemId = slot.ItemId,
                Count = slot.Count,
                ObtainTime = DateTimeOffset.UtcNow.ToUnixTimeSeconds(),
                UniqueId = slot.ItemStack.Item.UniqueId
            });
        }
    }
    saveData.SetInventoryItems(saveDataList);
}
```

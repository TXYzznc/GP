## 一、问题诊断

### 问题一：右键点击没有显示菜单

**当前实现流程**：

```
InventoryItemUI.OnPointerClick()（右键）
  └─> OnItemRightClick(eventData)
      └─> ShowContextMenu(position)
          ├─ 获取 InventoryUI
          ├─ 调用 InventoryUI.GetItemContextMenu()
          └─> ItemContextMenu.ShowContextMenu()
              ├─ 获取 ItemTable 数据
              ├─ RefreshMenuOptions()
              └─ gameObject.SetActive(true)
```

**常见失败原因**：

#### 1. ItemContextMenu 未添加到 Prefab

检查 Prefab 层级，确保结构如下：
```
Canvas (CombatPreparationUI 所在的 Canvas)
├─ InventoryUI
│  ├─ Content（背包格子）
│  ├─ Details Panel（物品详情）
│  └─ ItemContextMenu ← 需要在这里
│     ├─ Button_Use
│     ├─ Button_Split
│     └─ Button_Discard
```

#### 2. 菜单按钮未初始化

`ItemContextMenu.cs` 中的三个按钮字段（m_UseBtn、m_SplitBtn、m_DiscardBtn）从未赋值。解决方案：

```csharp
protected override void OnInit()
{
    base.OnInit();
    var buttons = GetComponentsInChildren<Button>(true);
    if (buttons.Length > 0) m_UseBtn = buttons[0];
    if (buttons.Length > 1) m_SplitBtn = buttons[1];
    if (buttons.Length > 2) m_DiscardBtn = buttons[2];
    gameObject.SetActive(false);
}
```

#### 3. 菜单位置计算错误

原代码直接使用屏幕坐标赋给 RectTransform.position，需改为 Canvas 本地坐标：

```csharp
public void ShowContextMenu(ItemStack itemStack, int slotIndex, Vector2 position)
{
    // ...
    var rectTransform = GetComponent<RectTransform>();
    var canvas = GetComponentInParent<Canvas>();

    RectTransformUtility.ScreenPointToLocalPointInRectangle(
        canvas.GetComponent<RectTransform>(),
        position,
        canvas.worldCamera,
        out var localPoint
    );
    rectTransform.anchoredPosition = localPoint;
    ClampMenuPosition(rectTransform, canvas.GetComponent<RectTransform>());

    gameObject.SetActive(true);
}

private void ClampMenuPosition(RectTransform menuRT, RectTransform canvasRT)
{
    var menuSize = menuRT.sizeDelta;
    var canvasSize = canvasRT.sizeDelta;
    var pos = menuRT.anchoredPosition;

    if (pos.x + menuSize.x / 2 > canvasSize.x / 2)
        pos.x = canvasSize.x / 2 - menuSize.x / 2;

    if (pos.y - menuSize.y / 2 < -canvasSize.y / 2)
        pos.y = -canvasSize.y / 2 + menuSize.y / 2;

    menuRT.anchoredPosition = pos;
}
```

### 问题二：格子高亮不显示

`InventorySlotUI` 中缺少 `IPointerEnterHandler`/`IPointerExitHandler` 接口实现：

```csharp
public partial class InventorySlotUI : UIItemBase, IPointerEnterHandler, IPointerExitHandler
{
    public void OnPointerEnter(PointerEventData eventData)
    {
        if (varHighLightImg != null && !varHighLightImg.gameObject.activeSelf)
            varHighLightImg.gameObject.SetActive(true);
    }

    public void OnPointerExit(PointerEventData eventData)
    {
        if (varHighLightImg != null && varHighLightImg.gameObject.activeSelf)
            varHighLightImg.gameObject.SetActive(false);
    }
}
```

可选：使用 DOTween 实现淡入淡出效果：

```csharp
public void OnPointerEnter(PointerEventData eventData)
{
    if (varHighLightImg != null)
    {
        varHighLightImg.gameObject.SetActive(true);
        varHighLightImg.DOFade(1f, 0.2f);
    }
}

public void OnPointerExit(PointerEventData eventData)
{
    if (varHighLightImg != null)
        varHighLightImg.DOFade(0f, 0.2f)
            .OnComplete(() => varHighLightImg.gameObject.SetActive(false));
}
```

```
├─ Image (背景)

├─ Button_Use
│  ├─ Height: 40
│  └─ Text: "使用"
│
├─ Button_Split
│  ├─ Height: 40
│  └─ Text: "拆分"
│
└─ Button_Discard
   ├─ Height: 40
   └─ Text: "丢弃"
```

**注意**：Canvas 的 `worldCamera` 必须正确设置，否则位置转换出错。

### InventorySlotUI 高亮配置

```
InventorySlotUI
├─ Image (varBg - 背景，品质颜色)
├─ Image (varHighLightImg - 高亮效果)  ← 新增
│  ├─ Name: HighLight
│  ├─ Color: 黄色 (1, 1, 0, 0.3)
│  ├─ Raycast Target: ✗ 不勾选
│  └─ RectTransform: Anchors 拉伸，Offsets (-2, -2, 2, 2)
├─ GameObject (varLock)
└─ InventoryItemUI (子物体)
```

```csharp
// 丢弃物品
private void OnClickDiscard()
{
    if (m_CurrentItemStack == null) return;
    // TODO: 显示确认对话框
    bool success = InventoryManager.Instance.RemoveItem(
        m_CurrentItemStack.Item.ItemId, m_CurrentItemStack.Count);
    if (success)
        DebugEx.Success("ItemContextMenu", $"丢弃物品: {m_CurrentItemStack.Item.Name}");
    HideContextMenu();
}
```

## 七、常见问题排查

| 问题 | 排查方法 |
|------|---------|
| 菜单按钮为 null | 确认 ItemContextMenu 下有 3 个 Button 子物体，顺序正确 |
| 菜单位置错误 | 确认 Canvas 的 worldCamera 不为 null；检查 Anchors 设置为 (0.5, 0.5) |
| 高亮不显示 | 检查 varHighLightImg 的颜色 alpha 值不为 0；检查 Sort Order |
| 点击菜单外不关闭 | 在 InventoryUI.OnUpdate 中添加鼠标点击检测 |

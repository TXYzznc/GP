## 二、RaycastTarget 配置规则

**正确配置**：

```
InventorySlotUI (GameObject)
├─ Image (varBg)
│  └─ raycastTarget: ✓ true  ← 接收点击事件的入口
├─ Image (varHighLightImg)
│  └─ raycastTarget: ✗ false ← 不拦截点击
├─ GameObject (varLock)
│  └─ Image
│     └─ raycastTarget: ✗ false
└─ InventoryItemUI (子物体)
   ├─ Image (varItemImg)
   │  └─ raycastTarget: true/false 均可
   └─ Text (varCountText)
      └─ raycastTarget: ✗ false
```

**常见错误**：

| 错误 | 症状 |
|------|------|
| varBg.raycastTarget = false | OnPointerClick 永远不触发 |
| varHighLightImg.raycastTarget = true | 高亮层拦截点击，事件不稳定 |
| InventoryItemUI 未实现 IPointerClickHandler | 无法接收点击 |
| InventoryItemUI 是同级而非子物体 | 点击触发但无法获取格子数据 |

**快速修复脚本**：

```csharp
public void FixRaycastTargets()
{
    foreach (var slot in m_InventorySlots)
    {
        var bgImg = slot.GetComponent<Image>();
        if (bgImg != null) bgImg.raycastTarget = true;

        var highlightImg = slot.transform.Find("HighLight")?.GetComponent<Image>();
        if (highlightImg != null) highlightImg.raycastTarget = false;

        if (slot.varLock != null)
        {
            var lockImg = slot.varLock.GetComponent<Image>();
            if (lockImg != null) lockImg.raycastTarget = false;
        }
    }
}
```

## 三、脚本职责表

### InventoryItemUI.cs — 点击入口与分发

```csharp
public partial class InventoryItemUI : UIItemBase, IPointerClickHandler
{
    public void OnPointerClick(PointerEventData eventData)
    {
        if (eventData.button == PointerEventData.InputButton.Left)
            OnItemClick();
        else if (eventData.button == PointerEventData.InputButton.Right)
            OnItemRightClick(eventData);
    }

    private void ShowItemDetailPanel()
    {
        var inventoryUI = GetComponentInParent<InventoryUI>();
        if (inventoryUI != null)
            inventoryUI.ShowItemDetail(m_ItemStack);
    }

    private void ShowContextMenu(Vector2 position)
    {
        var contextMenu = GetItemContextMenu();
        if (contextMenu != null)
        {
            var slotUI = GetComponentInParent<InventorySlotUI>();
            int slotIndex = slotUI != null ? slotUI.SlotIndex : -1;
            contextMenu.ShowContextMenu(m_ItemStack, slotIndex, position);
        }
    }
}
```

**约束**：不处理物品实际使用/删除逻辑，不直接显示/隐藏菜单。

```csharp
// 显示菜单（包含位置计算和选项刷新）
public void ShowContextMenu(ItemStack itemStack, int slotIndex, Vector2 position)
{
    // 刷新选项
    RefreshMenuOptions();

    // 屏幕坐标 → Canvas 本地坐标
    RectTransformUtility.ScreenPointToLocalPointInRectangle(
        canvas.GetComponent<RectTransform>(),
        position, canvas.worldCamera, out var localPoint);
    rectTransform.anchoredPosition = localPoint;
    ClampMenuPositionToScreen(rectTransform, canvas.GetComponent<RectTransform>());

    gameObject.SetActive(true);
}

// 根据物品类型显示对应按钮
private void RefreshMenuOptions()
{
    switch (itemType)
    {
        case 0: // 消耗品 → 使用、拆分(数量>1)、丢弃
        case 1: // 装备 → 装备、丢弃
        case 2: // 宝物 → 丢弃
        case 3: // 任务道具 → 无操作
    }
}
```

## 四、重构方案（InventoryClickHandler）

> 这是一个 **待实施** 的重构提案，参考 DragHandler 的模式统一点击事件。

### 现状问题

- InventoryItemUI 既显示数据，又处理点击事件（职责过重）
- 拖拽由 InventoryDragHandler 在 SlotUI 层处理，点击却在 ItemUI 层处理（不统一）

### 目标架构

```
InventorySlotUI (Prefab)
├─ Image (varBg, raycastTarget=true)
├─ Image (varHighLightImg, raycastTarget=false)
├─ InventoryClickHandler ← 新增，参考 DragHandler 模式
│  └─ OnPointerClick() → 分发给 InventorySlotUI
├─ InventoryDragHandler ← 已有
└─ InventoryItemUI
   └─ 只负责数据显示（移除所有点击处理代码）
```

### 三个组件的改动

**① 新建 InventoryClickHandler.cs**

```csharp
// Assets/AAAGame/Scripts/UI/Components/InventoryClickHandler.cs
public class InventoryClickHandler : MonoBehaviour, IPointerClickHandler
{
    public void OnPointerClick(PointerEventData eventData)
    {
        var slot = GetComponent<InventorySlotUI>() ?? GetComponentInParent<InventorySlotUI>();
        if (slot == null) return;

        var itemUI = slot.GetItemUI();
        if (itemUI == null || !itemUI.HasItem()) return;

        if (eventData.button == PointerEventData.InputButton.Left)
            slot.OnLeftClick();
        else if (eventData.button == PointerEventData.InputButton.Right)
            slot.OnRightClick(eventData.position);
    }
}
```

**② 扩展 InventorySlotUI.cs（新增方法）**

```csharp
public void OnLeftClick()
{
    var itemStack = GetItemUI()?.GetItemStack();
    if (itemStack == null || itemStack.IsEmpty) return;
    GetComponentInParent<InventoryUI>()?.ShowItemDetail(itemStack);
}

public void OnRightClick(Vector2 mousePosition)
{
    var itemStack = GetItemUI()?.GetItemStack();
    if (itemStack == null || itemStack.IsEmpty) return;
    var contextMenu = GetComponentInParent<InventoryUI>()?.GetItemContextMenu();
    contextMenu?.ShowContextMenu(itemStack, SlotIndex, mousePosition);
}
```

**③ InventoryItemUI.cs（移除的代码）**

- 移除 `IPointerClickHandler` 接口
- 移除 `OnPointerClick()`、`OnItemClick()`、`OnItemRightClick()`
- 移除 `ShowItemDetailPanel()`、`ShowContextMenu()`、`GetItemContextMenu()`
- 保留 `SetData()`、`HasItem()`、`GetItemStack()`

### 改动范围

| 新增 | InventoryClickHandler.cs | ~100 行 |
| 扩展 | InventorySlotUI.cs | ~60 行 |

## 核心类

### CardManager (单例)
```csharp
// 获取实例
CardManager.Instance

// 主要方法
CardManager.Instance.InitializeForCombat();      // 战斗开始时初始化（随机 8 张）
CardManager.Instance.Clear();                     // 战斗结束时清理
CardManager.Instance.AddCard(cardData);           // 添加卡牌
CardManager.Instance.RemoveCard(cardId);         // 移除卡牌
CardManager.Instance.GetAvailableCards();        // 获取所有卡牌

// 事件
CardManager.Instance.OnCardAdded += (card) => { };
CardManager.Instance.OnCardRemoved += (cardId) => { };
```

### CardData
```csharp
// 卡牌数据
public class CardData
{
    public int CardId { get; }
    public string Name { get; }
    public string Description { get; }
    public int Cost { get; }
    public int EffectId { get; }
    public bool IsSelected { get; set; }

```

### ICardEffect (接口)
```csharp
public interface ICardEffect
{
    void Execute(ChessEntity caster, List<ChessEntity> targets);

```

### CardEffectExecutor (动态创建)
```csharp
// 执行卡牌效果

executor.ExecuteEffect(effectId, caster, targets);
```

### CardEffectHelper (辅助类)

## 常见操作

### 添加新卡牌效果
1. 创建新类继承 `ICardEffect`
2. 实现 `Execute()` 方法
3. 在 `CardEffectExecutor` 中注册
4. 在 `CardTable` 中配置

### 修改卡牌参数
1. 编辑 `CardTable.xlsx`
2. 运行 DataTableGenerator
3. 重新启动游戏

### 调试卡牌系统
```csharp
// 查看所有卡牌
var cards = CardManager.Instance.GetAvailableCards();
foreach (var card in cards)
{
    DebugEx.LogModule("CardDebug", $"卡牌: {card.Name} (ID={card.CardId})");
}

// 手动添加卡牌
var cardData = new CardData(1001, cardTable.GetDataRow(1001));
CardManager.Instance.AddCard(cardData);
```

## 常见问题

### Q: 卡牌为什么没有显示？
A: 检查 CardManager 是否初始化，以及 CombatUI 是否订阅了 OnCardAdded 事件

### Q: 卡牌效果没有执行？
A: 检查 CardEffectExecutor 是否正确创建，以及目标是否有效

### Q: 特效没有播放？
A: 检查 CombatVFXManager 是否初始化，以及 effectId 是否有效

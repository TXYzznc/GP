# 配置表转换工具

快速将 TXT 配置表转换为 XLSX 的标准流程。

## 📋 概述

配置表转换工具已集成到 DataTableGenerator 工作流中。不再需要独立启动 TCP 服务器。

---

## 使用方式

### 标准流程

1. **编辑配置表源文件**
   ```
   Assets/AAAGame/DataTable/ConfigTable.txt
   ```

2. **通过 DataTableGenerator 生成**
   - 在 Unity Editor 中选择：**GameFramework → DataTable → Generate**
   - 工具自动转换 TXT → XLSX → .cs 类

3. **文件自动生成到**
   ```
   Assets/AAAGame/Scripts/DataTable/ConfigTable.cs  # 生成的数据表类
   Assets/AAAGame/DataTable/ConfigTable.bytes        # 二进制数据
   ```

### 手动转换（仅需要时）

如需手动调用转换器：

```python
from Tools.COG_txt2xlsx.txt_to_xlsx_converter import TxtToXlsxConverter

converter = TxtToXlsxConverter()
converter.convert(
    txt_file_path="Assets/AAAGame/DataTable/ItemTable.txt",
    output_path="Assets/AAAGame/DataTable/ItemTable.xlsx"
)
```

**工具位置**：`.Tools/COG-txt2xlsx/txt_to_xlsx_converter.py`

## 现有工具说明

### TCP 服务器 (mcp_server_tcp.py)
- **功能**: 提供 TCP 接口的转换服务
- **端口**: localhost:9000
- **协议**: JSON-RPC over TCP
- **工具**: convert_txt_to_xlsx, convert_directory_to_xlsx, validate_txt_format

### 转换客户端 (txt_converter.py)
- **功能**: 命令行转换工具，连接 TCP 服务器
- **模式**: 单文件转换、批量转换、交互模式
- **路径处理**: 自动处理相对路径和 MCP 工作区路径

### 核心转换器 (txt_to_xlsx_converter.py)
- **功能**: 实际的 TXT 到 XLSX 转换逻辑
- **格式**: 支持 DataTableProcessor 兼容格式
- **特性**: 自动文本格式化、数组列处理

## 使用示例

### 在 SKILL 中的标准用法

```python
# 1. 启动服务器（如果未运行）
listProcesses()  # 检查是否已有服务器进程

if not server_running:
    controlPwshProcess(
        action="start",
        command="start_tcp_server.bat",
        cwd=".kiro/skills/txt-to-xlsx-converter/Tool/Server"
    )

# 2. 生成 TXT 配置表
txt_content = """#	ItemTable
#	ID		Name	Type	Price
#	int		string	int	float
#	ID编号	备注	物品名称	类型	价格
	1		金币	1	100.0
	2		银币	1	10.0"""

fsWrite("MCP工作区/配置表/ItemTable.txt", txt_content)

# 3. 转换为 XLSX
executePwsh("python .kiro/skills/txt-to-xlsx-converter/Tool/Core/txt_converter.py -f 'MCP工作区/配置表/ItemTable.txt'")
```

### 批量转换示例

```python
# 批量转换所有配置表
executePwsh("python .kiro/skills/txt-to-xlsx-converter/Tool/Core/txt_converter.py --batch")
```

## 注意事项

1. **服务器启动** - 使用前必须先启动 TCP 服务器
2. **端口占用** - 确保 9000 端口未被占用
3. **路径处理** - 转换脚本会自动处理 MCP 工作区路径
4. **编码问题** - 确保 TXT 文件使用 UTF-8 编码
5. **服务器管理** - 可使用 stop_server.bat 停止服务器

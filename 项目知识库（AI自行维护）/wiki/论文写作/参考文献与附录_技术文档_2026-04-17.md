## 附录 A：配置表设计规范

### A.1 配置表命名规范

配置表文件名采用 `[Name]Table.xlsx` 的格式，例如：
- `BuffTable.xlsx` - Buff 配置表
- `CardTable.xlsx` - 卡牌配置表
- `ItemTable.xlsx` - 物品配置表

### A.2 配置表字段规范

**A.2.1 必需字段**

每个配置表都应包含以下必需字段：
- **ID**（int）：唯一标识，主键
- **Name**（string）：名称
- **Description**（string）：描述

**A.2.2 字段命名规范**

- 使用 PascalCase 命名
- 使用英文名称
- 避免使用特殊字符
- 字段名应具有描述性

**A.2.3 字段类型**

支持的字段类型：
- `int` - 整数
- `float` - 浮点数
- `string` - 字符串
- `bool` - 布尔值
- `DateTime` - 日期时间

### A.3 配置表示例

**BuffTable 示例**

| ID | Name | Description | Type | Duration | Value |
|----|------|-------------|------|----------|-------|
| 1 | 伤害增加 | 增加伤害 20% | Buff | 10 | 0.2 |
| 2 | 防御增加 | 增加防御 30% | Buff | 15 | 0.3 |
| 3 | 眩晕 | 眩晕 2 秒 | Debuff | 2 | 0 |

### A.4 配置表生成流程

1. 在 Excel 中编写配置表
2. 运行 DataTableGenerator 工具
3. 自动生成 .cs 类和 .bytes 文件
4. 在代码中使用生成的类

### A.5 配置表使用示例

```csharp
// 获取 Buff 配置
BuffTable buffTable = BuffTableManager.GetBuffById(buffId);
if (buffTable != null)
{
    // 使用配置数据
    float duration = buffTable.Duration;
    float value = buffTable.Value;
}
```

## 附录 B：关键代码示例

### B.1 事件系统使用

```csharp
// 定义事件
public class BattleStartEvent : GameEventArgs
{
    public int EnemyId { get; set; }
}

// 发送事件
EventManager.Instance.Fire(new BattleStartEvent { EnemyId = 1 });

// 监听事件
EventManager.Instance.Subscribe<BattleStartEvent>(OnBattleStart);

// 处理事件
private void OnBattleStart(BattleStartEvent e)
{
    DebugEx.Log("Battle", $"Battle started with enemy {e.EnemyId}");
}
```

// 创建对象池
ObjectPool<ChessEntity> chessPool = new ObjectPool<ChessEntity>(

    10

// 获取对象
ChessEntity chess = chessPool.Get();

// 使用对象
chess.Initialize(data);

{

    if (resource != null)

        // 使用资源
        image.sprite = resource;
    }

## 附录 E：术语表

| 术语 | 定义 |
|------|------|
| **Buff** | 增益效果，可以提升属性或改变状态 |
| **Debuff** | 减益效果，会降低属性或改变状态 |
| **棋子** | 战斗中的单位，可以是玩家角色或敌人 |
| **卡牌** | 战斗中可以使用的策略卡牌 |
| **灵力** | 游戏中的资源，用于使用卡牌和技能 |
| **配置表** | 存储游戏数据的 Excel 表格 |
| **事件系统** | 系统间通信的机制 |
| **对象池** | 缓存对象以减少创建销毁开销 |
| **热修复** | 在不重启游戏的情况下更新代码和数据 |
| **NPR** | 非真实感渲染，用于卡通风格 |

## 📋 使用指南

### 方式1: 直接复制Mermaid代码（推荐用于Word）
```
1. 打开对应的图表文件（.md）
2. 复制```mermaid和```之间的代码
3. 粘贴到 Mermaid Live Editor（https://mermaid.live）
4. 导出为PNG/SVG后插入Word文档
```

### 方式2: 在线编辑导出（Mermaid Live Editor）
```
1. 访问 https://mermaid.live
2. 粘贴Mermaid代码
3. 调整大小和样式（可选）
4. 点击"Download"导出PNG/SVG/PDF
5. 在Word中插入导出的图片
```

### 方式3: 命令行渲染（高质量）
```bash
# 需要Node.js + Mermaid CLI
npm install -g @mermaid-js/mermaid-cli

# 渲染单个图表
mmdc -i 01_系统用例图.md -o 01_系统用例图.png -w 1400 -H 900 -s 2

# 批量渲染
for file in *.md; do
  mmdc -i "$file" -o "${file%.md}.png" -w 1400 -H 900 -s 2
done
```

### 方式4: GitHub直接预览
所有图表都可以在GitHub中直接预览（在线渲染），无需额外工具

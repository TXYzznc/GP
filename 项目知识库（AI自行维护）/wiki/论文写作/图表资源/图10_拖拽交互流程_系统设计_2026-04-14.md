# 图10：物品拖拽交互流程

**位置**: 第5章 物品系统  
**章节**: 5.2 交互设计  
**类型**: 序列图  
**用途**: 展示交互的时序关系

## Graphviz DOT 代码

```dot
digraph G {
    rankdir=TB;
    splines=orthogonal;
    nodesep=0.8;
    ranksep=1.0;
    
    // 节点定义
    User [shape=box, label="用户", style=filled, fillcolor="#E8F4F8"];
    Input [shape=box, label="输入管理器", style=filled, fillcolor="#E8F4F8"];
    Drag [shape=box, label="拖拽系统", style=filled, fillcolor="#FFE4B5"];
    UI [shape=box, label="UI 系统", style=filled, fillcolor="#E8F4F8"];
    Item [shape=box, label="物品系统", style=filled, fillcolor="#E8F4F8"];
    Anim [shape=box, label="动画系统", style=filled, fillcolor="#F0E68C"];
    
    // 拖拽开始流程
    User -> Input [label="按下鼠标"];
    Input -> Drag [label="触发拖拽开始"];
    Drag -> UI [label="获取拖拽源物品"];
    UI -> Drag [label="返回物品信息"];
    Drag -> Anim [label="启动拖拽动画"];
    Anim -> Drag [label="动画就绪"];
    
    // 拖拽进行流程
    User -> Input [label="移动鼠标"];
    Input -> Drag [label="更新拖拽位置"];
    Drag -> UI [label="更新拖拽物品位置"];
    UI -> Drag [label="检测悬停目标"];
    
    // 拖拽结束流程
    User -> Input [label="释放鼠标"];
    Input -> Drag [label="触发拖拽结束"];
    Drag -> UI [label="获取拖拽目标"];
    UI -> Drag [label="返回目标信息"];
    
    // 验证与交换流程
    Drag -> Item [label="验证交换条件"];
    Item -> Drag [label="验证通过"];
    Item -> Drag [label="验证失败"];
    
    Drag -> Item [label="执行物品交换"];
    Item -> Anim [label="播放交换动画"];
    Anim -> Item [label="动画完成"];
    Item -> Drag [label="交换成功"];
    
    Drag -> Anim [label="播放回退动画"];
    Anim -> Drag [label="动画完成"];
    
    Drag -> UI [label="更新 UI 显示"];
    Drag -> UI [label="恢复原位置"];
    
    Drag -> User [label="交互完成"];
}
```

## 说明

物品拖拽交互的完整时序：

1. **拖拽开始** - 用户按下鼠标，系统获取拖拽源物品
2. **拖拽进行** - 用户移动鼠标，系统实时更新拖拽物品位置和悬停状态
3. **拖拽结束** - 用户释放鼠标，系统获取拖拽目标
4. **验证交换** - 验证目标是否有效
5. **执行交换** - 如果有效，执行物品交换并播放动画
6. **回退处理** - 如果无效，播放回退动画并恢复原位置

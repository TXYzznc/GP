### 步骤2：提取参考论文格式

#### 2.1 使用 docx-format-replicator 技能
```
输入: 参考论文 .docx 文件
提取内容:
  - 页面设置（纸张大小、边距、方向）
  - 字体设置（正文字体、标题字体、大小）
  - 段落设置（行距、段前段后、对齐方式）
  - 样式设置（标题样式、正文样式、列表样式）
  - 页眉页脚设置
  - 目录格式
输出: 格式配置文件 thesis_format.json
```

#### 2.2 格式配置示例
```json
{
  "page": {
    "size": "A4",
    "margins": {
      "top": 2.54,
      "bottom": 2.54,
      "left": 3.17,
      "right": 3.17
    }
  },
  "font": {
    "body": {
      "name": "宋体",
      "size": 12
    },
    "heading1": {
      "name": "黑体",
      "size": 18,
      "bold": true
    }
  },
  "paragraph": {
    "lineSpacing": 1.5,
    "spaceBefore": 0,
    "spaceAfter": 0
  }
}
```

#### 3.2 Mermaid 图表转换

需要将 Mermaid 代码转换为图片：

**方式1：在线编辑器**
```
1. 访问 https://mermaid.live
2. 复制 Mermaid 代码
3. 导出为 PNG/SVG
4. 保存到本地
```

**方式2：CLI 工具**
```bash
npm install -g @mermaid-js/mermaid-cli
mmdc -i 图1_游戏整体架构.mmd -o 图1.png
```

**方式3：Docker**
```bash
docker run --rm -v $(pwd):/data minlag/mermaid-cli \
  -i /data/图1_游戏整体架构.mmd -o /data/图1.png
```

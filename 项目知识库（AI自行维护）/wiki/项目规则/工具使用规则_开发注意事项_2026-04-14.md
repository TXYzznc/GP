## 1. TXT to XLSX 配置表转换工具

### 工具简介

游戏配置表转换工具，将 TXT 格式配置表转换为 XLSX 格式，支持 AI 生成 TXT 配置表。

**工具位置**：`.kiro/skills/txt-to-xlsx-converter/`

### 使用场景

- 创建游戏配置表
- TXT 转 XLSX 转换
- 生成符合 DataTableProcessor 格式的配置文件

### 标准流程

#### 步骤 1：生成 TXT 配置表

```python
# 生成 TXT 配置表到 MCP 工作区
fsWrite("MCP工作区/配置表/TableName.txt", content)
```

**TXT 格式要求**：
```
#	TableName													
#	ID		Field1	Field2	Field3
#	int		type1	type2	type3
#	ID编号	备注	说明1	说明2	说明3
	1		value1	value2	value3
```

**关键规则**：
- Tab 分隔符（\t）
- 第 3 列固定为空
- 数据行第 1 列为空
- 元数据行第 1 列是 #
- ID 列在第 2 列

#### 步骤 2：启动 TCP 服务器

```python
# 检查服务器是否已运行
listProcesses()

# 如果服务器未运行，启动后台服务器
controlPwshProcess(
    action="start",
    command="python mcp_server_tcp.py",
    cwd=".kiro/skills/txt-to-xlsx-converter/Tool/Server"
)

# 验证服务器启动
getProcessOutput(terminalId, lines=10)
# 应看到: [INFO] MCP Server started on localhost:9000
```

#### 步骤 3：执行转换

```python
# 从项目根目录执行，使用绝对路径
controlPwshProcess(
    action="start",
    command="python .kiro/skills/txt-to-xlsx-converter/Tool/Core/txt_converter.py -f 配置表/TableName.txt"
)

# 检查结果
getProcessOutput(terminalId, lines=50)
```

### 注意事项

- 服务器启动后会保持运行，无需每次任务都重启
- 转换完成后，XLSX 文件生成到 MCP 工作区
- 不要自动复制文件到项目其他目录

## 2. DataTable 生成器

### 工具简介

从 Excel 或 TXT 文件生成 C# 数据表类和二进制数据文件。

**菜单路径**：`Tools/GameData/Generate DataTables`

### 使用方法

1. 在 `AAAGameData/DataTables/` 目录下创建或修改 Excel 文件
2. 点击菜单 `Tools/GameData/Generate DataTables`
3. 生成的代码在 `Assets/AAAGame/Scripts/DataTable/` 目录
4. 生成的二进制文件在 `Assets/AAAGame/DataTable/` 目录

### 生成内容

- C# 数据表类：`TableName.cs`
- 二进制文件：`TableName.bytes`
- 文本文件（调试用）：`TableName.txt`

### 代码模板

模板文件位置：
`Assets/AAAGame/ScriptsBuiltin/Editor/DataTableGenerator/DataTableCodeTemplate/DataTableCodeTemplate.txt`

## 3. AssetBundle 打包工具

### 工具简介

打包游戏资源为 AssetBundle，支持增量打包和热更新。

**菜单路径**：`Tools/AssetBundle/Build AssetBundles`

### 使用方法

1. 配置 AssetBundle 标签
2. 点击菜单 `Tools/AssetBundle/Build AssetBundles`
3. 打包后的文件位于 `AB/` 目录

### 打包配置

- 版本号：在 `AppBuildSettings` ScriptableObject 中配置
- 打包平台：Windows、Android、iOS
- 打包模式：完整打包、增量打包

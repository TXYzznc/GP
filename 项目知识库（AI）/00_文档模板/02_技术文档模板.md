## 概述

存档系统提供了完整的游戏存档管理功能，支持多个存档、快速保存、自动保存等。

## 快速开始

### 基本使用

```csharp
// 获取当前存档
var saveData = PlayerAccountDataManager.Instance.CurrentSaveData;
```

## 详细参考

### PlayerAccountDataManager

#### 方法: CreateNewSave

**签名**:
```csharp
public SaveData CreateNewSave(string saveName, int summonerId)
```

**说明**: 创建新的存档

**参数**:
| 参数名 | 类型 | 说明 |
|-------|------|------|
| saveName | string | 存档名称 |
| summonerId | int | 召唤师 ID |

**返回值**: 新创建的存档对象

**示例**:
```csharp
var saveData = PlayerAccountDataManager.Instance.CreateNewSave("我的冒险", 1001);
```

---

## 常见问题

### Q: 如何加载存档？
**A**: 使用 `LoadSave(saveId)` 方法

### Q: 存档数据存在哪里？
**A**: 存档保存在本地文件系统中
```
```

## 常见错误

### 错误: 存档不存在

**症状**: 加载存档时报错

**原因**: 存档 ID 错误或已删除

**解决方案**: 使用 `GetAllSaveBriefInfos()` 检查可用的存档

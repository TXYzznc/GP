## 问题分析

### 1. Unity诊断结果

通过Unity Skills API诊断发现：

- **找到12个Canvas对象**，包括：
  - WorldCanvas (ID: 49348) - 伤害飘字显示的主要Canvas
  - 多个UI Canvas：PlayerSkillUI、CombatUI、StarPhoneUI等
  - 所有Canvas的渲染模式都显示为"Unknown"（可能是API读取问题）

- **未找到Text或TextMeshProUGUI组件**（可能是因为飘字对象在对象池中未激活）

- **控制台日志显示正常的伤害处理流程**：
  ```
  [CombatVFXManager] ShowDamagePopup: pos=(-4.64, 3.33, -2.53), damage=15, type=BurnDamage
  [CombatVFXManager] ConfigurePopup: type=BurnDamage, isBuffDamage=True, damage=20.0, color=RGBA(1.000, 0.400, 0.100, 1.000), scale=0.8
  ```

### 2. 代码分析

#### CombatVFXManager.ShowDamagePopup 流程：
1. 从对象池获取飘字对象 `GetPopupFromPool()`
2. 配置飘字属性 `ConfigurePopup()`
3. 添加到活跃列表 `s_ActivePopups.Add(popup)`
4. 播放动画 `popup.Play()`

#### 潜在问题点：

**A. 多Canvas渲染问题**
- 场景中存在12个Canvas，可能有多个Canvas同时渲染伤害飘字
- WorldCanvas可能被重复创建或存在多个实例

**B. 对象池管理问题**
- `GetPopupFromPool()` 可能返回了未正确重置的对象
- 飘字对象可能没有被正确回收到对象池

**C. 动画重叠问题**
- 相同位置的伤害可能触发了多次ShowDamagePopup调用
- 飘字的随机偏移可能不够大，导致视觉重叠

**D. UI层级问题**
- 不同Canvas的sortingOrder可能导致飘字在不同层级显示
- Canvas的渲染模式设置可能有问题

## 根本原因推测

基于分析，最可能的原因是：

1. **多个WorldCanvas实例**：场景中可能存在多个WorldCanvas，每个都在显示伤害飘字
2. **Canvas渲染模式问题**：WorldCanvas的渲染模式可能设置不正确，导致重复渲染
3. **对象池状态异常**：飘字对象可能没有被正确重置，导致显示异常

## 解决方案

### 1. 检查WorldCanvas配置

需要修改 `CombatVFXManager.cs` 中的初始化逻辑，添加更严格的Canvas检查：

**修改位置：** 第 185-195 行

```csharp
// 1. 查找场景中的 WorldCanvas
var worldCanvasObjects = GameObject.FindObjectsOfType<Canvas>()
    .Where(c => c.name == WORLD_CANVAS_NAME)
    .ToArray();

if (worldCanvasObjects.Length == 0)
{
    DebugEx.ErrorModule("CombatVFXManager", $"未找到 {WORLD_CANVAS_NAME} 对象！请在场景中创建该对象。");
    return;
}
else if (worldCanvasObjects.Length > 1)
{
    DebugEx.WarningModule("CombatVFXManager", $"发现多个 {WORLD_CANVAS_NAME} 对象({worldCanvasObjects.Length}个)！这可能导致飘字重复显示。");
    
    // 使用第一个，禁用其他的
    for (int i = 1; i < worldCanvasObjects.Length; i++)
    {
        worldCanvasObjects[i].gameObject.SetActive(false);
        DebugEx.LogModule("CombatVFXManager", $"禁用重复的WorldCanvas: {worldCanvasObjects[i].name}");
    }
}

var worldCanvasGo = worldCanvasObjects[0].gameObject;
s_PopupContainer = worldCanvasGo.transform;

// 验证Canvas配置
Canvas canvas = worldCanvasGo.GetComponent<Canvas>();
if (canvas != null)
{
    DebugEx.LogModule("CombatVFXManager", 
        $"WorldCanvas配置: RenderMode={canvas.renderMode}, SortingOrder={canvas.sortingOrder}, OverrideSorting={canvas.overrideSorting}");
}
```

### 2. 增强对象池重置逻辑

需要修改 `GetPopupFromPool()` 方法，确保对象被正确重置：

**修改位置：** 第 827-850 行

```csharp
private static DamagePopupItem GetPopupFromPool()
{
    if (s_PopupPool.Count > 0)
    {
        var popup = s_PopupPool.Pop();
        
        // ⭐ 重要：确保对象被正确重置
        if (popup != null && popup.gameObject != null)
        {
            popup.gameObject.SetActive(true);
            
            // 重置所有状态
            popup.transform.localScale = Vector3.one;
            popup.transform.localPosition = Vector3.zero;
            
            // 重置CanvasGroup
            var canvasGroup = popup.GetComponent<CanvasGroup>();
            if (canvasGroup != null)
            {
                canvasGroup.alpha = 1f;
            }
            
            DebugEx.LogModule("CombatVFXManager", $"从对象池获取飘字对象，剩余池大小: {s_PopupPool.Count}");
            return popup;
        }
        else
        {
            DebugEx.WarningModule("CombatVFXManager", "对象池中的飘字对象无效，创建新对象");
        }
    }

    // 池为空或对象无效，创建新对象
    return CreatePopupInstance();
}
```

### 3. 添加飘字去重机制

在ShowDamagePopup中添加位置检查，避免在相同位置同时显示多个飘字：

**修改位置：** 第 322-356 行

```csharp
public static void ShowDamagePopup(Vector3 worldPosition, double damage, DamageType damageType = DamageType.Physical)
{
    if (!s_IsInitialized)
    {
        Debug.LogWarning("[CombatVFXManager] 尚未初始化完成，无法显示飘字");
        return;
    }

    // ⭐ 新增：检查是否在相同位置已有飘字
    const float MIN_DISTANCE = 0.5f; // 最小距离阈值
    bool hasDuplicatePosition = s_ActivePopups.Any(popup => 
        popup != null && 
        Vector3.Distance(popup.transform.position, worldPosition) < MIN_DISTANCE);
    
    if (hasDuplicatePosition)
    {
        DebugEx.WarningModule("CombatVFXManager", 
            $"位置 {worldPosition} 附近已有飘字显示，跳过重复显示。当前活跃飘字数: {s_ActivePopups.Count}");
        return;
    }

    // 从对象池获取飘字
    var popup = GetPopupFromPool();
    if (popup == null) return;

    // 配置飘字
    ConfigurePopup(popup, worldPosition, damage, damageType);

    // 添加到活跃列表
    s_ActivePopups.Add(popup);

    // 播放动画
    popup.Play(() =>
    {
        // 动画完成后回收
        s_ActivePopups.Remove(popup);
        ReturnPopupToPool(popup);
    });

    DebugEx.LogModule("CombatVFXManager", 
        $"ShowDamagePopup: pos={worldPosition}, damage={damage}, type={damageType}, 活跃飘字数: {s_ActivePopups.Count}");
}
```

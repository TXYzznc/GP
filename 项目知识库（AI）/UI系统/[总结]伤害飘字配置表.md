## 技术要点

### 配置表设计
- **伤害类型覆盖**: 包含普通伤害、暴击伤害、法术伤害、真实伤害、护盾吸收、各种元素伤害（毒、火、冰、雷）、治疗效果、经验金币获得等20种类型
- **视觉参数**: 每种类型都配置了字体大小、文字颜色、描边颜色、渐变效果、动画类型、缩放倍数、持续时间等参数
- **资源引用**: 使用资源ID方式引用特效和音效资源，符合项目规范
- **优先级系统**: 为不同伤害类型设置了显示优先级，确保重要伤害信息优先显示

### 转换工具使用
- **TCP服务器**: 成功启动了MCP TCP服务器（localhost:9000）
- **路径处理**: 解决了路径配置不匹配的问题，使用绝对路径完成转换
- **格式兼容**: 生成的XLSX文件符合DataTableProcessor格式要求
- **文件大小**: 生成的XLSX文件大小为7826字节，包含完整的配置数据

### 配置表字段说明
| 字段名 | 类型 | 说明 |
|--------|------|------|
| ID | int | 伤害类型唯一标识 |
| Name | string | 伤害类型名称 |
| FontSize | float | 字体大小 |
| TextColor | string | 文字颜色(RGBA) |
| OutlineColor | string | 描边颜色(RGBA) |
| IsGradient | bool | 是否使用渐变效果 |
| GradientStartColor | string | 渐变起始色 |
| GradientEndColor | string | 渐变结束色 |
| AnimationType | int | 动画类型ID |
| ScaleMultiplier | float | 缩放倍数 |
| Duration | float | 持续时间(秒) |
| MoveDistance | float | 移动距离(像素) |
| FadeInTime | float | 淡入时间(秒) |
| FadeOutTime | float | 淡出时间(秒) |
| EffectId | int | 特效资源ID |
| SoundId | int | 音效资源ID |
| Priority | int | 显示优先级 |

## 注意事项

### ⚠️ 关键点
1. **SetSlotContainer() 必须在 SetContainerType() 之后调用**
   ```csharp
   slot.SetContainerType(SlotContainerType.Inventory);
   slot.SetSlotContainer(m_InventorySlotContainer);  // ✅ 正确顺序
   ```

2. **每个 UI 只需一个容器组件**
   - InventoryUI 内有 1 个 InventorySlotContainer
   - WarehouseUI 内有 1 个 WarehouseSlotContainer
   - 所有背包格子共享同一个容器实例

3. **容器在 OnInit 时创建，在 CollectSlotUIs() 时使用**
   ```csharp
   OnInit()
     ├─ 创建容器组件
     └─ 其他初始化

   CollectSlotUIs()
     ├─ 创建格子 UI
     └─ 绑定容器到格子
   ```

4. **FastBarManager 与 FastBarSlotContainer 的关系**
   - FastBarManager：管理快捷栏数据（InventorySlot 列表）
   - FastBarSlotContainer：提供容器接口，调用 FastBarManager 操作
   - 两者配合使用，不需要删除 FastBarManager

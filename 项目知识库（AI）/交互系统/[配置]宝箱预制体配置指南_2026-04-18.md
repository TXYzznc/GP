# 宝箱预制体配置指南

## 概述

宝箱预制体 `TreasureChest1.prefab` 已集成交互功能，支持以下特性：
- 靠近时显示描边效果
- 交互时播放开箱动画
- 打开宝箱 UI 显示物品
- 关闭 UI 时播放关闭动画
- 未开启时循环播放特效，开启后停止

## 脚本组件

### TreasureChestInteractable
主要交互脚本，负责：
- 管理宝箱状态（Locked/Opened）
- 处理交互逻辑
- 控制动画播放
- 生成随机物品

**位置**: `Assets/AAAGame/Scripts/Game/Interact/Items/TreasureChestInteractable.cs`

### ChestEffectCycler（内部类）
特效循环器，负责：
- 循环播放特效预制体
- 管理特效的启用/禁用

**位置**: 在 TreasureChestInteractable 中作为内部类实现

## 预制体配置步骤

### 1. 添加脚本组件

在预制体根节点 `TreasureChest1` 上添加 `TreasureChestInteractable` 脚本：

```
TreasureChest1 (GameObject)
├── 添加 Component → TreasureChestInteractable
├── 添加 Component → OutlineController（如果没有）
└── ...
```

### 2. 配置 TreasureChestInteractable

在 Inspector 中设置以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| **Treasure Box Id** | 宝箱 ID（从 TreasureBoxTable 读取） | 1 |
| **Chest Level** | 宝箱等级（1-100，影响物品数量） | 50 |
| **Open Anim Trigger** | 开箱动画 Trigger 名称 | "Open" |
| **Close Anim Trigger** | 关箱动画 Trigger 名称 | "Close" |
| **Outline Color** | 描边颜色 | (1, 0.85, 0, 1) 金色 |
| **Outline Size** | 描边宽度 | 1 |

### 3. 配置 Animator

在 `Lid` 子对象上配置 Animator：

- **Animator Controller**: 需要包含 `Open` 和 `Close` 两个 Trigger
- **Animation Clips**:
  - `Open`: 开箱动画（从关闭到打开）
  - `Close`: 关箱动画（从打开到关闭）

### 4. 配置特效循环器

在 `ChestEffectCycler` 子对象上添加 `ChestEffectCycler` 脚本（内部类）：

```
ChestEffectCycler (GameObject)
├── 添加 Component → ChestEffectCycler
└── 配置参数：
    - List Of Effects: 特效预制体列表
    - Loop Time Length: 循环时长（秒）
```

**特效配置示例**:
- 添加 2 个特效预制体到列表
- 设置循环时长为 2 秒
- 每 2 秒切换一次特效

### 5. 配置 OutlineController

确保根节点有 `OutlineController` 组件，用于显示/隐藏描边。

## 运行时行为

### 状态流转

```
初始化
  ↓
Locked 状态（未开启）
  ├─ 靠近时 → 启用特效循环 + 显示描边
  ├─ 离开时 → 禁用特效循环 + 隐藏描边
  └─ 交互时 → 播放开箱动画 → 打开 UI → 转为 Opened 状态
  
Opened 状态（已开启）
  ├─ 特效循环已禁用
  └─ 交互时 → 直接打开 UI（跳过动画）
  
UI 关闭时 → 播放关箱动画
```

### 日志输出

脚本使用 `DebugEx` 输出日志，便于调试：

```
[TreasureChest] 宝箱 [1] 容器已创建
[TreasureChest] 宝箱 [1] 特效循环器已禁用
[TreasureChest] 宝箱 [1] 特效循环已启用
[TreasureChest] 打开宝箱 [1] 金色宝箱 稀有度=3
[TreasureChest] 宝箱 [1] 关闭动画已完成
```

## 配置表依赖

### TreasureBoxTable
- **TreasureBoxId**: 宝箱 ID
- **Name**: 宝箱名称
- **Rarity**: 稀有度
- **ItemGroupId**: 物品组 ID
- **ItemCountMin**: 最少物品数
- **ItemCountMax**: 最多物品数

### ItemGroupTable
- **ItemGroupId**: 物品组 ID
- **ItemIds**: 物品 ID 数组

## 常见问题

### Q: 特效不显示？
A: 检查 ChestEffectCycler 的 `List Of Effects` 是否正确配置了预制体。

### Q: 动画不播放？
A: 确保 Animator 中配置了 `Open` 和 `Close` Trigger，且动画状态机正确。

### Q: 物品生成失败？
A: 检查 TreasureBoxTable 和 ItemGroupTable 是否正确配置，以及 ItemManager 是否初始化。

### Q: 描边不显示？
A: 确保根节点有 `OutlineController` 组件，且 `Outline Color` 和 `Outline Size` 配置正确。

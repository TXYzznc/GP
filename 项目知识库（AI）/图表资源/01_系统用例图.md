### 📐 优化参数说明

| 参数 | 值 | 作用 |
|------|-----|------|
| `rankdir` | LR | 左到右布局（紧凑），适合参与者→系统的流向 |
| `splines` | orthogonal | **强制正交线条**（所有转向都是90°） |
| `nodesep` | 0.6 | 节点水平间距（紧凑） |
| `ranksep` | 0.8 | 层级垂直间距（紧凑） |

### ✨ 优化特性

1. ✅ **强制正交线条** - 所有连接线都是 90° 转向（完全直角）
2. ✅ **最大空间利用** - 紧凑的间距参数
3. ✅ **清晰的分组** - 5个 cluster 清晰显示系统结构
4. ✅ **专业样式** - 不同系统使用不同颜色区分
5. ✅ **边标签** - 所有关系都有清晰的说明文字

### 🎯 使用方法

1. 复制上面的 DOT 代码
2. 访问 https://edotor.net/
3. 粘贴到编辑器 → **自动正交渲染**
4. 下载 PNG 或 SVG

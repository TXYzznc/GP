# 棋子系统类图

## Graphviz DOT 格式（优化正交线条版）

**适用于**: https://edotor.net/

```dot
digraph ChessSystem {
    rankdir=LR;              // 左到右布局（类层级流）
    splines=orthogonal;      // 强制正交线条（90度转向）
    nodesep=0.8;             // 节点间距（紧凑）
    ranksep=1.0;             // 层级间距（平衡）
    layout=dot;
    
    // 全局节点样式
    node [shape=box, style="rounded,filled", fontname="微软雅黑", fontsize=9];
    edge [arrowsize=1.0, penwidth=1.5];
    
    // ===== 第一层：核心实体 =====
    ChessEntity [label="ChessEntity\n\n+ChessAI m_AI\n+CombatController m_Combat\n+IChessSkill m_NormalAttack\n+Transform m_Transform\n\n+Initialize()\n+Tick()\n+TakeDamage(d, isCritical)\n+Die()", fillcolor="#e3f2fd"];
    ChessAI [label="ChessAI\n\n+ChessEntity m_Entity\n+bool m_IsInCombat\n\n+FindTarget()\n+Attack(target)\n+Tick()", fillcolor="#e3f2fd"];
    CombatController [label="CombatController\n\n+ChessEntity m_Entity\n+IChessSkill m_CurrentSkill\n\n+OnAttackExecute(skillId)\n+OnSkillExecute(skillId)", fillcolor="#e3f2fd"];
    
    // ===== 第二层：接口和工厂 =====
    IChessSkill [label="<<interface>>\nIChessSkill\n\n+ExecuteAttack(caster, target)*\n+CalculateDamage(caster)*\n+GetSkillConfig()*", fillcolor="#fff3e0"];
    IHitDetector [label="<<interface>>\nIHitDetector\n\n+Execute(context)*", fillcolor="#fff3e0"];
    HitDetectorFactory [label="<<static>>\nHitDetectorFactory\n\n+GetDetector(type):\n  IHitDetector", fillcolor="#f1f8e9"];
    
    // ===== 第三层：抽象类 =====
    ChessNormalAttackBase [label="<<abstract>>\nChessNormalAttackBase\n\n+SkillConfig m_Config\n\n+ExecuteAttack(*)\n+CalculateDamage(*)", fillcolor="#bbdefb"];
    HitDetectorBase [label="<<abstract>>\nHitDetectorBase\n\n+ApplyDamage(context)\n+ApplyBuffsOnHit()", fillcolor="#bbdefb"];
    
    // ===== 第四层：具体实现 =====
    InstantDetector [label="InstantDetector\n\n+Execute(context)", fillcolor="#c8e6c9"];
    MeleeDetector [label="MeleeDetector\n\n+Execute(context)", fillcolor="#c8e6c9"];
    ProjectileDetector [label="ProjectileDetector\n\n+Execute(context)", fillcolor="#c8e6c9"];
    
    // ===== 数据类 =====
    HitContext [label="HitContext\n\n+ChessEntity Attacker\n+ChessEntity LockedTarget\n+double BaseDamage\n+bool IsCritical\n+SkillConfig SkillConfig", fillcolor="#fce4ec"];
    
    // ===== 关系定义 =====
    // 核心实体关系
    ChessEntity -> ChessAI [label="owns"];
    ChessEntity -> CombatController [label="owns"];
    ChessEntity -> IChessSkill [label="has"];
    CombatController -> IChessSkill [label="executes"];
    ChessAI -> ChessEntity [label="targets", constraint=false];
    
    // 技能体系
    ChessNormalAttackBase -> IChessSkill [style=dashed, label="implements"];
    ChessNormalAttackBase -> HitDetectorFactory [label="uses"];
    
    // 命中检测体系
    HitDetectorFactory -> IHitDetector [label="creates"];
    HitDetectorBase -> IHitDetector [style=dashed, label="implements"];
    InstantDetector -> HitDetectorBase [style=dashed, label="extends"];
    MeleeDetector -> HitDetectorBase [style=dashed, label="extends"];
    ProjectileDetector -> HitDetectorBase [style=dashed, label="extends"];
    HitDetectorBase -> HitContext [label="uses"];
}
```

### 📊 类关系说明

| 关系 | 符号 | 含义 |
|------|------|------|
| 关联 | → | 一个类使用或拥有另一个类 |
| 实现 | ⋯→ (虚线) | 类实现接口 |
| 继承 | ⋯→ (虚线) | 子类继承父类 |

### 📐 优化参数说明

| 参数 | 值 | 作用 |
|------|-----|------|
| `rankdir` | LR | 左到右布局（类层级流）|
| `splines` | orthogonal | **强制正交线条**（90°转向） |
| `nodesep` | 0.8 | 节点间距（紧凑） |
| `ranksep` | 1.0 | 层级间距（平衡可读性） |

### 🎨 颜色编码

- **蓝色**：核心实体类
- **橙色**：接口定义
- **浅蓝**：抽象基类
- **绿色**：具体实现类
- **粉色**：数据结构类
- **浅绿**：工厂类

### ✨ 优化特性

1. ✅ **强制正交线条** - 所有箭头完全直角
2. ✅ **清晰的类层级** - 4层垂直分布
3. ✅ **颜色分类** - 快速识别类型
4. ✅ **空间最优化** - 紧凑的节点排列
5. ✅ **关系清晰** - 虚线表示继承/实现

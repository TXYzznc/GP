## 🎯 选择建议

### 使用高分辨率 PNG（推荐用于 Word）

**优点：**
- ✅ 分辨率高（2000×1200），内容清晰
- ✅ 直接插入 Word 文档
- ✅ 兼容性好，所有平台都支持
- ✅ 文件大小合理（平均 300KB）

**缺点：**
- ❌ 放大后会失真（但通常不需要放大）

**使用场景：**
- 📄 Word/PDF 论文
- 🖨️ 打印输出
- 📊 演示文稿

**文件夹：** `高分辨率/`

---

### 使用 SVG 矢量图（推荐用于网页/演示）

**优点：**
- ✅ 矢量格式，无限放大不失真
- ✅ 文件小（2.0MB 总计）
- ✅ 可在网页中显示
- ✅ 可用矢量编辑软件编辑（如 Illustrator、Inkscape）

**缺点：**
- ❌ Word 支持有限，可能显示不佳
- ❌ 需要额外工具转换为其他格式

**使用场景：**
- 🌐 网页展示
- 🎨 矢量编辑和修改
- 📱 响应式设计
- 🖼️ 高端印刷排版

**文件夹：** `矢量图/`

## 💻 如何使用

### 方案 1：PNG 插入 Word（最简单）

1. **打开 Word 文档**
   ```
   论文初版-数字2201-20226980-唐鑫阳.docx
   ```

2. **定位到插入位置**
   - 参考 `INDEX.md` 中的"论文位置"列
   - 例如：图3-1 应该插入在"第3章 需求分析"中

3. **插入图片**
   - 菜单：`插入` → `图片` → `本此设备`
   - 选择：`高分辨率/[图表名].png`

4. **调整大小**（参考以下建议）
   ```
   小型图表（用例、状态机）：7-8cm
   中型图表（类图、流程图）：10-12cm  
   大型图表（架构图）：14-16cm
   ```

5. **添加标题和说明**
   ```
   图3-1  系统用例图
   
   [图片]
   
   图3-1说明：本系统主要包括...
   ```

### 方案 2：SVG 在线编辑

1. **用支持 SVG 的编辑器打开**
   - Inkscape（免费）：https://inkscape.org/zh-hans/
   - Adobe Illustrator（付费）

2. **修改和导出**
   - 修改配色、字体等
   - 导出为 PNG、PDF 等格式

### 方案 3：SVG 转 PNG（高质量）

如果需要更高的 DPI 或其他格式，可用以下工具：

```bash
# 使用 ImageMagick 转换 SVG 到 PNG（300 DPI）
convert -density 300 矢量图/图名.svg -resize 2000x1200 输出.png

# 或使用 Inkscape 命令行
inkscape 矢量图/图名.svg --export-png=输出.png --export-dpi=300
```

## 🔧 分辨率参数说明

### 高分辨率 PNG 生成参数

```bash
mmdc -i input.md \
     -o output.png \
     -w 2000    # 宽度（像素）
     -H 1200    # 高度（像素）
     -s 3       # 缩放因子（3倍清晰度）
```

**分辨率对比：**
| 版本 | 分辨率 | 缩放 | 用途 |
|------|--------|------|------|
| 标准版 | 1400×900 | 2× | 屏幕预览 |
| **高分辨率** | **2000×1200** | **3×** | **论文印刷** |
| 超高清 | 2800×1680 | 4× | 超大海报 |

如需其他分辨率，提供以下参数组合：
- **A4 论文打印**：`-w 2000 -H 1200 -s 3`（推荐）
- **16:9 演示**：`-w 2560 -H 1440 -s 3`
- **高清屏幕**：`-w 1920 -H 1080 -s 2`

## ✨ 图表质量检查

### 清晰度测试

PNG 图片插入 Word 后，检查以下内容：
- ✅ 文字清晰可读
- ✅ 线条平滑，无锯齿
- ✅ 颜色准确，无模糊
- ✅ 箭头连接清晰

### 常见问题解决

**问题：插入后仍然模糊**
- 解决：使用 SVG 版本（矢量，永不模糊）
- 或者：调整 Word 显示缩放到 100%

**问题：颜色显示不对**
- 解决：检查 Word 色彩管理设置
- 或者：重新导出 PNG（调整输出 DPI）

**问题：文件太大**
- 解决：使用 SVG 版本（2.0MB 总计）
- 或者：用在线工具压缩 PNG（tinypng.com）

## 📤 批量导出为其他格式

### 转换所有 SVG 为 PNG（指定 DPI）

```bash
cd 矢量图/
for file in *.svg; do
  convert -density 300 "$file" "../高分辨率/DPI300/${file%.svg}.png"
done
```

### 转换所有 SVG 为 PDF

```bash
cd 矢量图/
for file in *.svg; do
  convert "$file" "../论文PDF/${file%.svg}.pdf"
done
```

# 图15：配置表结构概览

**位置**: 第3章 系统架构  
**章节**: 3.5 数据设计  
**类型**: 表格  
**用途**: 说明配置表的设计

## Graphviz DOT 代码

```dot
digraph G {
    rankdir=LR;
    splines=orthogonal;
    nodesep=0.7;
    ranksep=1.0;
    
    // 配置表类定义
    BuffTable [shape=record, label="{BuffTable|id: int|name: string|duration: int|value: float|effect_type: string|priority: int}", style=filled, fillcolor="#FFE4B5"];
    
    SkillTable [shape=record, label="{SkillTable|id: int|name: string|cost: int|damage: float|cooldown: int|description: string}", style=filled, fillcolor="#F0E68C"];
    
    CardTable [shape=record, label="{CardTable|id: int|name: string|rarity: int|cost: int|effect: string|description: string}", style=filled, fillcolor="#DDA0DD"];
    
    ItemTable [shape=record, label="{ItemTable|id: int|name: string|type: int|rarity: int|max_stack: int|description: string}", style=filled, fillcolor="#87CEEB"];
    
    EnemyTable [shape=record, label="{EnemyTable|id: int|name: string|hp: int|attack: int|defense: int|level: int}", style=filled, fillcolor="#FF6347"];
    
    SynergyTable [shape=record, label="{SynergyTable|id: int|name: string|bonus_type: int|bonus_value: float|description: string}", style=filled, fillcolor="#98FB98"];
}
```

## 说明

主要配置表的字段结构和用途：

| 表名 | 主要字段 | 用途 |
|------|---------|------|
| **BuffTable** | id, name, duration, value, effect_type | 定义 Buff 属性和效果 |
| **SkillTable** | id, name, cost, damage, cooldown | 定义技能属性和消耗 |
| **CardTable** | id, name, rarity, cost, effect | 定义卡牌属性和效果 |
| **ItemTable** | id, name, type, rarity, max_stack | 定义物品属性和堆叠规则 |
| **EnemyTable** | id, name, hp, attack, defense, level | 定义敌人属性 |
| **SynergyTable** | id, name, bonus_type, bonus_value | 定义协同效果 |

# 图5：物品系统架构

```mermaid
classDiagram
    class ItemSystem {
        +物品管理
        +容器管理
        +拖拽交互
        +物品效果
        +持久化
    }
    
    class ItemManager {
        +创建物品
        +销毁物品
        +查询物品
        +属性修改
    }
    
    class ContainerManager {
        +背包
        +装备栏
        +快捷栏
        +银行
    }
    
    class DragSystem {
        +拖拽验证
        +快速操作
        +预览反馈
        +撤销机制
    }
    
    class EffectSystem {
        +消耗品效果
        +装备效果
        +条件判定
        +参数化
    }
    
    class PersistenceSystem {
        +序列化
        +反序列化
        +版本管理
        +备份恢复
    }
    
    ItemSystem --> ItemManager
    ItemSystem --> ContainerManager
    ItemSystem --> DragSystem
    ItemSystem --> EffectSystem
    ItemSystem --> PersistenceSystem
```

## 说明

物品系统由多个子模块组成：

- **物品管理**：管理物品的生命周期
- **容器管理**：管理物品的存储空间
- **拖拽交互**：处理玩家的拖拽操作
- **物品效果**：执行物品的效果
- **持久化**：保存和加载物品数据

这些模块共同提供了完整的物品管理功能。

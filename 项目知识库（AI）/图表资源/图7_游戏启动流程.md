# 图7：游戏启动流程

**位置**: 第1章 绪论 / 第3章 系统架构  
**章节**: 1.4 技术方案 或 3.4 启动流程  
**类型**: 流程图  
**用途**: 说明游戏的初始化过程

## Mermaid 代码

```dot
digraph {
    rankdir=TB;
    nodesep=0.8;
    ranksep=1.0;
    splines=orthogonal;
    
    Start [label="游戏启动", style="filled", fillcolor="#90EE90", shape="ellipse"];
    Init [label="初始化 GameFramework", style="filled", fillcolor="#87CEEB", shape="box"];
    LoadConfig [label="加载配置表", style="filled", fillcolor="#87CEEB", shape="box"];
    InitProcedure [label="初始化 Procedure 系统", style="filled", fillcolor="#87CEEB", shape="box"];
    InitFSM [label="初始化 FSM 系统", style="filled", fillcolor="#87CEEB", shape="box"];
    InitEvent [label="初始化事件系统", style="filled", fillcolor="#87CEEB", shape="box"];
    InitEntity [label="初始化实体系统", style="filled", fillcolor="#87CEEB", shape="box"];
    InitUI [label="初始化 UI 系统", style="filled", fillcolor="#87CEEB", shape="box"];
    InitResource [label="初始化资源系统", style="filled", fillcolor="#87CEEB", shape="box"];
    LoadScene [label="加载启动场景", style="filled", fillcolor="#FFD700", shape="box"];
    EnterMenu [label="进入菜单?", style="filled", fillcolor="#87CEEB", shape="diamond"];
    MenuProcedure [label="执行菜单流程", style="filled", fillcolor="#DDA0DD", shape="box"];
    GameProcedure [label="执行游戏流程", style="filled", fillcolor="#DDA0DD", shape="box"];
    Ready1 [label="游戏就绪", style="filled", fillcolor="#FFD700", shape="box"];
    Ready2 [label="游戏就绪", style="filled", fillcolor="#FFD700", shape="box"];
    End [label="启动完成", style="filled", fillcolor="#FFB6C6", shape="ellipse"];
    
    Start -> Init;
    Init -> LoadConfig;
    LoadConfig -> InitProcedure;
    InitProcedure -> InitFSM;
    InitFSM -> InitEvent;
    InitEvent -> InitEntity;
    InitEntity -> InitUI;
    InitUI -> InitResource;
    InitResource -> LoadScene;
    LoadScene -> EnterMenu;
    EnterMenu -> MenuProcedure [label="是"];
    EnterMenu -> GameProcedure [label="否"];
    MenuProcedure -> Ready1;
    GameProcedure -> Ready2;
    Ready1 -> End;
    Ready2 -> End;
}
```

## 说明

游戏启动流程包括以下关键步骤：

1. **框架初始化** - 初始化 GameFramework 的各个系统
2. **配置加载** - 加载所有配置表数据
3. **系统初始化** - 依次初始化 Procedure、FSM、Event、Entity、UI、Resource 系统
4. **场景加载** - 加载启动场景
5. **流程选择** - 根据游戏状态进入菜单或游戏流程
6. **就绪状态** - 游戏完全初始化，可以开始运行

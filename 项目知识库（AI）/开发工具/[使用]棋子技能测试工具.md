# 棋子技能测试工具使用指南

**创建时间**: 2026-04-12  
**工具位置**: `Assets/Editor/unity开发集成工具箱/Panels/ChessSkillTestPanel.cs`  
**菜单路径**: `工具 > 测试工具 > 棋子技能测试器`

## 功能概述

棋子技能测试工具是一个Unity Editor工具，用于在运行时测试棋子的技能效果、动画和特效。支持手动触发技能并实时修改技能参数。

## 主要功能

### 1. 棋子选择
- **目标棋子**: 选择要测试的棋子实体（ChessEntity）
- **索敌目标**: 可选，指定技能的目标对象
- 显示棋子的基本信息（名称、阵营、HP、MP）

### 2. 技能触发
提供三个按钮手动触发技能：
- **普攻**: 触发棋子的普通攻击
- **技能一**: 触发棋子的第一个主动技能
- **大招**: 触发棋子的大招/技能二

**注意事项**:
- 必须在运行模式（Play Mode）下使用
- 棋子不会自动行动，需要手动点击按钮触发
- 如果没有指定索敌目标，普攻会自动查找最近的敌人

### 3. 技能参数编辑
支持运行时查看和修改技能的所有配置参数。

#### 从配置表加载
- 输入技能ID
- 点击"加载配置"按钮
- 从SummonChessSkillTable配置表中读取该技能的所有参数
- 必须在运行模式下使用

#### 参数类别

#### 基础信息
- 技能名称
- 技能类型
- 技能描述

#### 伤害参数
- 伤害类型（0=无伤害, 1=物理, 2=魔法, 3=真实）
- 伤害系数（基于攻击力/法强的百分比）
- 基础固定伤害值
- 触发次数（多段伤害）

#### 命中参数
- 命中类型（0=瞬发, 1=近战, 2=投射物, 3=AOE, 4=射线）
- 施法/生效范围
- AOE半径（0=单体）
- 穿透数量

#### 投射物参数（仅命中类型=2时显示）
- 投射物预制体
- 投射物速度

#### 消耗和冷却
- 法力消耗
- 法力回复
- 冷却时间（秒）
- 持续时间（引导类技能）

#### Buff参数
- Buff触发类型（0=执行时, 1=命中时, 2=条件触发）
- 附加Buff列表（对目标）
- 自身Buff列表（对自己）

#### 特效参数
- 技能特效预制体
- 特效生成高度
- 受击特效预制体
- 技能图标

## 使用流程

### 步骤1: 打开工具
1. 在Unity编辑器中，点击菜单 `工具 > 测试工具 > 棋子技能测试器`
2. 工具窗口会打开，最小尺寸为 500x700

### 步骤2: 进入Play Mode
1. 点击Unity编辑器的Play按钮进入运行模式
2. 确保场景中有已初始化的棋子实体

### 步骤3: 选择测试对象
1. 在"目标棋子"字段中，从场景中拖入或选择一个ChessEntity对象
2. （可选）在"索敌目标"字段中选择一个敌方棋子作为技能目标
3. 工具会自动加载该棋子的技能配置和技能ID

### 步骤3.5: 通过技能ID加载配置（可选）
如果想测试其他技能配置：
1. 切换到对应的技能Tab（普攻/技能一/大招）
2. 在"从配置表加载"区域输入技能ID
3. 点击"加载配置"按钮
4. 工具会从SummonChessSkillTable配置表中读取该技能的所有参数

### 步骤4: 测试技能
1. 点击对应的技能按钮（普攻/技能一/大招）
2. 观察场景中的技能效果、动画和特效
3. 查看技能状态信息（是否可用、冷却中等）

### 步骤5: 调整参数（可选）
1. 切换到对应的技能Tab（普攻/技能一/大招）
2. 在参数编辑区域修改技能参数
3. 再次触发技能查看效果变化

**注意**: 当前版本的参数修改仅用于查看配置，实际修改配置表参数的功能需要进一步实现。

## 技术实现

### 核心组件
- **ChessSkillTestPanel**: 主面板类，实现IToolHubPanel接口
- **SkillRuntimeParams**: 技能运行时参数数据结构
- **ChessSkillTestWindow**: 独立窗口包装器

### 技能触发机制
工具通过调用以下接口触发技能：
- `ChessCombatController.TriggerAttackFromAI(target)` - 触发普攻
- `ChessCombatController.TriggerSkill1FromAI()` - 触发技能一
- `ChessCombatController.TriggerSkill2FromAI()` - 触发大招

### 自动索敌
如果没有指定索敌目标，普攻会自动查找：
1. 遍历场景中所有ChessEntity
2. 过滤掉同阵营和已死亡的棋子
3. 选择距离最近的敌人作为目标

## 配置表字段映射

工具中的参数对应 `SummonChessSkillTable` 配置表的字段：

| 工具参数 | 配置表字段 | 说明 |
|---------|-----------|------|
| 技能名称 | Name | 技能显示名称 |
| 技能类型 | SkillType | 1=被动, 2=普攻效果, 3=主动技能, 4=大招 |
| 伤害类型 | DamageType | 0=无伤害, 1=物理, 2=魔法, 3=真实 |
| 伤害系数 | DamageCoeff | 基于攻击力/法强的百分比 |
| 命中类型 | EffectHitType | 0=瞬发, 1=近战, 2=投射物, 3=AOE, 4=射线 |
| 投射物预制体ID | ProjectilePrefabId | 资源ID（工具中转为对象引用） |
| Buff触发类型 | BuffTriggerType | 0=执行时, 1=命中时, 2=条件触发 |
| 基础伤害 | BaseDamage | 固定伤害值 |
| 法力消耗 | MpCost | 释放技能消耗的法力 |
| 法力回复 | MpRestore | 释放技能回复的法力 |
| 冷却时间 | Cooldown | 技能冷却时间（秒） |
| 施法范围 | CastRange | 技能生效范围 |
| AOE半径 | AreaRadius | 范围伤害半径（0=单体） |
| 持续时间 | Duration | 引导类技能的持续时间 |
| 触发次数 | HitCount | 多段伤害的触发次数 |
| 穿透数量 | PenetrationCount | 投射物可穿透的敌人数量 |
| 附加Buff | BuffIds | 对目标附加的Buff ID数组 |
| 自身Buff | SelfBuffIds | 对自己附加的Buff ID数组 |
| 投射物速度 | ProjectileSpeed | 投射物移动速度 |
| 技能图标ID | IconId | 图标资源ID |
| 技能特效ID | EffectId | 特效资源ID |
| 特效高度 | EffectSpawnHeight | 特效生成的相对高度 |
| 受击特效ID | HitEffectId | 受击特效资源ID |

## 常见问题

### Q: 为什么按钮是灰色的？
A: 可能的原因：
1. 没有进入Play Mode
2. 目标棋子没有对应的技能（例如某些棋子可能没有技能一）
3. 棋子的战斗控制器未初始化

### Q: 点击按钮后没有反应？
A: 检查以下几点：
1. 确保棋子已正确初始化（调用了Initialize方法）
2. 检查技能是否在冷却中
3. 对于普攻，确保有可攻击的目标
4. 查看Console是否有错误日志

### Q: 如何测试不同的技能配置？
A: 有两种方式：
1. 选择不同的棋子实体，工具会自动加载该棋子的技能配置
2. 使用"从配置表加载"功能，输入任意技能ID并加载，可以测试配置表中的任何技能

### Q: 技能ID在哪里查看？
A: 
1. 打开 `Assets/AAAGame/DataTable/SummonChessSkillTable.txt` 配置表
2. 第一列就是技能ID
3. 例如：11=后羿被动，12=后羿普攻，13=神力，14=日陨
A: 
1. 选择目标棋子后，工具会自动加载配置
2. 切换到对应的技能Tab查看所有参数
3. 参数来自 `SummonChessSkillTable` 配置表

### Q: 修改参数后为什么没有生效？
A: 当前版本的参数编辑器主要用于查看配置，实际修改配置表数据的功能需要：
1. 将修改后的参数应用到技能实例
2. 或者直接修改配置表并重新加载

## 🐛 Bug 修复清单

### Bug #1: 仓库内交换导致数据丢失
- [x] **定位**: WarehouseSlotContainerImpl.cs Line 63-66
- [x] **原因**: SwapSlots 已完成交换，不应再调用 RemoveItem
- [x] **修复**: 添加条件 `targetContainer is not WarehouseSlotContainerImpl`
- [x] **验证**: 仓库→仓库交换不再丢失物品

### Bug #2: 快捷栏内交换导致数据丢失
- [x] **定位**: FastBarSlotContainerImpl.cs Line 52-55
- [x] **原因**: SwapSlots 已完成交换，不应再调用 ClearSlot
- [x] **修复**: 添加条件 `targetContainer is not FastBarSlotContainerImpl`
- [x] **验证**: 快捷栏→快捷栏交换不再丢失物品

### Bug #3: 快捷栏与仓库的不合理交互
- [x] **定位**:
  - FastBarSlotContainerImpl.cs Line 25-29 (CanInteractWith)
  - WarehouseSlotContainerImpl.cs Line 36-40 (CanInteractWith)
- [x] **原因**: 快捷栏应是背包的快速访问栏，不应直接与仓库交互
- [x] **修复**: 调整规则，禁止快捷栏↔仓库直接拖拽
- [x] **验证**: 规则生效，相关拖拽被阻止

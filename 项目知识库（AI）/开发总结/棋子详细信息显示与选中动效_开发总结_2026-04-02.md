# 棋子详细信息显示与选中动效 — 修正总结

**完成时间**: 2026-04-02
**修正内容**: DetailInfoUI 支持准备阶段显示 + 选中动效实现

---

## 核心问题修复

### 问题 1：信息显示不正确 ✅
- **原因**：准备阶段选中时，棋子未部署（Entity=null），无法显示信息
- **解决**：新增 SetChessConfig(config, globalState) 方法
  - 接收 SummonChessConfig 和 GlobalChessState
  - 显示完整的棋子信息（名称、星级、属性、人口消耗等）

### 问题 2：缺少选中动效 ✅
- **原因**：ChessItemUI 选中时没有视觉反馈
- **解决**：添加 PlayChessSelectAnimation / PlayChessDeselectAnimation
  - 向上移动 20 像素（SELECTED_OFFSET）
  - 播放脉冲动画（1.0 → 1.1 → 1.0）
  - 参考 CardSlotItem 的实现

## 修改详情

### DetailInfoUI.cs

#### 新增字段
```csharp
private SummonChessConfig m_ChessConfig;
private GlobalChessState m_GlobalState;
private int m_CurrentMode = 0; // 0=卡牌, 1=棋子实体, 2=棋子配置
```

#### 新增方法
```csharp
/// <summary>设置棋子配置数据（准备阶段）</summary>
public void SetChessConfig(SummonChessConfig config, GlobalChessState globalState)

/// <summary>刷新棋子配置UI显示（准备阶段，不含Buff）</summary>
private void RefreshChessConfigUI()

/// <summary>显示棋子信息（统一处理）</summary>
private void DisplayChessInfo(SummonChessConfig config, double currentHp, double maxHp)
```

#### 修改的方法
- `RefreshUI()` - 添加 Mode 2 支持
- `RefreshChessUnitUI()` - 重构为调用 DisplayChessInfo（复用逻辑）
- `SetChessUnitData()` - 添加 m_ChessConfig = null 重置

#### 显示信息结构

| 部分 | 内容 | 来源 |
|------|------|------|
| **标题** | 棋子名称 + 星级（★） | config.Name + config.StarLevel |
| **描述** | 棋子描述文本 | config.Description |
| **其他** | 血量、法力、攻击、护甲、人口消耗 | config + globalState |
| **Buff** | 身上的 Buff（只在战斗时） | entity.BuffManager（隐藏） |

## 数据流程

### 准备阶段（未部署）
```
点击棋子卡片
  ↓
SelectChess(instanceId)
  ├─ ChessDataManager.TryGetConfig(chessId) → SummonChessConfig
  ├─ GlobalChessManager.GetChessState(chessId) → GlobalChessState
  └─ detailUI.SetChessConfig(config, globalState)
       ├─ m_CurrentMode = 2
       └─ RefreshChessConfigUI()
            └─ DisplayChessInfo(config, currentHp, maxHp)
                 ├─ 显示标题：名称 + 星级
                 ├─ 显示描述：棋子描述
                 └─ 显示其他：属性 + 人口消耗
  ↓
播放选中动效
  ├─ 向上移动 20px（0.3秒）
  └─ 脉冲动画（1.0 → 1.1 → 1.0）
```

### 战斗阶段（已部署）
```
点击棋子对象（ChessSelectionManager）
  ↓
detailUI.SetChessUnitData(entity)
  └─ m_CurrentMode = 1
       └─ RefreshChessUnitUI()
            ├─ DisplayChessInfo(config, currentHp, maxHp)
            └─ RefreshAllBuffs()
                 └─ 显示棋子身上的 Buff
```

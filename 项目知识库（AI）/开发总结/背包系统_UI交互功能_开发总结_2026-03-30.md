## 技术要点

### 事件系统集成
- 使用 EventTrigger 处理右键点击
- 通过 PointerEventData 获取鼠标位置
- 支持左键（详情面板）和右键（上下文菜单）

### 数据表访问
- 从 ItemTable 读取物品配置
- 使用 RarityColorHelper 获取颜色
- 缓存数据表查询结果

### UI 组件通信
- InventoryItemUI → InventoryUI → ItemDetailPanel/ItemContextMenu
- 通过 GetComponentInParent 获取父级组件
- 支持动态查找和缓存

### 物品类型处理
- 根据 ItemTable.Type 字段判断物品类型
- 动态显示/隐藏菜单选项
- 支持可堆叠物品的拆分

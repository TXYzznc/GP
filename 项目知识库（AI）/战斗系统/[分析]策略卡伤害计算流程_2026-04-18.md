#### DamageWithCoefficientApplier（系数伤害）
```csharp
public class DamageWithCoefficientApplier : ICardEffectApplier
{
    private ChessEntity m_CasterChess;
    
    public void ApplyEffect(List<ChessEntity> targets, CardData cardData)
    {
        // 伤害 = BaseDamage + DamageCoeff × 施法者攻击力
        float casterAtk = (float)(m_CasterChess?.Attribute?.AtkDamage ?? 0d);
        float damage = cardData.TableRow.BaseDamage + 
                      cardData.TableRow.DamageCoeff * casterAtk;
        
        foreach (var target in targets)
        {
            CardEffectHelper.DealDamage(target, damage, damageType);
        }
    }
}
```

**使用卡牌**:
- 1005 (暗影突袭) - 物理伤害，系数2.5倍
- 1018 (吸血之刃) - 物理伤害，系数2倍

**公式**:
```
最终伤害 = BaseDamage + DamageCoeff × 施法者攻击力
```

### 3️⃣ ChessAttribute.cs - TakeDamage()
**位置**: `Assets/AAAGame/Scripts/Game/SummonChess/Component/ChessAttribute.cs`

```csharp
public void TakeDamage(double damage, bool isMagic = false, 
                       bool isTrueDamage = false, bool isCritical = false,
                       CombatVFXManager.DamageType damageType = CombatVFXManager.DamageType.Physical)
{
    if (damage <= 0) return;
    
    // 第一步：计算实际伤害（考虑防御/魔抗）
    double actualDamage;
    if (isTrueDamage)
    {
        actualDamage = damage;  // 真实伤害忽略防御
    }
    else
    {
        actualDamage = isMagic ? 
            CalculateMagicDamage(damage) : 
            CalculatePhysicalDamage(damage);
    }
    
    // 第二步：应用伤害减免Buff
    if (m_DamageTakenMultiplier != 1.0)
    {
        actualDamage *= m_DamageTakenMultiplier;
    }
    
    // 第三步：护盾吸收
    if (m_Shield > 0)
    {
        double shieldAbsorb = Math.Min(m_Shield, actualDamage);
        ModifyShield(-shieldAbsorb);
        actualDamage -= shieldAbsorb;
        
        if (actualDamage <= 0)
        {
            OnDamageTaken?.Invoke(0, isMagic);
            return;
        }
    }
    
    // 第四步：扣血
    ModifyHp(-actualDamage);
    
    // 第五步：显示伤害飘字
    CombatVFXManager.ShowDamagePopup(position, actualDamage, autoType);
    
    // 第六步：触发伤害事件
    OnDamageTaken?.Invoke(actualDamage, isMagic);
}
```

### 4️⃣ ChessAttribute.cs - CalculatePhysicalDamage()

```csharp
public double CalculatePhysicalDamage(double baseDamage)
{
    // 公式：实际伤害 = 基础伤害 × (100 / (100 + 护甲))
    double damageReduction = 100.0 / (100.0 + m_Armor);
    double actualDamage = baseDamage * damageReduction;
    
    return Math.Max(0, actualDamage);
}
```

**减伤公式**:
```
物理伤害 = 基础伤害 × (100 / (100 + 护甲))

示例：
- 基础伤害 = 100
- 护甲 = 50
- 实际伤害 = 100 × (100 / 150) = 66.67
```

**护甲效果**:
| 护甲值 | 减伤倍数 | 伤害减少 |
|-------|--------|--------|
| 0 | 1.0 | 0% |
| 50 | 0.667 | 33.3% |
| 100 | 0.5 | 50% |
| 200 | 0.333 | 66.7% |

| 类型 | 值 | 减伤方式 | 飘字颜色 |
|------|---|--------|--------|
| 物理 | 1 | 护甲减伤 | 白色 |
| 魔法 | 2 | 魔抗减伤 | 蓝色 |
| 真实 | 3 | 无减伤 | 黄色 |

### 2. 护盾吸收
```csharp
if (m_Shield > 0)
{
    double shieldAbsorb = Math.Min(m_Shield, actualDamage);
    ModifyShield(-shieldAbsorb);
    actualDamage -= shieldAbsorb;

```

**优先级**: 护盾 > 血量

### 3. 伤害减免Buff
```csharp

{
    actualDamage *= m_DamageTakenMultiplier;
}
```

**应用时机**: 在防御计算之后，护盾吸收之前

## ⚠️ 常见问题

### Q1: 为什么伤害被减伤了？
**A**: 检查目标的护甲/魔抗属性，以及是否有伤害减免Buff

### Q2: 真实伤害是否会被护盾吸收？
**A**: 是的，护盾吸收优先级最高，在所有伤害类型之前

### Q3: 系数伤害如何计算？
**A**: `BaseDamage + DamageCoeff × 施法者攻击力`

### Q4: 伤害减免Buff何时应用？
**A**: 在防御计算之后，护盾吸收之前

### Q5: 如何添加新的伤害类型？
**A**: 需要修改 DamageType 枚举和对应的计算方法

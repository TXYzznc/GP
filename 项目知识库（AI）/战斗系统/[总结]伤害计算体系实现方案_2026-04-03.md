| # | 公式 | 说明 | 参数来源 |

### 二、破甲机制（非对称防御）

#### 核心机制

```
正数护甲（防御）：
  减伤效果强 → 最多85%减伤

负数护甲（破甲）：
  增伤效果弱 → 最多30%增伤
  比例：破甲强度 = 护甲强度 × 50%

示例：
  +100护甲 → 减伤30% → 伤害系数70%
  -100破甲 → 增伤15% → 伤害系数115%
  差异：防御强 > 破甲
```

#### 关键参数（硬编码）

```csharp
// 护甲参数
ARMOR_LOW_THRESHOLD = 100       // 低阈值
ARMOR_LOW_RATE = 0.03f          // 3%
ARMOR_HIGH_BASE = 0.03f         // 基础3%
ARMOR_HIGH_RATE = 0.01f         // 递减1%
ARMOR_MAX_REDUCTION = 0.85f     // 最多85%减伤

// 破甲参数
NEGATIVE_ARMOR_RATE = 0.015f    // 破甲系数 = 3% × 50%
MAX_POSITIVE_ARMOR_INCREASE = 0.30f  // 最多增伤30%
MIN_ARMOR_LIMIT = -200f         // 护甲下限
```

public float GetFinalArmor()
{
    return Mathf.Max(m_BaseArmor + m_ArmorModifier, -200f);
}

public void ApplyArmorModifier(float modifier)
{
    m_ArmorModifier += modifier;
    DebugEx.LogModule("ChessAttribute", 
        $"护甲修改: {modifier:+0;-0}, 最终: {GetFinalArmor()}");
}

### Step 4：技能中应用破甲

```csharp
// 破甲技能示例
public class BreakArmorSkill : ChessSkillBase
{
    public override void ExecuteSkill(ChessEntity caster, ChessEntity target)
    {
        // 造成伤害
        double damage = DamageCalculator.CalculateFinalPhysicalDamage(
            caster.Attribute.Attack, 1.2, target.Attribute.GetFinalArmor(),
            target.Attribute.MaxHealth);
        target.TakeDamage((int)damage);
        
        // 应用破甲（-50护甲，3回合后恢复）
        target.Attribute.ApplyArmorModifier(-50);
        ScheduleArmorRecovery(target, 50, 3);  // 3回合后恢复
    }
}
```

## 📊 数值对标

### 物理防御体系

```
目标护甲 = 100，攻击方伤害 = 100

| 破甲方案        | 目标护甲 | 伤害系数 | 最终伤害 |
|---------------|--------|--------|--------|
| 无破甲         | 100    | 97%    | 97     |
| -50破甲        | 50     | 98.5%  | 98.5   |
| -100破甲       | 0      | 100%   | 100    |
| -150破甲       | -50    | 100.75%| 100.75 |
| -200破甲(上限) | -100   | 101.5% | 101.5  |
```

**关键发现**：
- 防御强度（减伤）> 破甲强度（增伤）
- 破甲增伤平缓，防止伤害爆炸
- 形成健康的防御生态

### 法抗系统

```
法抗参数比护甲更弱
- 基础减伤：3.5% vs 3%（更高）
- 破法增伤：1.8系数 vs 1.5系数（更强）
- 最大减伤：80% vs 85%（更低）

目的：鼓励魔法输出职业
```

#### 1️⃣ ShowEscapeResult() - 显示脱战结果UI
**功能**: 创建脱战结果数据，通过UIParams传递给EscapeResultUI

**实现代码**:
```csharp
private async UniTask ShowEscapeResult(bool success)
{
    // 创建结果数据
    EscapeResultData resultData = new EscapeResultData
    {
        Success = success,
        CorruptionCost = success ? m_CurrentRule.CorruptionCost : 0,
        HealthLoss = success ? 0 : (float)m_CurrentRule.HealthLossPenalty,
        CooldownTurns = success ? 0 : m_CurrentRule.CooldownTurns
    };

    // 通过UIParams传递数据
    UIParams uiParams = UIParams.Create();
    uiParams.Set("EscapeResultData", resultData);

    // 打开UI并等待
    try
    {
        await GF.UI.OpenUIFormAwait("EscapeResultUI", "Dialog",
            priority: 50, pauseCoveredUIForm: true, userData: uiParams);
    }
    catch (System.Exception ex)
    {
        DebugEx.ErrorModule("CombatEscapeSystem", $"打开EscapeResultUI失败: {ex.Message}");
    }
}
```

**数据流**:
```
ShowEscapeResult(true)
  ↓
创建EscapeResultData { Success=true, CorruptionCost=10, ... }
  ↓
创建UIParams并Set数据
  ↓
OpenUIFormAwait 打开 EscapeResultUI
  ↓
EscapeResultUI.OnOpen() 接收UIParams
  ↓
ShowResult() 显示UI内容
  ↓
玩家点击确认
  ↓
关闭UI，返回ExitCombat()
```

## 🔑 核心类速查

### CardRangePreview (范围预览管理器)

```csharp
// 显示范围预览
CardRangePreview.Instance.ShowPreview(
    position: Vector3,     // 中心位置
    radius: float          // 半径
);

// 隐藏预览
CardRangePreview.Instance.HidePreview();

// 隐藏所有预览
CardRangePreview.Instance.HideAllPreviews();
```

**工作原理：**
- 在 OnDrawGizmos() 中绘制黄色半透明圆形
- 支持淡入淡出动画 (0.2s)
- 支持脉冲动画 (1.0 → 1.05 → 1.0, 循环)
- 仅在 Scene 视图中显示

**使用示例：**
```csharp
// 冰霜新星效果
public void Execute(Vector3 targetPosition)
{
    float radius = m_CardData.TableRow.AreaRadius;
    
    // 范围内敌人逻辑...
    
    // 显示预览
    CardRangePreview.Instance?.ShowPreview(targetPosition, radius);
}
```

## 📋 TargetType 识别逻辑

### 类型对照表

| TargetType | 值 | 识别逻辑 | 预览 |
|-----------|-----|---------|------|
| **自身** | 1 | - | 无 |
| **友方单体** | 2 | 用户选择 | 可选 |
| **友方全体** | 3 | `Camp == Player` | 全部高亮 |
| **敌方单体** | 4 | 用户选择 | 可选 |
| **敌方全体** | 5 | `Camp == Enemy` | 全部高亮 |
| **全场** | 6 | 全部 | 全部高亮 |

### 代码模板

```csharp
// 全体目标
var targets = allChess.FindAll(c => c.Camp == (int)CampType.Enemy);

// 范围目标 (AreaRadius > 0)
var targets = allChess.FindAll(c =>
    c.Camp == (int)CampType.Enemy &&
    Vector3.Distance(c.transform.position, targetPosition) <= aoeRadius);

// 单体最近目标
ChessEntity closest = null;
float minDist = float.MaxValue;
foreach (var chess in allChess)
{
    if (chess.Camp == (int)CampType.Enemy)
    {
        float dist = Vector3.Distance(chess.transform.position, targetPosition);
        if (dist < minDist)
        {
            minDist = dist;
            closest = chess;
        }
    }
}
```

### 3️⃣ 目标高亮（需创建）

```csharp
public class TargetHighlightPreview : MonoBehaviour
{
    private List<ChessEntity> m_Highlighted = new List<ChessEntity>();
    
    public void HighlightTargets(List<ChessEntity> targets)
    {
        ClearHighlight();
        
        foreach (var target in targets)
        {
            var renderer = target.GetComponent<SpriteRenderer>();
            if (renderer) renderer.color = Color.yellow;
            m_Highlighted.Add(target);
        }
    }
    
    public void ClearHighlight()
    {
        foreach (var target in m_Highlighted)
        {
            var renderer = target.GetComponent<SpriteRenderer>();
            if (renderer) renderer.color = Color.white;
        }
        m_Highlighted.Clear();
    }
}
```

## 🎨 配置表字段说明

### 与预览相关的字段

```
TargetType   (目标类型)
├─ 1=自身、2=友方单体、3=友方全体
├─ 4=敌方单体、5=敌方全体、6=全场
└─ 决定预览逻辑和高亮范围

AreaRadius   (AOE 半径)
├─ 0 = 非范围目标
└─ >0 = 圆形 AOE，预览圆半径 = AreaRadius

CastRange    (施法范围)
├─ 0 = 无限制
└─ >0 = 必须在此距离内施法（待实现）

DamageType   (伤害类型)
├─ 0=无、1=物理、2=魔法、3=真实
└─ 影响伤害计算

InstantBuffs / HitBuffs
└─ 释放或命中时施加的 Buff
```

## 💡 常见问题速解

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 预览不显示 | m_IsShowingPreview=false | 检查 ShowPreview() 调用 |
| 圆形太小/太大 | AreaRadius 配置错 | 检查 CardTable.txt 中的值 |
| 预览显示在错位置 | 坐标系转换错 | 实现正确的 ScreenToWorld 转换 |
| 拖拽时预览卡顿 | 每帧多次调用 ShowPreview | 改为仅更新位置 |
| 高亮没清除 | 忘记调用 ClearHighlight | 在 OnEndDrag 中调用 |

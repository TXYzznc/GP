# 账号数据编辑器工具使用指南

## 功能概述

账号数据编辑器是一个 Unity Editor 工具，用于快速修改游戏存档数据。支持：
- 浏览所有账号和存档
- 在线编辑存档数据（等级、金币、位置等）
- 实时保存修改
- 数据刷新和重新加载

## 打开工具

在 Unity Editor 菜单栏选择：**工具 → 数据工具 → 账号数据编辑器**

## 使用流程

### 1. 选择数据路径

- 点击"浏览"按钮选择存档数据目录
- 通常路径为：`C:/Users/[用户名]/AppData/LocalLow/珍珠奶茶/Clash_Of_Gods/PlayerSaves`
- 工具会自动记住上次选择的路径

### 2. 加载账号和存档

- 点击"刷新"按钮扫描目录下的所有账号
- 左侧列表显示所有账号及其存档数量
- 点击账号名称选中该账号

### 3. 选择存档

- 选中账号后，中间列表显示该账号的所有存档
- 存档名称格式：`[存档ID] 存档名称`
- 带 `*` 标记表示该存档有未保存的修改

### 4. 编辑存档数据

选中存档后，右侧显示可编辑的字段：

#### 基础信息
- **存档ID**：存档的唯一标识符
- **存档名称**：玩家给存档起的名字
- **等级**：玩家当前等级
- **经验值**：玩家当前经验

#### 货币
- **金币**：游戏内金币数量
- **灵石**：高级货币数量
- **圣水**：特殊货币数量

#### 召唤师信息
- **当前召唤师ID**：玩家选择的召唤师ID
- **召唤师等级**：召唤师的等级
- **召唤师经验**：召唤师的经验值

#### 位置信息
- **玩家位置**：玩家在游戏世界中的坐标 (X, Y, Z)
- **当前场景ID**：玩家所在场景的ID

#### 游戏进度
- **教程完成**：是否完成了新手教程
- **最后保存时间**：存档最后保存的时间戳

### 5. 保存修改

- 修改任何字段后，存档名称旁会显示 `*` 标记
- 点击"保存修改"按钮将修改写入 JSON 文件
- 保存成功后会显示状态提示

### 6. 重新加载

- 点击"重新加载"按钮放弃当前修改，重新从文件加载数据
- 用于撤销未保存的修改

## 常见操作

### 快速修改玩家等级和金币

1. 选择数据路径 → 刷新
2. 选择账号 → 选择存档
3. 修改"等级"和"金币"字段
4. 点击"保存修改"

### 修改玩家位置

1. 在"位置信息"部分修改"玩家位置"
2. 同时修改"当前场景ID"（如需要）
3. 点击"保存修改"

### 切换召唤师

1. 修改"当前召唤师ID"字段
2. 同时可修改"召唤师等级"和"召唤师经验"
3. 点击"保存修改"

## 注意事项

- **备份重要存档**：修改前建议备份存档文件
- **JSON 格式**：存档以 JSON 格式保存，工具会自动处理序列化
- **路径有效性**：确保选择的路径包含有效的账号文件夹
- **文件权限**：确保有读写权限访问存档文件
- **游戏运行中**：修改存档时最好关闭游戏，避免数据冲突

## 状态提示

工具底部显示当前状态：
- **已加载 X 个账号**：成功加载的账号数量
- **正在加载...**：正在扫描目录
- **路径不存在**：选择的路径无效
- **存档已保存**：修改已成功保存
- **保存失败**：保存时出现错误

## 技术细节

- 工具位置：`Assets/Editor/unity开发集成工具箱/Panels/AccountDataEditorPanel.cs`
- 支持的数据类型：所有 `PlayerSaveData` 中的公开字段
- 存档文件格式：JSON（使用 `JsonUtility` 序列化）
- 自动保存路径记忆：上次选择的路径会被记住

## 故障排除

### 找不到账号数据
- 确认路径是否正确
- 检查是否有有效的账号文件夹（如 `000001`）
- 确保文件夹中有 `Save_*.json` 文件

### 保存失败
- 检查文件是否被其他程序占用
- 确认有写入权限
- 查看 Console 窗口的错误信息

### 数据显示为空
- 点击"重新加载"按钮
- 检查 JSON 文件是否有效
- 查看 Console 窗口的错误日志

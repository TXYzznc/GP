## 提示词使用指南

### 如何使用这些提示词

1. **复制完整提示词** - 选择对应技能的提示词
2. **粘贴到 AI 工具** - 在 Midjourney、Stable Diffusion 等工具中粘贴
3. **调整参数**（可选）：
   - 分辨率：512x512（推荐）或 768x768（更高质量）
   - 质量：`--quality 2` 或 `--quality 1`（Midjourney）
   - 风格权重：`--style raw` 或 `--niji`

### 常见调整

**如果想要更高质量：**
```
--quality 2 --style raw --ar 1:1
```

**如果想要更多细节：**
```
--quality 2 --detail high --style raw
```

**如果想要特定风格：**
```
--niji --style anime  // 日系风格
--style raw           // 原始风格
--style 4a            // 艺术风格
```

### 质量检查清单

生成后检查以下要点：

- ✅ 没有文字、数字、符号
- ✅ 色彩饱和度高
- ✅ 技能特效清晰可见
- ✅ 背景与主体协调
- ✅ 光效表现到位
- ✅ 符合游戏美学
- ✅ 技能特性表现准确

## 生成建议

### 最佳实践

1. **保持一致性** - 同一角色的所有技能应有视觉连贯性
2. **突出特效** - 确保技能特效（灼烧、冰晶、破碎）清晰可见
3. **背景衬托** - 背景应衬托主体，不抢戏
4. **色彩搭配** - 遵循职业色彩系统
5. **动感表现** - 被动稳定，大招爆发

### 常见问题

**Q: 生成的图片有文字怎么办？**
A: 在提示词中已明确禁止文字。如果仍有文字，可在工具中添加 `--no text` 参数。

**Q: 色彩不够饱和？**
A: 在提示词中添加 `--style raw` 或 `--quality 2` 以提高饱和度。

**Q: 背景太复杂？**
A: 修改背景描述，使用更简洁的表述，如 "simple gradient background"。

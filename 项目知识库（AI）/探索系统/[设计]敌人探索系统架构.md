## 📋 系统概述

敌人探索系统负责管理探索场景中的敌人行为，包括巡逻、警戒、追击、广播协作和触发战斗。

### 核心功能
- **AI状态机**：休息、巡逻、警戒、追击、战斗
- **敌人类型**：普通、精英、Boss、特殊
- **广播协作**：精英敌人广播玩家位置，召集附近敌人
- **群体战斗**：检测范围内敌人，动态调整难度
- **净化系统**：战斗胜利后净化Boss/特殊敌人

```
    └─ PurificationSystem (净化系统)
    ↓
配置层：EnemyEntityTable (实体配置)
    └─ EnemyTable (战斗配置)
```

## 🎭 AI状态机

### 状态定义

```csharp
public enum EnemyAIState
{
    Idle,       // 休息：原地待机，可被偷袭
    Patrol,     // 巡逻：在巡逻范围内随机移动
    Alert,      // 警戒：发现玩家，进入警戒状态
    Chase,      // 追击：追击玩家，接近后触发战斗
    Combat      // 战斗：已进入战斗状态（由战斗系统接管）
}
```

### 状态转换规则

```
Idle ←→ Patrol
  ↓       ↓
  Alert ←─┘
    ↓
  Chase
    ↓
  Combat
```

**转换条件**：
- `Idle → Patrol`：休息时间结束
- `Patrol → Idle`：随机触发休息（概率配置）
- `Patrol/Idle → Alert`：检测到玩家（远距离）
- `Alert → Chase`：警戒时间结束且玩家仍在范围内
- `Chase → Combat`：进入战斗距离

### 状态实现

#### 1. IdleState（休息状态）

```csharp
public class EnemyIdleState : EnemyStateBase
{
    private float m_RestTimer;
    
    public override void OnEnter()
    {
        m_RestTimer = m_AI.Entity.Config.RestDuration;
        m_AI.Entity.StopMovement();
        DebugEx.Log("EnemyIdleState", $"{m_AI.Entity.Name} 开始休息");
    }
    
    public override void OnUpdate()
    {
        // 检测玩家
        if (DetectPlayer())
        {
            m_AI.ChangeState(EnemyStateType.Alert);
            return;
        }
        
        // 休息计时
        m_RestTimer -= Time.deltaTime;
        if (m_RestTimer <= 0)
        {
            m_AI.ChangeState(EnemyStateType.Patrol);
        }
    }
}
```

#### 2. PatrolState（巡逻状态）

```csharp
public class EnemyPatrolState : EnemyStateBase
{
    private Vector3 m_PatrolTarget;
    
    public override void OnEnter()
    {
        SelectNewPatrolPoint();
    }
    
    public override void OnUpdate()
    {
        // 检测玩家
        if (DetectPlayer())
        {
            m_AI.ChangeState(EnemyStateType.Alert);
            return;
        }
        
        // 移动到巡逻点
        MoveToTarget(m_PatrolTarget);
        
        // 到达巡逻点
        if (ReachedTarget())
        {
            // 随机决定休息或继续巡逻
            if (Random.value < m_AI.Entity.Config.RestProbability)
            {
                m_AI.ChangeState(EnemyStateType.Idle);
            }
            else
            {
                SelectNewPatrolPoint();
            }
        }
    }
}
```

#### 3. AlertState（警戒状态）

```csharp
public class EnemyAlertState : EnemyStateBase
{
    private float m_AlertTimer;
    
    public override void OnEnter()
    {
        m_AlertTimer = m_AI.Entity.Config.AlertTime;
        m_AI.Entity.StopMovement();
        ShowAlertUI();  // 显示警戒UI（！）
    }
    
    public override void OnUpdate()
    {
        // 玩家离开范围
        if (!IsPlayerInRange())
        {
            m_AI.ChangeState(EnemyStateType.Patrol);
            return;
        }
        
        // 警戒计时
        m_AlertTimer -= Time.deltaTime;
        if (m_AlertTimer <= 0)
        {
            m_AI.ChangeState(EnemyStateType.Chase);
        }
    }
}
```

#### 4. ChaseState（追击状态）

```csharp
public class EnemyChaseState : EnemyStateBase
{
    private bool m_HasBroadcasted = false;
    
    public override void OnEnter()
    {
        m_HasBroadcasted = false;
        DebugEx.Log("EnemyChaseState", $"{m_AI.Entity.Name} 开始追击");
    }
    
    public override void OnUpdate()
    {
        float distanceToPlayer = Vector3.Distance(
            m_AI.Entity.CachedTransform.position,
            m_AI.PlayerTransform.position
        );
        
        // 广播逻辑（精英敌人，只执行一次）
        if (!m_HasBroadcasted && 
            m_AI.Entity.Config.IsElite && 
            distanceToPlayer <= m_AI.Entity.Config.BroadcastDistance)
        {
            BroadcastPlayerPosition();
            m_HasBroadcasted = true;
        }
        
        // 进入战斗距离
        if (distanceToPlayer <= m_AI.Entity.Config.CombatDistance)
        {
            TriggerCombat();
            return;
        }
        
        // 玩家逃离
        if (distanceToPlayer > m_AI.Entity.Config.ChaseDistance)
        {
            m_AI.ChangeState(EnemyStateType.Patrol);
            return;
        }
        
        // 继续追击
        MoveTowardsPlayer();
    }
    
    public override void OnExit()
    {
        m_HasBroadcasted = false;  // 重置广播标志
    }
}
```

## 📡 广播协作系统

### 工作原理

1. **精英敌人追击玩家**
2. **进入广播距离**（配置表设置）
3. **广播玩家位置**
4. **附近敌人接收广播**
5. **附近敌人切换到追击状态**

### 实现代码

```csharp
// 广播玩家位置
private void BroadcastPlayerPosition()
{
    EnemyGroupManager.Instance?.BroadcastPlayerPosition(
        m_AI.Entity,
        m_AI.PlayerTransform.position,
        m_AI.Entity.Config.BroadcastDistance
    );
    
    DebugEx.Log("EnemyChaseState", 
        $"{m_AI.Entity.Name} 广播玩家位置");
}

// EnemyGroupManager 处理广播
public void BroadcastPlayerPosition(
    EnemyEntity broadcaster, 
    Vector3 playerPos, 
    float range)
{
    var nearbyEnemies = GetEnemiesInRange(broadcaster.Position, range);
    
    foreach (var enemy in nearbyEnemies)
    {
        if (enemy != broadcaster && 
            enemy.AI.CurrentState != EnemyStateType.Combat)
        {
            enemy.AI.ReceiveBroadcast(playerPos);
        }
    }
}

// 敌人接收广播
public void ReceiveBroadcast(Vector3 playerPosition)
{
    m_BroadcastedPlayerPosition = playerPosition;
    ChangeState(EnemyStateType.Chase);
}
```

## 🎯 群体战斗系统

### 检测机制

战斗触发时，检测范围内的所有敌人：

```csharp
public List<EnemyEntity> GetNearbyEnemies(Vector3 position, float range)
{
    return m_AllEnemies
        .Where(e => e.IsAlive && 
                    Vector3.Distance(e.Position, position) <= range)
        .ToList();
}
```

### 难度调整

根据参战敌人数量动态调整：

```csharp
// 1个敌人：标准难度
// 2个敌人：+20% 血量和攻击
// 3个敌人：+40% 血量和攻击
// 4+个敌人：+60% 血量和攻击（上限）

float difficultyMultiplier = 1.0f + (enemyCount - 1) * 0.2f;
difficultyMultiplier = Mathf.Min(difficultyMultiplier, 1.6f);
```

## ✨ 净化系统

### 净化条件

- 敌人类型为 Boss 或 Special
- 战斗胜利
- 配置了 `PurificationCardId`

### 实现流程

```csharp
// 战斗胜利后检查
public void OnCombatVictory(List<EnemyEntity> defeatedEnemies)
{
    foreach (var enemy in defeatedEnemies)
    {
        if (CanPurify(enemy))
        {
            ShowPurificationUI(enemy);
        }
    }
}

private bool CanPurify(EnemyEntity enemy)
{
    return (enemy.Config.EnemyType == EnemyType.Boss || 
            enemy.Config.EnemyType == EnemyType.Special) &&
           enemy.Config.PurificationCardId > 0;
}

// 净化后获得卡牌
private void Purify(EnemyEntity enemy)
{
    int cardId = enemy.Config.PurificationCardId;
    PlayerInventory.AddCard(cardId);
    
    DebugEx.Success("PurificationSystem", 
        $"净化 {enemy.Name}，获得卡牌 {cardId}");
}
```

### 常见问题

**Q: 敌人不追击玩家？**
- 检查 `AlertDistance` 和 `ChaseDistance` 配置
- 确认玩家在检测范围内
- 查看日志确认状态切换

**Q: 广播不生效？**
- 确认敌人类型为 Elite
- 检查 `CanBroadcast` 字段
- 确认 `BroadcastDistance` 配置合理

**核心特性：**
- 摄像机独立轨道控制（鼠标控制摄像机旋转，而非角色旋转）
- 移动方向基于摄像机视角
- 角色模型平滑转向移动方向
- 智能防遮挡系统
- 滚轮缩放
- 动态 FOV（快跑时增加视野）

## 新增/修改的脚本

### 1. ThirdPersonCamera.cs（新增）
位置：`Assets/AAAGame/Scripts/Game/Camera/ThirdPersonCamera.cs`

**功能：**
- 轨道控制：鼠标控制摄像机绕角色旋转
- 平滑跟随：使用 SmoothDamp 实现滞后跟随
- 缩放控制：滚轮控制距离
- 动态 FOV：快跑时增加视野
- 智能防遮挡：SphereCast 检测 + 平滑恢复

**主要参数：**
```
轨道控制:
- orbitSensitivityX: 水平旋转灵敏度 (默认 3)
- orbitSensitivityY: 垂直旋转灵敏度 (默认 2)
- orbitDamping: 旋转阻尼时间 (默认 0.1)
- minPitch/maxPitch: 俯仰角限制 (-40 ~ 80)

跟随设置:
- followSmoothTime: 跟随平滑时间 (默认 0.1)
- targetOffset: 目标偏移 (默认 0, 1.5, 0)

缩放设置:
- defaultDistance: 默认距离 (默认 5)
- minDistance/maxDistance: 距离限制 (2 ~ 10)
- zoomSpeed: 缩放速度 (默认 2)

动态 FOV:
- defaultFOV: 默认视野 (默认 60)
- sprintFOV: 冲刺视野 (默认 70)

防遮挡:
- occlusionLayers: 遮挡检测层
- occlusionRadius: 检测球体半径 (默认 0.3)
- occlusionRecoverySpeed: 恢复速度 (默认 2)
```

### 2. PlayerController.cs（重构）
位置：`Assets/AAAGame/Scripts/Game/Player/PlayerController.cs`

**主要变化：**
- 移除了鼠标控制角色旋转的逻辑
- 新增基于摄像机方向的移动计算
- 新增平滑转身逻辑（SmoothDampAngle）
- 新增斜坡速度投影
- 新增与摄像机的 FOV 联动

**主要参数：**
```
移动设置:
- walkSpeed: 行走速度 (默认 2)
- slowRunSpeed: 慢跑速度 (默认 4)
- fastRunSpeed: 快跑速度 (默认 8)
- acceleration: 加速度 (默认 10)
- deceleration: 减速度 (默认 8)

旋转设置:
- turnSmoothTime: 转身平滑时间 (默认 0.1)

组件引用:
- cameraRig: ThirdPersonCamera 引用
```

## 操作说明

| 操作 | 效果 |
|------|------|
| 鼠标移动 | 控制摄像机旋转（角色不旋转） |
| W/A/S/D | 基于摄像机方向移动，角色自动转向移动方向 |
| 双击 W | 进入慢跑状态 |
| Shift（慢跑时） | 切换到快跑状态 |
| Shift（快跑时） | 切换回慢跑状态 |
| 滚轮 | 缩放摄像机距离 |
| Tab | 切换鼠标锁定状态 |

## 行为说明

### 移动方向
- 按 W：角色背对摄像机，向前跑
- 按 S：角色转身面对摄像机，向屏幕方向跑
- 按 A/D：角色向左/右跑，身体朝向左/右

### 转身行为
- 快速转向时，角色会画出平滑的弧线转身
- 不会出现瞬间 180 度翻转

### 防遮挡行为
- 当障碍物出现在摄像机和角色之间时，摄像机自动拉近
- 障碍物消失后，摄像机平滑恢复到原始距离

### 核心特性
- ✅ **鼠标控制玩家旋转**：鼠标移动直接控制角色旋转
- ✅ **摄像机固定跟随**：摄像机保持固定偏移跟随角色
- ✅ **摄像机跟随旋转**：角色旋转时摄像机同步旋转
- ✅ **基于玩家朝向移动**：WASD 移动基于角色朝向，不是世界坐标
- ✅ **鼠标锁定/解锁**：Tab 键切换鼠标显示和锁定状态
- ✅ **平滑加速/减速**：角色移动有惯性，手感自然
- ✅ **碰撞检测**：摄像机遇到障碍物自动拉近

## 控制方式

### 基础控制
- **WASD**：移动（基于角色朝向）
  - W：向角色前方移动
  - S：向角色后方移动
  - A：向角色左侧移动
  - D：向角色右侧移动
- **鼠标移动**：旋转角色（鼠标锁定时）
- **Tab**：切换鼠标锁定/解锁
- **Shift**：冲刺（已预留）

### 鼠标状态
- **锁定状态**（默认）：
  - 鼠标隐藏
  - 鼠标移动控制角色旋转
  - 无法点击 UI
  
- **解锁状态**（按 Tab）：
  - 鼠标显示
  - 鼠标移动不影响角色
  - 可以点击 UI

## 系统组件

### 1. PlayerInputManager（输入管理器）
**位置：** `Assets/AAAGame/Scripts/Game/Input/PlayerInputManager.cs`

**功能：**
- 统一处理键盘和鼠标输入
- 管理鼠标锁定状态
- 提供归一化的移动和鼠标输入

**配置参数：**
- `Mouse Sensitivity`: 鼠标灵敏度（默认 1.0）
- `Lock Cursor`: 启动时是否锁定鼠标（默认 true）

**公共方法：**
```csharp
// 设置鼠标锁定状态
PlayerInputManager.Instance.SetCursorLock(true/false);

// 切换鼠标锁定状态
PlayerInputManager.Instance.ToggleCursorLock();

// 检查鼠标是否锁定
bool isLocked = PlayerInputManager.Instance.IsMouseLocked;
```

### 2. PlayerController（角色控制器）
**位置：** `Assets/AAAGame/Scripts/Game/Player/PlayerController.cs`

**功能：**
- 鼠标控制角色旋转
- 基于角色朝向的移动
- 平滑加速/减速
- 动画系统集成

**配置参数：**
- `Max Move Speed`: 最大移动速度（默认 5 m/s）
- `Acceleration`: 加速度（默认 10 m/s²）
- `Deceleration`: 减速度（默认 15 m/s²）
- `Rotation Speed`: 旋转速度（默认 180 度/秒）
- `Gravity`: 重力（默认 -9.81）

**调优建议：**
- **旋转太慢：** 提高 Rotation Speed（200-300）
- **旋转太快：** 降低 Rotation Speed（100-150）或降低 Mouse Sensitivity
- **移动太飘：** 提高 Deceleration
- **移动太僵硬：** 降低 Acceleration 和 Deceleration

### 3. ThirdPersonCameraRig（摄像机装备）
**位置：** `Assets/AAAGame/Scripts/Game/Camera/ThirdPersonCameraRig.cs`

**功能：**
- 固定偏移跟随角色
- 跟随角色旋转
- 碰撞检测与避免
- 平滑跟随

**配置参数：**

#### 跟随设置
- `Camera Offset`: 摄像机相对于角色的偏移（默认 (0, 5, -8)）
  - X：左右偏移（正值向右）
  - Y：高度偏移（正值向上）
  - Z：前后偏移（负值向后）
- `Follow Smooth Speed`: 位置跟随平滑速度（默认 10）
- `Rotation Smooth Speed`: 旋转跟随平滑速度（默认 10）

#### 摄像机朝向
- `Look At Offset`: 摄像机看向目标的偏移（默认 (0, 1, 0)）

#### 碰撞设置
- `Enable Collision`: 是否启用碰撞检测（默认 true）
- `Collision Layers`: 碰撞检测层
- `Collision Buffer`: 碰撞缓冲距离（默认 0.2m）

**调优建议：**

**调整视角：**
```
俯视角度（类似 LOL）：
Camera Offset: (0, 10, -5)

斜后方视角（类似暗黑）：
Camera Offset: (0, 8, -10)

近距离视角：
Camera Offset: (0, 3, -5)
```

**调整平滑度：**
```
快速响应：
Follow Smooth Speed: 15-20
Rotation Smooth Speed: 15-20

平滑跟随：
Follow Smooth Speed: 5-10
Rotation Smooth Speed: 5-10
```

## 使用流程

### 1. 自动集成
系统已自动集成到角色生成流程中。当玩家角色生成时：
1. 自动创建 ThirdPersonCameraRig
2. 自动配置摄像机偏移和跟随
3. 自动锁定鼠标
4. 自动连接 PlayerController

### 2. 测试步骤

#### 基础移动测试
1. 启动游戏，进入游戏场景
2. 鼠标应该被隐藏和锁定
3. 移动鼠标，观察角色是否旋转
4. 使用 WASD 移动，观察移动方向是否基于角色朝向

#### 鼠标锁定测试
1. 按 Tab 键
2. 鼠标应该显示出来
3. 移动鼠标，角色不应该旋转
4. 再按 Tab 键，鼠标应该隐藏，角色恢复旋转

#### 摄像机跟随测试
1. 旋转角色
2. 观察摄像机是否跟随旋转
3. 移动角色
4. 观察摄像机是否保持固定偏移跟随

#### 碰撞测试
1. 移动到墙角或障碍物附近
2. 观察摄像机是否自动拉近避免穿墙
3. 离开障碍物，观察摄像机是否平滑返回

### 3. 调优建议

#### 手感调优
如果角色控制感觉：
- **旋转太慢：** 提高 Rotation Speed 或 Mouse Sensitivity
- **旋转太快：** 降低 Rotation Speed 或 Mouse Sensitivity
- **移动太飘：** 提高 Deceleration
- **移动太僵硬：** 降低 Acceleration 和 Deceleration

#### 视角调优
如果摄像机感觉：
- **太高：** 减小 Camera Offset 的 Y 值
- **太低：** 增大 Camera Offset 的 Y 值
- **太远：** 减小 Camera Offset 的 Z 值（绝对值）
- **太近：** 增大 Camera Offset 的 Z 值（绝对值）
- **跟随太慢：** 提高 Follow Smooth Speed
- **跟随太快/抖动：** 降低 Follow Smooth Speed

## 常见问题

### Q: 鼠标移动但角色不旋转？
A: 检查鼠标是否被锁定（按 Tab 切换），只有锁定状态下才能控制角色旋转。

### Q: 按 W 但角色不向前移动？
A: 检查角色的朝向，W 键是向角色前方移动，不是向世界坐标的前方。

### Q: 摄像机不跟随角色旋转？
A: 检查 ThirdPersonCameraRig 的 target 是否正确设置为角色 Transform。

### Q: 摄像机穿墙？
A: 检查 Collision Layers 设置，确保包含墙体所在的层。

### Q: 动画不播放？
A: 检查角色是否有 Animator 组件，以及动画控制器是否有 Speed 和 IsMoving 参数。

### Q: 进入游戏后鼠标没有隐藏？
A: 检查 PlayerInputManager 的 Lock Cursor 选项是否勾选。

## 与其他系统的区别

### 与自由视角第三人称的区别

**本系统（固定视角）：**
- 鼠标控制角色旋转
- 摄像机跟随角色旋转
- 移动基于角色朝向
- 适合 ARPG、MOBA 类游戏

**自由视角第三人称：**
- 鼠标控制摄像机旋转
- 角色独立旋转
- 移动基于摄像机朝向
- 适合动作冒险、开放世界游戏

## 扩展建议

### 添加冲刺功能
在 PlayerController 中：
```csharp
if (PlayerInputManager.Instance.Sprint)
{
    targetSpeed *= 2f; // 冲刺时速度翻倍
}
```

### 添加技能释放
在 PlayerInputManager 中已经预留了技能输入：
```csharp
if (PlayerInputManager.Instance.SkillDown(1))
{
    // 释放技能 1
}
```

### 自定义摄像机视角
可以在运行时动态调整摄像机偏移：
```csharp
ThirdPersonCameraRig cameraRig = FindObjectOfType<ThirdPersonCameraRig>();
cameraRig.SetCameraOffset(new Vector3(0f, 12f, -6f));
```

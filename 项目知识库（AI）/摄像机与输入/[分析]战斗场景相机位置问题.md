## 问题分析

### 问题1：相机没有移动到 CameraAnchor

**现象：**
- 日志中没有看到 `MoveCameraToCameraAnchor` 的相关输出
- 说明该方法没有被调用

**原因：**
- `SpawnBattleArena()` 是异步方法，但 `MoveCameraToCameraAnchor()` 的调用没有在 `await` 之后
- 需要确保在场地生成完成后再调用相机移动

**解决方案：**
- 将 `MoveCameraToCameraAnchor()` 的调用移到 `await DissolveTransitionManager.Instance.TransitionToBattle(arena)` 之后

---

### 问题2：玩家位置没有恢复

**现象：**
```
[14:52:23.241] [Warning] [EnemyEntityManager] 无法恢复玩家位置：玩家角色不存在
```

**原因：**
- 进入探索状态时，玩家角色还没有生成
- `ExplorationState.OnEnter()` 在 `GameProcedure` 生成玩家之前就被调用了

**时序问题：**
```
1. [14:52:23.240] [ExplorationState] 进入探索状态
2. [14:52:23.241] [Warning] 无法恢复玩家位置：玩家角色不存在  ← 这里玩家还没生成
3. [14:52:23.283] [PlayerCharacterManager] 开始生成玩家角色流程  ← 玩家在这里才生成
```

**解决方案：**
- 不在 `ExplorationState.OnEnter()` 中恢复玩家位置
- 改为在战斗结束时（`CombatState.OnLeave()` 或 `InGameState.OnCombatEnd()`）恢复玩家位置
- 或者在玩家生成完成后再恢复位置

## 修复方案

### 方案1：在战斗结束时立即恢复玩家位置（推荐）

**优点：**
- 时序清晰，不依赖玩家生成时机
- 在切换状态前就完成位置恢复

**修改位置：**
1. 移除 `ExplorationState.OnEnter()` 中的位置恢复代码
2. 在 `InGameState.OnCombatEnd()` 中恢复玩家位置

---

### 方案2：在玩家生成后恢复位置

**优点：**
- 确保玩家存在时才恢复

**缺点：**
- 需要修改 `PlayerCharacterManager` 或 `GameProcedure`
- 时序复杂

### 修改1：CombatPreparationState.cs - 修复相机移动时机和实现

**文件路径：** `Assets/AAAGame/Scripts/GameState/States/CombatPreparationState.cs`

**修改位置 1：** `SpawnBattleArena()` 方法

```csharp
/// <summary>
/// 生成战斗场地（带溶解过渡）
/// </summary>
private async void SpawnBattleArena()
{
    if (BattleArenaManager.Instance != null)
    {
        // 在玩家当前位置附近生成战斗场地
        Vector3 spawnPos = Vector3.zero;
        if (PlayerCharacterManager.Instance != null &&
            PlayerCharacterManager.Instance.CurrentPlayerCharacter != null)
        {
            await BattleArenaManager.Instance.SpawnArenaAsync(PlayerCharacterManager.Instance.CurrentPlayerCharacter.transform);
        }

        // 播放溶解过渡效果（从探索场景显示战斗场地）
        var arena = BattleArenaManager.Instance.CurrentArena;
        if (arena != null)
        {
            await DissolveTransitionManager.Instance.TransitionToBattle(arena);
        }

        // ⭐ 在过渡完成后移动相机（确保场地已加载）
        MoveCameraToCameraAnchor();

        // 加载敌人波次配置（使用保存的战斗数据）
        LoadEnemyWaveConfig();
    }
    else
    {
        DebugEx.WarningModule("CombatPreparationState", "BattleArenaManager 未初始化");
    }
}
```

**修改原因：** 确保相机移动在场地加载和过渡完成后执行。

**修改位置 2：** 添加/修改 `MoveCameraToCameraAnchor()` 方法（约第 350 行之后）

```csharp
/// <summary>
/// 将相机移动到战斗场地的 CameraAnchor 位置
/// </summary>
private void MoveCameraToCameraAnchor()
{
    DebugEx.LogModule("CombatPreparationState", "开始移动相机到 CameraAnchor...");

    // 获取战斗场地
    var arena = BattleArenaManager.Instance?.CurrentArena;
    if (arena == null)
    {
        DebugEx.WarningModule("CombatPreparationState", "战斗场地不存在，跳过相机移动");
        return;
    }

    // 查找 CameraAnchor
    Transform cameraAnchor = arena.transform.Find("CameraAnchor");
    if (cameraAnchor == null)
    {
        DebugEx.WarningModule("CombatPreparationState", "未找到 CameraAnchor，跳过相机移动");
        return;
    }

    // ⭐ 通过 CameraRegistry 获取玩家相机
    Camera playerCamera = CameraRegistry.PlayerCamera;
    if (playerCamera == null)
    {
        DebugEx.ErrorModule("CombatPreparationState", "未找到玩家相机（CameraRegistry.PlayerCamera 为空）");
        return;
    }

    // 移动相机到锚点位置（世界坐标）
    playerCamera.transform.position = cameraAnchor.position;
    playerCamera.transform.rotation = cameraAnchor.rotation;

    DebugEx.LogModule("CombatPreparationState",
        $"相机已移动到 CameraAnchor: Pos={cameraAnchor.position}, Rot={cameraAnchor.rotation.eulerAngles}");
}
```

**修改原因：** 使用 `CameraRegistry.PlayerCamera` 静态属性获取相机，无需添加额外引用。

### 修改3：InGameState.cs - 在战斗结束时恢复玩家位置

**文件路径：** `Assets/AAAGame/Scripts/GameState/States/InGameState.cs`

**修改位置：** `OnCombatEnd()` 方法

```csharp
/// <summary>
/// 处理战斗结束事件
/// </summary>
private void OnCombatEnd(object sender, GameEventArgs e)
{
    CombatEndEventArgs eventArgs = (CombatEndEventArgs)e;
    DebugEx.LogModule("InGameState", $"收到战斗结束事件 - {(eventArgs.IsVictory ? "胜利" : "失败")}");

    // ⭐ 在切换状态前先恢复玩家位置
    if (EnemyEntityManager.Instance != null)
    {
        EnemyEntityManager.Instance.RestorePlayerPosition();
    }

    // 战斗结束后自动切换到探索状态
    SwitchToExploration();
}
```

**修改原因：** 在战斗结束、切换到探索状态之前恢复玩家位置，此时玩家角色已存在。

## 测试验证

### 测试步骤

1. **测试相机移动**
   - 进入战斗准备阶段
   - 查看 Console 日志，应该看到：
     ```
     [CombatPreparationState] 开始移动相机到 CameraAnchor...
     [CombatPreparationState] 相机已移动到 CameraAnchor: Pos=(...), Rot=(...)
     ```
   - 观察相机视角是否正确

2. **测试玩家位置恢复**
   - 在不同位置触发战斗
   - 战斗结束后查看日志，应该看到：
     ```
     [EnemyEntityManager] 记录玩家位置: Pos=(...), Rot=(...)
     [EnemyEntityManager] 恢复玩家位置: Pos=(...), Rot=(...)
     ```
   - 确认玩家回到原位置

3. **检查时序**
   - 确保相机移动在场地加载完成后
   - 确保位置恢复在玩家存在时

## 额外优化建议

### 1. 添加相机平滑过渡

当前相机是瞬间移动，可以添加平滑过渡：

```csharp
private async void MoveCameraToCameraAnchor()
{
    // ... 获取相机和锚点 ...

    // 平滑移动（可选）
    float duration = 0.5f;
    Vector3 startPos = playerCamera.transform.position;
    Quaternion startRot = playerCamera.transform.rotation;
    
    float elapsed = 0f;
    while (elapsed < duration)
    {
        elapsed += Time.deltaTime;
        float t = elapsed / duration;
        
        playerCamera.transform.position = Vector3.Lerp(startPos, cameraAnchor.position, t);
        playerCamera.transform.rotation = Quaternion.Slerp(startRot, cameraAnchor.rotation, t);
        
        await UniTask.Yield();
    }
    
    // 确保最终位置精确
    playerCamera.transform.position = cameraAnchor.position;
    playerCamera.transform.rotation = cameraAnchor.rotation;
}
```

### 2. 添加玩家位置验证

在恢复位置前检查位置是否合法：

```csharp
public void RestorePlayerPosition()
{
    if (PlayerCharacterManager.Instance != null &&
        PlayerCharacterManager.Instance.CurrentPlayerCharacter != null)
    {
        // 检查位置是否合法（不是零向量）
        if (m_PlayerPositionBeforeCombat == Vector3.zero)
        {
            DebugEx.WarningModule("EnemyEntityManager", "玩家位置记录为零，跳过恢复");
            return;
        }

        Transform playerTransform = PlayerCharacterManager.Instance.CurrentPlayerCharacter.transform;
        playerTransform.position = m_PlayerPositionBeforeCombat;
        playerTransform.rotation = m_PlayerRotationBeforeCombat;

        DebugEx.LogModule("EnemyEntityManager",
            $"恢复玩家位置: Pos={m_PlayerPositionBeforeCombat}, Rot={m_PlayerRotationBeforeCombat.eulerAngles}");
    }
    else
    {
        DebugEx.WarningModule("EnemyEntityManager", "无法恢复玩家位置：玩家角色不存在");
    }
}
```

用户反馈：按下技能对应的按键（J/K/L）后，技能没有反应。

## 问题分析

### 原始问题
1. **PlayerSkillManager.Update() 只处理槽位1**：
   - 原代码只检测槽位1的输入（`SkillDown(1)`）
   - 只释放第一个技能（`Skills[0].TryCast()`）
   - 没有建立槽位索引到技能的映射关系

2. **技能槽位和列表索引的混淆**：
   - 配置表中技能有 `SlotIndex`（1, 2, 3）
   - 技能列表按添加顺序存储在 `Skills[0]`, `Skills[1]`, `Skills[2]`
   - 需要根据 `SlotIndex` 查找对应的技能

3. **按键提示显示不直观**：
   - 原代码显示槽位数字（1, 2, 3）
   - 应该显示实际的按键名称（J, K, L）

## 解决方案

### 修改 1: PlayerSkillManager.cs

**文件路径**：`Assets/AAAGame/Scripts/Game/Player/PlayerSkill/PlayerSkillManager.cs`

**修改内容**：
1. 修改 `Update()` 方法，遍历所有槽位（1-3）
2. 添加 `FindSkillBySlot()` 方法，根据槽位索引查找对应的技能

**修改后的代码**：
```csharp
private void Update()
{
    float dt = Time.deltaTime;

    // 更新所有技能的冷却
    for (int i = 0; i < Skills.Count; i++)
        Skills[i].Tick(dt);

    // 检测技能输入（槽位1-3对应J/K/L键）
    if (PlayerInputManager.Instance != null)
    {
        for (int slot = 1; slot <= 3; slot++)
        {
            if (PlayerInputManager.Instance.SkillDown(slot))
            {
                // 查找对应槽位的技能
                IPlayerSkill skill = FindSkillBySlot(slot);
                if (skill != null)
                {
                    skill.TryCast();
                }
            }
        }
    }
}

/// <summary>
/// 根据槽位索引查找技能
/// </summary>
private IPlayerSkill FindSkillBySlot(int slotIndex)
{
    for (int i = 0; i < Skills.Count; i++)
    {
        if (Skills[i].SkillId != 0)
        {
            // 从配置表读取槽位信息
            var tb = GF.DataTable.GetDataTable<PlayerSkillTable>();
            var row = tb?.GetDataRow(Skills[i].SkillId);
            if (row != null && row.SlotIndex == slotIndex)
            {
                return Skills[i];
            }
        }
    }
    return null;
}
```

**修改原因**：
- 遍历所有槽位，检测对应的按键输入
- 使用 `FindSkillBySlot()` 根据槽位索引查找技能
- 找到技能后调用 `TryCast()` 释放技能

### 修改 2: PlayerSkillSlot.cs

**文件路径**：`Assets/AAAGame/Scripts/UI/Item/PlayerSkillSlot.cs`

**修改内容**：
1. 修改 `BindSkill()` 方法，使用 `GetKeyNameBySlot()` 获取按键名称
2. 添加 `GetKeyNameBySlot()` 方法，将槽位索引映射到按键名称

**修改后的代码**：
```csharp
/// <summary>
/// 绑定技能数据
/// </summary>
public async void BindSkill(IPlayerSkill skill, SkillCommonConfig config, int slotIndex)
{
    m_Skill = skill;
    m_SkillConfig = config;
    m_SlotIndex = slotIndex;

    // 设置按键提示（显示对应的按键名称）
    if (varKeyHint != null)
    {
        string keyName = GetKeyNameBySlot(slotIndex);
        varKeyHint.text = keyName;
    }

    // ... 其他代码保持不变 ...
}

/// <summary>
/// 根据槽位索引获取对应的按键名称
/// </summary>
private string GetKeyNameBySlot(int slotIndex)
{
    return slotIndex switch
    {
        1 => "J",
        2 => "K",
        3 => "L",
        _ => slotIndex.ToString()
    };
}
```

**修改原因**：
- 显示实际的按键名称（J/K/L）而不是槽位数字（1/2/3）
- 提高用户体验，让玩家知道按哪个键释放技能

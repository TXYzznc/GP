## 功能需求

### 核心功能

1. **获取中心位置** - `GetCenterPosition(GameObject/Transform)`
   - 优先使用 Renderer.bounds.center
   - 其次使用 Collider.bounds.center
   - 最后使用 transform.position + 默认偏移
   - 支持通用 GameObject/Transform（不只是 ChessEntity）

2. **获取底部位置** - `GetBottomPosition(GameObject/Transform)`
   - 使用 Renderer/Collider 的 bounds.min.y
   - 返回模型实际底部位置

3. **获取顶部位置** - `GetTopPosition(GameObject/Transform)`
   - 使用 Renderer/Collider 的 bounds.max.y

4. **按比例获取位置** - `GetPositionAtRatio(GameObject/Transform, float ratio)`
   - ratio 从 0 到 1，表示从底部到顶部的位置
   - 0 = 底部，0.5 = 中心，1 = 顶部
   - 灵活支持各种场景（特效生成、UI定位等）

5. **获取自定义高度位置** - `GetPositionAtHeight(GameObject/Transform, float heightOffset)`
   - 支持指定 Y 轴偏移量（相对于底部）

6. **获取包围盒信息** - `GetBounds(GameObject/Transform)`
   - 返回实体的完整包围盒

7. **计算底部偏移量** - `CalculateBottomOffset(GameObject/Transform)`
   - 用于棋子放置时的底部对齐
   - 返回从 transform.position 到模型底部的偏移量

### 辅助功能

- **调试日志** - 可选的详细日志输出
- **空值检查** - 安全的空引用处理
- **性能优化** - 避免不必要的组件查找

## 技术方案

### 工具类设计

```csharp
/// <summary>
/// 实体坐标获取工具类
/// 提供统一的实体位置获取方法，避免重复实现
/// </summary>
public static class EntityPositionHelper
{
    // 默认高度偏移（当没有 Renderer/Collider 时使用）
    private const float DEFAULT_HEIGHT_OFFSET = 1f;
    
    // 核心方法
    public static Vector3 GetCenterPosition(ChessEntity entity, bool enableLog = false)
    public static Vector3 GetBottomPosition(ChessEntity entity)
    public static Vector3 GetTopPosition(ChessEntity entity, bool enableLog = false)
    public static Vector3 GetPositionAtHeight(ChessEntity entity, float heightOffset)
    public static Bounds? GetBounds(ChessEntity entity)
    
    // 内部辅助方法
    private static Renderer GetRenderer(ChessEntity entity)
    private static Collider GetCollider(ChessEntity entity)
}
```

### 实现策略

1. **组件查找顺序**：
   - Renderer（优先，通常用于渲染的模型）
   - Collider（其次，用于物理碰撞）
   - Transform + 默认偏移（最后，保底方案）

2. **性能考虑**：
   - 不缓存组件引用（保持无状态）
   - 使用 `GetComponentInChildren<T>()` 查找子对象
   - 调用频率不高（非每帧调用），性能影响可接受

3. **错误处理**：
   - 空引用检查
   - 返回安全的默认值
   - 可选的调试日志

## 使用示例

### 获取中心位置（用于瞄准）
```csharp
// 旧代码
Vector3 targetCenter = GetEntityCenterPosition(target);

// 新代码
Vector3 targetCenter = EntityPositionHelper.GetCenterPosition(target);
```

### 获取自定义高度位置（用于特效生成）
```csharp
// 旧代码
Vector3 effectPos = entity.transform.position + Vector3.up * 1.5f;

// 新代码
Vector3 effectPos = EntityPositionHelper.GetPositionAtHeight(entity, 1.5f);
```

### 获取包围盒（用于范围检测）
```csharp
// 新代码
Bounds? bounds = EntityPositionHelper.GetBounds(entity);
if (bounds.HasValue)
{
    float height = bounds.Value.size.y;
}
```

## 注意事项

1. **不要在 Update 中频繁调用** - 组件查找有一定开销
2. **优先使用缓存** - 如果需要多次获取同一实体的位置，建议缓存结果
3. **调试日志** - 生产环境关闭日志输出，避免性能影响

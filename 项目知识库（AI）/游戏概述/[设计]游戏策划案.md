# 游戏策划案

## 一、核心机制

**四大核心玩法的融合**：卡牌 + 自走棋 + 肉鸽 + 搜打撤

---

## 二、核心循环

### 2.1 基地准备阶段
配置卡组（神话生物召唤卡、战斗策略卡）、升级科技（Buff、圣水携带量、属性等）、购买补给、选择召唤师身份、升级技能树、神话生物备战（穿戴宝物）、整理物品与卡组。

### 2.2 地图探索（PVE）
进入地图 → 探索场景 → 收集资源/触发事件 → 遭遇敌人。

玩家通过移动/探索触发事件。无法跳跃但可通过传送到达特定地点（地图设计）。

### 2.3 战斗阶段
1. **战斗触发**：敌人到达一定距离时触发战斗UI并切换战斗场景
2. **准备阶段**：部署召唤物、穿戴装备、选择策略卡
3. **战斗进行**：
   - 自动：召唤物自动攻击，玩家自动普攻
   - 手动：使用策略卡（消耗灵力）、手动释放技能、控制索敌与站位
4. **战斗结算**：
   - 胜利：获得经验、掉落物，棋子血量状态继承
   - 失败：召唤师阵亡（恢复满状态），玩家受到惩罚

每个事件胜利后获得金币，用于升级、进阶、购买装备与Buff。

**装备机制**：玩家死亡时，召唤物装备遗失在原地。支持给装备上锁（最多2个），降低死亡惩罚。

### 2.4 撤离/结算
- **成功**：在地图中获得钥匙 → 携带战利品撤离
- **失败**：中途死亡 → 丢失大部分战利品，保留少量保底奖励

**钥匙**：撤离必要道具，击败Boss必定掉落，场景中有概率搜索到。

### 2.5 重要说明
- **召唤物等级**：无全局等级，每局游戏开始时等级为1级，通过游戏活动强化。强化1-2次后可选择进阶路线（每个生物多条路线）
- **玩家与召唤师**：召唤师是玩家战斗状态。召唤师被击败时，玩家结束此次战斗并受到惩罚（非游戏结束）

## 三、召唤师系统

### 3.1 属性

| 属性 | 说明 |
|------|------|
| **等级** | 玩家等级，达到20/30/40/45/50级时可进行进阶 |
| **阶级** | 进阶后获得属性、技能、特殊效果提升 |
| **生命值** | 本体血量。当所有召唤物被封印后敌方直接攻击本体。血量为零则传送回基地（仅带走部分战利品） |
| **灵力值** | 核心资源。用于技能、策略卡、合成圣水 |
| **统御力** | 基础为2，每升一阶提升1-2点。每点表示一个基础人口召唤数量（五阶基础统御力10点） |
| **技能点** | 升级或特殊活动/道具获得。用于技能升级。技能可重置以获得全部技能点 |

### 3.2 进阶系统
进阶获得属性/技能/特殊效果提升。进阶任务包含：
1. 提交特殊材料
2. 完成挑战任务

随着等阶增加，基础友方召唤数量增加。

### 3.3 技能系统

**每职业技能构成**：
- 被动技能：1个职业固定被动 + 2个分支特化被动二选一
- 主动技能：1个职业固定主动 + 4个分支特化主动四选二

**技能解锁进度**：
- 第一阶：固定被动和主动
- 第二阶：无新技能
- 第三阶：分支特化主动技能一
- 第四阶：分支特化被动技能
- 第五阶：分支特化主动技能二

### 3.4 职业详解

#### 狂战士

**固定技能**：
- 【狂怒之心】（被动）：友方生命值<50%时伤害+15%。玩家生命值<40%时自身伤害再+15%，全体召唤物伤害效果翻倍
- 【战意激昂】（主动）：消耗20血量，10秒内提升召唤物攻速+20%、伤害+15%

**路线一（战争天灾）**：狂战→战争领主→霸王→War→战争天灾
- 【王者号令】（三阶）：命令所有召唤物强力普攻（+50%）并嘲讽400码敌人15秒
- 【钢铁洪流】（四阶）：场上召唤物>4个时获得冲锋效果（攻速-25%，伤害+30%，每次攻击眩晕0.5s）
- 【天灾降临】（五阶）：指定区域召唤战争风暴，持续造成巨额伤害5秒，为友方提供护盾和生命偷取

**路线二（末日裁决）**：狂战→战争领主→寂灭者→暗夜君王→终末裁决
- 【寂灭斩】（三阶）：超高伤害斩击，击败目标则重置冷却
- 【孤影】（四阶）：身边无友方时伤害大幅提升，获得高额闪避率
- 【裁决之刻】（五阶）：标记敌人，延迟后无视70%防御的终极斩击，击败目标对其他敌人造成范围恐惧

#### 术士

**固定技能**：
- 【暗影咒体】（被动）：所有友方对诅咒敌人伤害+10%。诅咒机制：术士受伤80%概率诅咒来源，造成伤害30%概率诅咒敌人
- 【生命虹吸】（主动）：吸取敌人生命值，等量治疗自己和最近友方，敌人进入虚弱（移速-20%，攻速-15%）

**路线一（命运编织者）**：术士→咒法者→织秘者→诡秘导士→命运编织者
- 【命运丝线】（三阶）：连接敌人与友方，敌人受伤50%转移为治疗
- 【厄运编织】（四阶）：诅咒30%几率传染附近敌人
- 【终末之咒】（五阶）：BOSS/精英施加终末印记，期间伤害50%累积，结束时一次爆发（无视防御）

**路线二（冥府主宰）**：术士→咒法者→死灵法师→亡语者→冥府主宰
- 【死亡法术】（三阶）：真实伤害（无视防御）。目标生命值<25%直接秒杀（对BOSS无效）
- 【亵渎之地】（四阶）：被击败敌人30%概率复活为亡灵仆从（持续30秒，最多3个）
- 【黄泉之门】（五阶）：开启冥府之门（持续30秒），持续召唤亡灵生物，每8秒对范围敌人施加恐惧

#### 混沌术士

**固定技能**：
- 【混沌回响】（被动）：技能15%几率双倍伤害或治疗，10%几率半伤害，8%几率反噬自身
- 【混沌飞弹】（主动）：发射3-5发混沌能量弹，极低几率使目标变形、眩晕或恢复能量

**路线一（因果观测者）**：混沌→无序术士→命运赌徒→概率操控者→因果观测者
- 【命运骰子】（三阶）：投掷六面骰子，1点沉默3秒，2-3小增益，4-5中增益（如攻击+50%），6点史诗增益
- 【幸运储备】（四阶）：储存最多2次骰子使用，投1点概率降至1/12，4点以上重置技能冷却1秒
- 【逆乱因果】（五阶）：指定事件必然发生（技能最佳效果/重置所有冷却/时间静止/复活友方）

**路线二（混乱化身）**：混沌→无序术士→混乱之源→混乱天灾→Chaos!
- 【混乱新星】（三阶）：周围能量波，随机伤害眩晕、治疗、加速、减速、互换位置、魅惑
- 【混乱领域】（四阶）：8码范围永久领域，敌人攻击20%落空，10%攻击友军，友军10%双倍伤害
- 【混沌迷雾】（五阶）：10秒内战场规则混乱，每秒最多触发一次随机效果（位置交换/冷却重置/体型伤害变化），结束时巨额混沌伤害

#### 德鲁伊

**固定技能**：
- 【大地恩泽】（被动）：每10秒产生自然果实（火/土/水/风），拾取获得对应增益（暴击、防御恢复、冷却恢复、移速攻速），相同果实叠加持续时间
- 【自然之子】（主动）：在人兽形态间切换（冷却5秒）。人形态每秒恢复1%最大生命值；兽形态护甲/魔抗/移速/攻击+15%但每秒消耗1%灵力。生命值或灵力≤50%时召唤树精（冷却150秒）

**路线一（自然之魂）**：德鲁伊→自然法师→荒野行者→万兽统御→自然之魂
- 【召唤古树】（三阶）：召唤强大树人守卫，高生命值和范围嘲讽，周期性为友军施加护盾
- 【统御】（四阶）：所有自然生物伤害生命值+25%，兽形态时召唤物获得狂怒（攻速+20%）
- 【万物苏生】（五阶）：广阔区域降生命之雨6秒，每秒恢复友方5%生命值并净化负面效果，被击败友方50%概率复活（恢复30%血量）

**路线二（蛮荒圣兽）**：德鲁伊→自然法师→蛮荒掠食者→狂野巨兽→蛮荒圣兽
- 【野蛮冲撞】（三阶）：朝方向猛烈冲撞，伤害并眩晕1.5秒
- 【血怒】（四阶）：击败单位或召唤物被击败时获得血怒层数（最10层），每层伤害+3%，满层时攻击造成流血3秒（每秒损失5%当前生命值）
- 【圣兽降临】（五阶）：变身巨大蛮荒圣兽，获得最大生命值30%护盾（持续15秒）。形态下无法用其他技能但拥有毁灭普攻和两个专属技能【战争践踏】【毁灭咆哮】，普攻改为践踏（眩晕0.5秒）

## 四、召唤物系统

### 4.1 属性
生命值、能量、技能、等阶、羁绊类型

**装备栏**：3个宝物格 + 2个装备格

### 4.2 局内成长
使用金币进阶（最高四阶）。进阶到三阶时选择进阶路线。四阶需特殊进阶道具（探索/商店获得）。每个生物可能多条进阶路线。

### 4.3 人口占比
- 特殊/衍生召唤物（自然、亡灵等）：不占人口
- 正常神话召唤物：1人口
- 局内获得支援角色：1人口
- 四阶神话召唤物：2人口

### 4.4 血量继承
召唤物生命值在整局游戏内继承（脱离战斗不自动回复，需通过技能/道具/回到基地恢复）

## 五、策略卡系统

**使用机制**：
- 进入游戏前配置卡组（8种卡片）
- 战斗时刷出八张卡片，随机刷新
- 每轮战斗结束后补满卡片（可消耗1金币主动刷新）
- 使用卡片消耗召唤师灵力值

非单位卡牌，提供治疗、伤害、控制等辅助功能。

## 六、探索与地图系统

### 6.1 场景设计
- 预设场景，游戏开始后动态烘焙（或提前保存固定场景烘焙信息）
- 玩家可按【M】打开地图查看当前位置
- Boss在某区域内随机属性，每次进入可能刷出不同Boss（同一关多个Boss，增加变化性）

### 6.2 事件系统

**触发方式**：交互触发或进入区域触发

**事件类型**：
- 宝箱：直接获取资源
- 商人：消耗货币购买局内强化（装备、进阶道具等）
- 强化祭坛：消耗血量/金币献祭获得Buff或装备
- 老虎机：随机抽取道具

## 七、敌人AI设计

### 7.1 场景敌人状态机（5种状态）

| 状态 | 说明 |
|------|------|
| **巡逻(Idle)** | 在设定巡逻范围内随机移动 |
| **警戒(Alert)** | 玩家进入警戒范围，警戒值上升。玩家距离越近上升越快。敌人原地静止直至玩家离开或警戒条满；离开后警戒条下降 |
| **追击(Chase)** | 警戒条满时追击玩家。向巡逻范围内敌人广播玩家位置，收到广播敌人转为追击。敌人有固定活动区域，离开后回到区域中心 |
| **战斗(Combat)** | 敌人到达玩家距离R1内触发战斗。先检测R2范围内所有敌人（R2>R1），根据敌人数量修改难度（敌人数=奖励份数） |
| **休息(Rest)** | 原地静止休息，玩家接近不触发警戒。可直接进入战斗或离开。定期在休息与巡逻间切换 |

### 7.2 战斗敌人
使用行为树控制，自动追击、普攻和使用技能。

### 7.3 敌人类型
- **普通敌人**：基础掉落
- **精英敌人**：高价值掉落
- **Boss**：必定掉落钥匙

特殊敌人（所有Boss和部分特殊敌人，头上有标识），战胜后可消耗圣水净化获得对应召唤卡牌。

## 八、战斗系统

### 8.1 战斗流程

1. **遭遇**：地图触发战斗事件
2. **准备**：显示敌人阵容 → 部署召唤物 → 穿戴装备/进阶 → 点击开始
3. **战斗**：自动与手动相结合（详见2.3）
4. **结算**：胜利获得经验和掉落；失败受到惩罚

### 8.2 圣水机制

**定义**：消耗性资源

**用途**：
- 战斗胜利后，消耗圣水净化"污神"获得稀有卡牌/素材
- 特殊宝箱需消耗圣水净化才能开启

**补充来源**：基地补充、地图稀有事件

## 九、数据与网络

### 9.1 数据库
1. 进入游戏时注册账号并上传云端数据库
2. 存档默认本地保存，可上传云端（云存档）或从云端下载
3. 数据保存使用二进制格式，强化安全性

### 9.2 网络联机（Mirror）

**局域网对战（P2P）**：
- P2P模式或局域网广播发现房间
- 逻辑：同步双方卡组数据 → 进入战斗场景 → 仅同步战斗指令和随机数种子（Lockstep或状态同步）

**PVP线上联机对战**：暂定，后续版本开发加入

## 十、彩蛋与特殊物品

### 彩蛋
- 会唱各种哈基米音乐的小猫宠物
- 场景老虎机：可能获得各种Buff

### 特殊物品
- **转职之心**：稀有道具，用于玩家转职

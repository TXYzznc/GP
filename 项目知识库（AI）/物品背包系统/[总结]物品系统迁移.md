`PlayerAccountData.cs` 中存在旧的 `ItemData` 类定义，与新的物品系统 `AAAGame.Item.ItemData` 冲突。

## 需要修改的内容

### 1. 删除旧的 ItemData 类定义

**位置**：第 428-433 行

**需要删除的代码**：
```csharp
[Serializable]
public class ItemData
{
    public int ItemId;
    public int Count;
    public long ObtainTime;
}
```

**原因**：这个简单的 ItemData 类已经被新的物品系统取代，新系统使用 `AAAGame.Item` 命名空间下的完整物品类。

### 2. 修改背包相关方法

由于背包系统现在由 `InventoryManager` 统一管理，`PlayerSaveData` 中的背包相关方法需要调整。

#### 方案A：保留简化的存档数据结构（推荐）

将背包数据简化为物品ID和数量的列表，用于存档：

**修改位置**：第 238-260 行

**原代码**：
```csharp
/// <summary>

/// </summary>
public List<ItemData> GetInventoryItems()

    if (string.IsNullOrEmpty(InventoryItems))
```

**修改后的代码**：

## 修改说明

### 设计思路

1. **存档数据简化**：`PlayerSaveData` 只存储简化的物品数据（ID、数量、时间等）
2. **运行时数据完整**：游戏运行时使用 `InventoryManager` 管理完整的物品实例
3. **数据转换**：在加载/保存存档时，在简化数据和完整物品实例之间转换

### 数据流程

```
存档加载时：
InventoryItemSaveData → InventoryManager.AddItem(itemId, count)

存档保存时：
InventoryManager.GetAllSlots() → InventoryItemSaveData
```

### 优点

1. **存档文件小**：只存储必要的数据
2. **灵活性高**：物品系统可以独立演化，不影响存档格式
3. **兼容性好**：旧存档可以通过数据迁移升级

## 后续工作

修改完成后，需要在 `PlayerAccountDataManager` 中添加：

1. **存档加载时**：将 `InventoryItemSaveData` 转换为物品实例并添加到 `InventoryManager`
2. **存档保存时**：从 `InventoryManager` 获取物品数据并转换为 `InventoryItemSaveData`

示例代码：

```csharp
// 加载背包数据
private void LoadInventoryData(PlayerSaveData saveData)
{
    var inventoryItems = saveData.GetInventoryItems();
    foreach (var itemData in inventoryItems)
    {
        InventoryManager.Instance.AddItem(itemData.ItemId, itemData.Count);
    }
}

// 保存背包数据
private void SaveInventoryData(PlayerSaveData saveData)
{
    var saveDataList = new List<InventoryItemSaveData>();
    var slots = InventoryManager.Instance.GetAllSlots();
    
    foreach (var slot in slots)
    {
        if (!slot.IsEmpty)
        {
            var itemSaveData = new InventoryItemSaveData
            {
                ItemId = slot.ItemId,
                Count = slot.Count,
                ObtainTime = DateTimeOffset.UtcNow.ToUnixTimeSeconds(),
                UniqueId = slot.ItemStack.Item.UniqueId
            };
            saveDataList.Add(itemSaveData);
        }
    }
    
    saveData.SetInventoryItems(saveDataList);
}
```

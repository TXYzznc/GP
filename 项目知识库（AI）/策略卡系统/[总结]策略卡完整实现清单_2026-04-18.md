## 🔧 核心实现组件

### 目标选择器（8种）
```
SelfSelector                      → 自身
AllAlliesSelector                 → 全体友方（含召唤师）
AllAllyExcludeSummonerSelector    → 全体友方（不含召唤师）
AllEnemiesSelector                → 敌方全体
SingleAlly/ClosestAllySelector    → 单体友方（最近）
SingleEnemy/ClosestEnemySelector  → 单体敌方（最近）
AlliesInRadiusSelector            → 范围内友方
EnemiesInRadiusSelector           → 范围内敌方
```

### 效果应用器（4种）
```
DamageApplier                     → 固定伤害
DamageWithCoefficientApplier      → 系数伤害（基于施法者属性）
HealApplier                       → 治疗
BuffApplier                       → 命中Buff
InstantBuffApplier                → 立即Buff
```

### 辅助工具
```
CardEffectHelper
  - DealDamage()      → 造成伤害
  - HealTarget()      → 治疗目标
  - ApplyBuff()       → 应用Buff
  - PlayEffect()      → 播放特效
```

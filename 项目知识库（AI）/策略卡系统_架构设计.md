## 🎬 扇形排列算法

```
算法目标：计算卡牌在圆弧上的位置和旋转

输入：
  - 卡牌总数 N
  - 圆弧半径 R
  - 展开角度 A
  - 圆心位置 (cx, cy)

计算步骤：

1. 角度范围计算
   ├─ fanRangeRad = A * π/180         // 角度转弧度
   ├─ halfFanRange = fanRangeRad / 2  // 左右对称
   └─ angleStep = fanRangeRad / (N-1) // 单张卡的角度步长

2. 逐张卡计算位置
   对于第 i 张卡 (i = 0 to N-1):
   ├─ angle = -halfFanRange + i * angleStep  // 当前角度
   ├─ x = sin(angle) * R
   ├─ y = cos(angle) * R - R
   ├─ position = (cx + x, cy + y)
   └─ rotation = -angle * 180/π  // 与切线对齐

3. 拖拽时临时偏移
   当用户拖拽卡牌时：
   ├─ 被拖卡牌：返回容器进行拖动
   ├─ 中间卡牌：根据拖拽方向左移或右移
   └─ 偏移距离 = cardWidth * 0.5

图形化展示（10张卡，展开50°）：

            0°
            ↑
            │
     ╱  ╱  │  ╲  ╲
   ╱  ╱  4│3  ╲  ╲
  ╱  ╱  5  │  2  ╲  ╲
 ╱  ╱  6   │   1  ╲  ╲
╱  ╱   7   │    0  ╲  ╲  ◄─ 圆心(cx, cy)
╲  ╲   8   │    9  ╱  ╱
 ╲  ╲  9   │  10 ╱  ╱
  ╲  ╲     │    ╱  ╱
   ╲  ╲    │   ╱  ╱
    ╲  ╲   │  ╱  ╱
     ╲  ╲  │ ╱  ╱
       ╲  ╲│╱  ╱
        180°

核心代码：
float angle = -halfFanRangeRad + i * angleStep;
float x = Mathf.Sin(angle) * m_FanRadius;
float y = Mathf.Cos(angle) * m_FanRadius - m_FanRadius;
transforms[i].AnchoredPos = center + new Vector2(x, y);
transforms[i].RotationZ = -angle * Mathf.Rad2Deg;
```

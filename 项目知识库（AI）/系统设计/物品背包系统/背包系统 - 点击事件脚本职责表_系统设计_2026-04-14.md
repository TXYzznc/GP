# 背包系统 - 点击事件脚本职责表

## 快速查询表

### 左键点击 → 显示物品详情

| 层级 | 脚本 | 类 | 方法 | 职责 |
|------|------|-----|------|------|
| 1️⃣ 接收点击 | InventoryItemUI.cs | InventoryItemUI | `OnPointerClick(PointerEventData)` | 接收点击事件（实现 IPointerClickHandler） |
| 2️⃣ 判断按钮 | InventoryItemUI.cs | InventoryItemUI | `OnPointerClick()` 中的 if | 判断是否左键（`button == Left`） |
| 3️⃣ 处理左键 | InventoryItemUI.cs | InventoryItemUI | `OnItemClick()` | 左键处理：调用 ShowItemDetailPanel() |
| 4️⃣ 显示详情 | InventoryItemUI.cs | InventoryItemUI | `ShowItemDetailPanel()` | 获取 InventoryUI，调用其 ShowItemDetail() |
| 5️⃣ 更新面板 | InventoryUI.cs | InventoryUI | `ShowItemDetail(ItemStack)` | 更新 UI 显示：名称、图标、重量、描述等 |

### 右键点击 → 显示上下文菜单

| 层级 | 脚本 | 类 | 方法 | 职责 |
|------|------|-----|------|------|
| 1️⃣ 接收点击 | InventoryItemUI.cs | InventoryItemUI | `OnPointerClick(PointerEventData)` | 接收点击事件（实现 IPointerClickHandler） |
| 2️⃣ 判断按钮 | InventoryItemUI.cs | InventoryItemUI | `OnPointerClick()` 中的 else if | 判断是否右键（`button == Right`） |
| 3️⃣ 处理右键 | InventoryItemUI.cs | InventoryItemUI | `OnItemRightClick(PointerEventData)` | 右键处理：调用 ShowContextMenu(position) |
| 4️⃣ 显示菜单 | InventoryItemUI.cs | InventoryItemUI | `ShowContextMenu(Vector2)` | 获取菜单，调用其 ShowContextMenu() |
| 5️⃣ 菜单逻辑 | ItemContextMenu.cs | ItemContextMenu | `ShowContextMenu()` | 获取物品配置，刷新菜单选项，位置计算，显示菜单 |
| 6️⃣ 菜单按钮 | ItemContextMenu.cs | ItemContextMenu | `OnClickUse/Discard/Split()` | 处理菜单选项（使用、丢弃、拆分） |

### 📌 InventoryItemUI.cs（单一职责模式）

**完整文件位置**：`Assets/AAAGame/Scripts/UI/Item/InventoryItemUI.cs`

**核心职责**：点击事件的总入口和初步分发

```csharp
public partial class InventoryItemUI : UIItemBase, IPointerClickHandler
{
    // ============ 职责 1：接收点击事件 ============
    public void OnPointerClick(PointerEventData eventData)
    {
        // 分发给左/右键处理器
        if (eventData.button == PointerEventData.InputButton.Left)
            OnItemClick();           // ← 职责 2
        else if (eventData.button == PointerEventData.InputButton.Right)
            OnItemRightClick(eventData);  // ← 职责 3
    }

    // ============ 职责 2：左键处理 ============
    private void OnItemClick()
    {
        // 检查物品存在性
        // 调用 ShowItemDetailPanel()
        ShowItemDetailPanel();
    }

    // ============ 职责 3：右键处理 ============
    private void OnItemRightClick(PointerEventData eventData)
    {
        // 检查物品存在性
        // 调用 ShowContextMenu()
        ShowContextMenu(eventData.position);
    }

    // ============ 职责 4：显示详情面板 ============
    private void ShowItemDetailPanel()
    {
        // 获取 InventoryUI（必须是背包中）
        var inventoryUI = GetComponentInParent<InventoryUI>();
        if (inventoryUI != null)
        {
            // 调用 InventoryUI 来更新详情显示
            inventoryUI.ShowItemDetail(m_ItemStack);
        }
        // 注意：仓库物品无法显示详情（因为无法获取 InventoryUI）
    }

    // ============ 职责 5：显示上下文菜单 ============
    private void ShowContextMenu(Vector2 position)
    {
        // 获取菜单（从 InventoryUI 获取）
        var contextMenu = GetItemContextMenu();
        if (contextMenu != null)
        {
            // 获取格子索引
            var slotUI = GetComponentInParent<InventorySlotUI>();
            int slotIndex = slotUI != null ? slotUI.SlotIndex : -1;

            // 调用菜单显示逻辑
            contextMenu.ShowContextMenu(m_ItemStack, slotIndex, position);
        }
    }

    // ============ 职责 6：获取菜单 ============
    private ItemContextMenu GetItemContextMenu()
    {
        // 从 InventoryUI 获取菜单（因为菜单在 InventoryUI 下）
        var inventoryUI = GetComponentInParent<InventoryUI>();
        if (inventoryUI != null)
            return inventoryUI.GetItemContextMenu();

        return null;
    }
}
```

**关键约束**：
- ❌ 不处理物品实际使用/删除逻辑（那是 ItemContextMenu 的职责）
- ❌ 不显示/隐藏菜单（那是 ItemContextMenu 的职责）
- ✅ 只负责点击事件的接收和初步分发

### 📌 InventoryUI.cs（UI 面板管理）

**完整文件位置**：`Assets/AAAGame/Scripts/UI/InventoryUI.cs`

**核心职责**：
1. 管理整个背包 UI 面板
2. 提供物品详情显示接口
3. 提供上下文菜单访问接口

```csharp
public partial class InventoryUI : UIFormBase
{
    // ============ 职责 1：显示物品详情 ============
    public void ShowItemDetail(ItemStack itemStack)
    {
        if (itemStack == null || itemStack.IsEmpty)
        {
            ClearItemDetail();
            return;
        }

        var item = itemStack.Item;
        var itemData = item.ItemData;

        // 显示物品信息
        if (varItemName != null)
            varItemName.text = item.Name;

        if (varItemDetailImg != null)
            _ = LoadItemDetailIconAsync(itemData.GetIconId());

        if (varRarityText != null)
        {
            // 显示稀有度和颜色
            var itemTable = GF.DataTable.GetDataTable<ItemTable>();
            var row = itemTable?.GetDataRow(item.ItemId);
            if (row != null)
            {
                varRarityText.text = $"稀有度: {row.Rarity}";
                varRarityText.color = RarityColorHelper.GetColor(row.Rarity);
            }
        }

        if (varWeightText != null)
        {
            var itemTable = GF.DataTable.GetDataTable<ItemTable>();
            var row = itemTable?.GetDataRow(item.ItemId);
            if (row != null)
                varWeightText.text = $"重量: {row.Weight}";
        }

        if (varDescriptionText != null)
            varDescriptionText.text = itemData.Description ?? "暂无描述";

        DebugEx.Success("InventoryUI", $"物品详情显示完成: {item.Name}");
    }

    // ============ 职责 2：获取上下文菜单 ============
    public ItemContextMenu GetItemContextMenu()
    {
        var contextMenu = GetComponentInChildren<ItemContextMenu>(true);

        if (contextMenu == null)
        {
            DebugEx.Warning("InventoryUI", "未找到 ItemContextMenu 组件");
        }

        return contextMenu;
    }

    // ============ 职责 3：清空物品详情 ============
    private void ClearItemDetail()
    {
        if (varItemName != null)
            varItemName.text = "";

        if (varItemDetailImg != null)
        {
            varItemDetailImg.sprite = null;
            varItemDetailImg.color = new Color(1, 1, 1, 0);
        }

        if (varRarityText != null)
            varRarityText.text = "";

        if (varWeightText != null)
            varWeightText.text = "";

        if (varDescriptionText != null)
            varDescriptionText.text = "";

        DebugEx.Log("InventoryUI", "物品详情已清空");
    }

    // ============ 职责 4：异步加载图标 ============
    private async UniTask LoadItemDetailIconAsync(int iconId)
    {
        if (iconId <= 0)
            return;

        try
        {
            if (varItemDetailImg != null)
            {
                await GameExtension.ResourceExtension.LoadSpriteAsync(iconId, varItemDetailImg, 1f, null);
                varItemDetailImg.color = Color.white;
                DebugEx.Log("InventoryUI", $"物品详情图标加载成功: IconId={iconId}");
            }
        }
        catch (System.Exception e)
        {
            DebugEx.Error("InventoryUI", $"加载物品详情图标异常: IconId={iconId}, Error:{e.Message}");
        }
    }
}
```

**关键约束**：
- ❌ 不处理 ItemContextMenu 的显示/隐藏（菜单自己管理）
- ❌ 不处理菜单按钮的点击（那是 ItemContextMenu 的职责）
- ✅ 只负责显示物品详情和提供菜单访问接口

### 📌 ItemContextMenu.cs（菜单管理和操作）

**完整文件位置**：`Assets/AAAGame/Scripts/UI/Item/ItemContextMenu.cs`

**核心职责**：显示菜单、处理菜单选项、执行物品操作

```csharp
public partial class ItemContextMenu : UIItemBase
{
    // ============ 职责 1：初始化菜单按钮 ============
    protected override void OnInit()
    {
        base.OnInit();

        // 自动查找菜单按钮
        var buttons = GetComponentsInChildren<Button>(true);
        if (buttons.Length > 0) m_UseBtn = buttons[0];
        if (buttons.Length > 1) m_SplitBtn = buttons[1];
        if (buttons.Length > 2) m_DiscardBtn = buttons[2];

        gameObject.SetActive(false);
    }

    // ============ 职责 2：显示菜单 ============
    public void ShowContextMenu(ItemStack itemStack, int slotIndex, Vector2 position)
    {
        if (itemStack == null || itemStack.IsEmpty)
            return;

        m_CurrentItemStack = itemStack;
        m_CurrentSlotIndex = slotIndex;

        // 获取物品配置
        var itemTable = GF.DataTable.GetDataTable<ItemTable>();
        m_CurrentItemRow = itemTable?.GetDataRow(itemStack.Item.ItemId);

        // 刷新菜单选项（根据物品类型）
        RefreshMenuOptions();

        // 计算菜单位置（屏幕坐标→Canvas本地坐标）
        var rectTransform = GetComponent<RectTransform>();
        var canvas = GetComponentInParent<Canvas>();
        if (rectTransform != null && canvas != null)
        {
            RectTransformUtility.ScreenPointToLocalPointInRectangle(
                canvas.GetComponent<RectTransform>(),
                position,
                canvas.worldCamera,
                out var localPoint
            );

            rectTransform.anchoredPosition = localPoint;

            // 边界检查
            ClampMenuPositionToScreen(rectTransform, canvas.GetComponent<RectTransform>());
        }

        // 显示菜单
        gameObject.SetActive(true);
    }

    // ============ 职责 3：刷新菜单选项 ============
    private void RefreshMenuOptions()
    {
        HideAllButtons();

        int itemType = m_CurrentItemRow?.Type ?? -1;

        switch (itemType)
        {
            case 0: // 消耗品
                ShowButton(m_UseBtn, "使用", OnClickUse);
                if (m_CurrentItemStack.Count > 1)
                    ShowButton(m_SplitBtn, "拆分", OnClickSplit);
                ShowButton(m_DiscardBtn, "丢弃", OnClickDiscard);
                break;

            case 1: // 装备
                ShowButton(m_UseBtn, "装备", OnClickUse);
                ShowButton(m_DiscardBtn, "丢弃", OnClickDiscard);
                break;

            case 2: // 宝物
                ShowButton(m_DiscardBtn, "丢弃", OnClickDiscard);
                break;

            case 3: // 任务道具
                // 无操作
                break;
        }
    }

    // ============ 职责 4：处理"使用"操作 ============
    private void OnClickUse()
    {
        var inventoryManager = InventoryManager.Instance;
        if (inventoryManager != null)
        {
            if (inventoryManager.UseItem(m_CurrentSlotIndex))
            {
                DebugEx.Success("ItemContextMenu", $"使用物品: {m_CurrentItemStack.Item.Name}");
            }
        }

        HideContextMenu();
    }

    // ============ 职责 5：处理"丢弃"操作 ============
    private void OnClickDiscard()
    {
        var inventoryManager = InventoryManager.Instance;
        if (inventoryManager != null)
        {
            bool success = inventoryManager.RemoveItem(
                m_CurrentItemStack.Item.ItemId,
                m_CurrentItemStack.Count
            );
            if (success)
            {
                DebugEx.Success("ItemContextMenu", $"丢弃物品: {m_CurrentItemStack.Item.Name}");
            }
        }

        HideContextMenu();
    }

    // ============ 职责 6：处理"拆分"操作 ============
    private void OnClickSplit()
    {
        // TODO: 显示拆分对话框
        DebugEx.Log("ItemContextMenu", $"拆分物品: {m_CurrentItemStack.Item.Name}");

        HideContextMenu();
    }

    // ============ 职责 7：隐藏菜单 ============
    public void HideContextMenu()
    {
        gameObject.SetActive(false);
        m_CurrentItemStack = null;
        m_CurrentItemRow = null;

        DebugEx.Log("ItemContextMenu", "上下文菜单已隐藏");
    }

    // ============ 职责 8：边界检查 ============
    private void ClampMenuPositionToScreen(RectTransform menuRT, RectTransform canvasRT)
    {
        // 防止菜单超出屏幕边界
        var menuSize = menuRT.sizeDelta;
        var canvasSize = canvasRT.sizeDelta;
        var pos = menuRT.anchoredPosition;

        if (pos.x + menuSize.x / 2 > canvasSize.x / 2)
            pos.x = canvasSize.x / 2 - menuSize.x / 2;

        if (pos.y - menuSize.y / 2 < -canvasSize.y / 2)
            pos.y = -canvasSize.y / 2 + menuSize.y / 2;

        menuRT.anchoredPosition = pos;
    }
}
```

**关键约束**：
- ❌ 不处理 InventoryUI 的显示/隐藏
- ❌ 不处理物品详情的显示（那是 InventoryUI 的职责）
- ✅ 只负责菜单的显示、选项刷新、按钮处理

## 关键数据流向

```
用户左键点击
  │
  ├─ EventSystem 检测点击
  │
  ├─ InventoryItemUI.OnPointerClick() ← ⭐ 第一个接收
  │  │
  │  └─ InventoryItemUI.OnItemClick()
  │     │
  │     └─ InventoryItemUI.ShowItemDetailPanel()
  │        │
  │        └─ InventoryUI.ShowItemDetail(itemStack) ← 更新 UI
  │           │
  │           ├─ 更新 varItemName
  │           ├─ 更新 varItemDetailImg
  │           ├─ 更新 varRarityText
  │           └─ 更新 varDescriptionText
  │
  └─ 用户看到物品详情


用户右键点击
  │
  ├─ EventSystem 检测点击
  │
  ├─ InventoryItemUI.OnPointerClick() ← ⭐ 第一个接收
  │  │
  │  └─ InventoryItemUI.OnItemRightClick(eventData)
  │     │
  │     └─ InventoryItemUI.ShowContextMenu(position)
  │        │
  │        ├─ 获取 InventoryUI
  │        │
  │        └─ InventoryUI.GetItemContextMenu()
  │           │
  │           └─ ItemContextMenu.ShowContextMenu() ← ⭐ 显示菜单
  │              │
  │              ├─ RefreshMenuOptions()
  │              ├─ 计算位置
  │              └─ gameObject.SetActive(true)
  │
  └─ 用户看到菜单

用户点击菜单按钮
  │
  ├─ ItemContextMenu.OnClickUse/Discard/Split() ← ⭐ 处理操作
  │  │
  │  ├─ 调用 InventoryManager（实际执行操作）
  │  │  ├─ UseItem(slotIndex)
  │  │  └─ RemoveItem(itemId, count)
  │  │
  │  └─ ItemContextMenu.HideContextMenu()
  │
  └─ 菜单关闭，背包刷新
```

## 脚本间的依赖关系

```
InventoryItemUI
  ├─ depends on: InventorySlotUI（获取格子索引）
  ├─ depends on: InventoryUI（获取菜单和显示详情）
  └─ depends on: ItemContextMenu（通过 InventoryUI 间接访问）

InventoryUI
  ├─ depends on: InventoryItemUI（无直接依赖，但监听其事件）
  └─ depends on: ItemContextMenu（查找并提供给 InventoryItemUI）

ItemContextMenu
  ├─ depends on: InventoryManager（执行物品操作）
  ├─ depends on: ItemTable（获取物品配置）
  └─ no depends on InventoryItemUI/InventoryUI（独立运作）
```

**优点**：低耦合，高内聚
- 修改菜单逻辑不影响详情显示
- 修改详情显示不影响菜单逻辑
- InventoryItemUI 只负责分发，不处理具体业务

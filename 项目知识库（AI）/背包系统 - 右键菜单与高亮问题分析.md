### 🔴 问题排查清单

#### 1️⃣ ItemContextMenu 未添加到 Prefab

**现象**：右键点击后没有菜单显示

**排查方法**：
```
检查 CombatPreparationUI.prefab（或 InventoryUI.prefab）的层级：
  ├─ Canvas
  │  ├─ InventoryUI（背包容器）
  │  │  ├─ Content（格子列表）
  │  │  └─ Details Panel（物品详情）
  │  └─ ItemContextMenu？❌ 缺少
```

**解决方案**：
- [ ] 确保 ItemContextMenu 预制体已添加到 InventoryUI 或 Canvas 下
- [ ] ItemContextMenu 应该是 Canvas 的直接子物体（以便显示在最上层）

#### 2️⃣ ItemContextMenu 的按钮未初始化

**问题代码** - ItemContextMenu.cs:18-20
```csharp
// 菜单选项按钮
private Button m_UseBtn;
private Button m_SplitBtn;
private Button m_DiscardBtn;
// ❌ 这三个字段从未赋值！
```

**原因**：
- 这三个 Button 没有在 OnInit() 中初始化
- RefreshMenuOptions() 调用 ShowButton() 时，按钮为 null，操作无效

**解决方案**：需要实现 OnInit() 方法来查找这些按钮组件

#### 3️⃣ 菜单位置可能超出屏幕

**问题代码** - ItemContextMenu.cs:73-77
```csharp
// 设置位置并显示
var rectTransform = GetComponent<RectTransform>();
if (rectTransform != null)
{
    rectTransform.position = position;  // ⚠️ 屏幕坐标，可能超出范围

```

**问题**：
- `position` 是屏幕坐标，直接赋值给 RectTransform.position 不对
- 应该转换为 Canvas 本地坐标（如 InventoryDragHandler 中的方法）

#### Step 2：修复菜单位置计算

将屏幕坐标转换为 Canvas 本地坐标：

```csharp
public void ShowContextMenu(ItemStack itemStack, int slotIndex, Vector2 position)
{
    // ... 现有代码 ...

    // 🔧 修复：转换屏幕坐标为 Canvas 本地坐标
    var rectTransform = GetComponent<RectTransform>();
    var canvas = GetComponentInParent<Canvas>();

    if (rectTransform != null && canvas != null)
    {
        RectTransformUtility.ScreenPointToLocalPointInRectangle(
            canvas.GetComponent<RectTransform>(),
            position,
            canvas.worldCamera,
            out var localPoint
        );

        rectTransform.anchoredPosition = localPoint;

        // 🔧 可选：边界检查，防止菜单超出屏幕
        ClampMenuPosition(rectTransform, canvas.GetComponent<RectTransform>());
    }

    gameObject.SetActive(true);
}

/// <summary>
/// 限制菜单位置在屏幕范围内
/// </summary>
private void ClampMenuPosition(RectTransform menuRT, RectTransform canvasRT)
{
    var menuSize = menuRT.sizeDelta;
    var canvasSize = canvasRT.sizeDelta;
    var pos = menuRT.anchoredPosition;

    // 右边界检查
    if (pos.x + menuSize.x / 2 > canvasSize.x / 2)
        pos.x = canvasSize.x / 2 - menuSize.x / 2;

    // 下边界检查
    if (pos.y - menuSize.y / 2 < -canvasSize.y / 2)
        pos.y = -canvasSize.y / 2 + menuSize.y / 2;

    menuRT.anchoredPosition = pos;
}
```

        // 淡出效果

# 图11：卡牌效果执行流程

**位置**: 第6章 卡牌系统  
**章节**: 6.2 效果系统  
**类型**: 流程图  
**用途**: 说明卡牌效果的执行机制

## Mermaid 代码

```mermaid
flowchart TD
    Start([卡牌使用]) --> GetCard["获取卡牌数据"]
    GetCard --> CheckCost["检查资源消耗"]
    CheckCost --> HasResource{资源充足?}
    
    HasResource -->|否| NoResource["提示资源不足"]
    NoResource --> End1([使用失败])
    
    HasResource -->|是| DeductCost["扣除资源"]
    DeductCost --> GetEffect["获取卡牌效果"]
    GetEffect --> CheckCondition["检查触发条件"]
    CheckCondition --> ConditionMet{条件满足?}
    
    ConditionMet -->|否| NoCondition["效果不触发"]
    NoCondition --> End2([使用完成])
    
    ConditionMet -->|是| ExecuteEffect["执行效果链"]
    ExecuteEffect --> Loop["遍历每个效果"]
    Loop --> GetEffectType["获取效果类型"]
    
    GetEffectType --> IsDamage{伤害效果?}
    IsDamage -->|是| CalcDamage["计算伤害"]
    CalcDamage --> ApplyDamage["应用伤害"]
    ApplyDamage --> NextEffect
    
    GetEffectType --> IsBuff{Buff效果?}
    IsBuff -->|是| ApplyBuff["应用 Buff"]
    ApplyBuff --> NextEffect
    
    GetEffectType --> IsHeal{治疗效果?}
    IsHeal -->|是| CalcHeal["计算治疗量"]
    CalcHeal --> ApplyHeal["应用治疗"]
    ApplyHeal --> NextEffect
    
    GetEffectType --> IsOther{其他效果?}
    IsOther -->|是| ExecuteOther["执行其他效果"]
    ExecuteOther --> NextEffect
    
    NextEffect["下一个效果"] --> MoreEffect{还有效果?}
    MoreEffect -->|是| Loop
    MoreEffect -->|否| PlayAnim["播放效果动画"]
    PlayAnim --> UpdateUI["更新 UI"]
    UpdateUI --> End3([使用成功])
    
    style Start fill:#90EE90
    style End1 fill:#FFB6C6
    style End2 fill:#FFD700
    style End3 fill:#90EE90
    style ExecuteEffect fill:#FFD700
    style ApplyDamage fill:#FF6347
    style ApplyBuff fill:#87CEEB
    style ApplyHeal fill:#90EE90
```

## 说明

卡牌效果执行的完整流程：

1. **资源检查** - 检查是否有足够的资源使用卡牌
2. **资源扣除** - 扣除使用卡牌所需的资源
3. **条件检查** - 检查卡牌的触发条件是否满足
4. **效果执行** - 依次执行卡牌的所有效果：
   - 伤害效果：计算伤害并应用
   - Buff 效果：应用 Buff 到目标
   - 治疗效果：计算治疗量并应用
   - 其他效果：执行特殊效果
5. **动画播放** - 播放效果相关的动画
6. **UI 更新** - 更新游戏 UI 显示

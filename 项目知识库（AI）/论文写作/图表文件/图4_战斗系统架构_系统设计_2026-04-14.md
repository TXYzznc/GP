# 图4：战斗系统架构

```mermaid
classDiagram
    class CombatSystem {
        +棋子系统
        +Buff系统
        +技能系统
        +AI系统
        +伤害计算系统
    }
    
    class ChessSystem {
        +属性管理
        +状态管理
        +装备系统
        +生命值继承
    }
    
    class BuffSystem {
        +Buff应用
        +Buff移除
        +Buff叠加
        +条件判定
    }
    
    class SkillSystem {
        +技能选择
        +技能执行
        +冷却管理
        +效果链
    }
    
    class AISystem {
        +决策算法
        +评估函数
        +难度调整
        +目标选择
    }
    
    class DamageSystem {
        +伤害计算
        +命中判定
        +暴击判定
        +伤害应用
    }
    
    CombatSystem --> ChessSystem
    CombatSystem --> BuffSystem
    CombatSystem --> SkillSystem
    CombatSystem --> AISystem
    CombatSystem --> DamageSystem
```

## 说明

战斗系统由多个子模块组成：

- **棋子系统**：管理战斗中的角色
- **Buff系统**：管理角色的增益/减益效果
- **技能系统**：管理角色的技能
- **AI系统**：管理敌人的决策
- **伤害计算系统**：计算战斗中的伤害

这些模块通过事件系统相互协作。

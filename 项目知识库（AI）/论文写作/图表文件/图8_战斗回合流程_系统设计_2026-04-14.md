# 图8：战斗回合流程

**位置**: 第4章 战斗系统  
**章节**: 4.2 战斗流程  
**类型**: 流程图  
**用途**: 展示战斗的核心逻辑

## Mermaid 代码

```mermaid
flowchart TD
    Start([回合开始]) --> GetOrder["获取行动顺序"]
    GetOrder --> NextActor["获取下一个行动者"]
    NextActor --> IsPlayer{是玩家?}
    
    IsPlayer -->|是| PlayerAction["等待玩家输入"]
    PlayerAction --> PlayerChoose["玩家选择行动"]
    PlayerChoose --> ValidateAction["验证行动有效性"]
    
    IsPlayer -->|否| AIDecide["AI 决策"]
    AIDecide --> AIChoose["AI 选择行动"]
    AIChoose --> ValidateAction
    
    ValidateAction --> IsValid{行动有效?}
    IsValid -->|否| InvalidAction["提示无效行动"]
    InvalidAction --> NextActor
    
    IsValid -->|是| ExecuteAction["执行行动"]
    ExecuteAction --> ApplyEffect["应用效果"]
    ApplyEffect --> UpdateBuff["更新 Buff"]
    UpdateBuff --> CheckDamage["计算伤害"]
    CheckDamage --> ApplyDamage["应用伤害"]
    ApplyDamage --> CheckDeath{角色死亡?}
    
    CheckDeath -->|是| RemoveEntity["移除角色"]
    CheckDeath -->|否| ContinueBattle["继续战斗"]
    RemoveEntity --> ContinueBattle
    
    ContinueBattle --> CheckEnd{战斗结束?}
    CheckEnd -->|否| NextActor
    CheckEnd -->|是| End([回合结束])
    
    style Start fill:#90EE90
    style End fill:#FFB6C6
    style PlayerAction fill:#87CEEB
    style AIDecide fill:#87CEEB
    style ExecuteAction fill:#FFD700
    style ApplyEffect fill:#FFD700
    style CheckDeath fill:#FF6347
```

## 说明

战斗回合流程的核心步骤：

1. **获取行动顺序** - 根据速度属性确定行动顺序
2. **行动选择** - 玩家输入或 AI 决策选择行动
3. **行动验证** - 检查行动是否有效（资源、条件等）
4. **行动执行** - 执行选定的行动
5. **效果应用** - 应用行动的效果（伤害、Buff 等）
6. **状态更新** - 更新 Buff、检查死亡等
7. **战斗检查** - 检查战斗是否结束
8. **循环** - 继续下一个行动者的回合

# 图11：卡牌效果执行流程

**位置**: 第6章 卡牌系统  
**章节**: 6.2 效果系统  
**类型**: 流程图  
**用途**: 说明卡牌效果的执行机制

## Graphviz DOT 代码

```dot
digraph G {
    rankdir=TB;
    splines=orthogonal;
    nodesep=0.8;
    ranksep=1.0;
    
    // 节点定义
    Start [shape=ellipse, label="卡牌使用", style=filled, fillcolor="#90EE90"];
    GetCard [shape=box, label="获取卡牌数据"];
    CheckCost [shape=box, label="检查资源消耗"];
    HasResource [shape=diamond, label="资源充足?"];
    
    NoResource [shape=box, label="提示资源不足"];
    End1 [shape=ellipse, label="使用失败", style=filled, fillcolor="#FFB6C6"];
    
    DeductCost [shape=box, label="扣除资源"];
    GetEffect [shape=box, label="获取卡牌效果"];
    CheckCondition [shape=box, label="检查触发条件"];
    ConditionMet [shape=diamond, label="条件满足?"];
    
    NoCondition [shape=box, label="效果不触发"];
    End2 [shape=ellipse, label="使用完成", style=filled, fillcolor="#FFD700"];
    
    ExecuteEffect [shape=box, label="执行效果链", style=filled, fillcolor="#FFD700"];
    Loop [shape=box, label="遍历每个效果"];
    GetEffectType [shape=box, label="获取效果类型"];
    
    IsDamage [shape=diamond, label="伤害效果?"];
    CalcDamage [shape=box, label="计算伤害"];
    ApplyDamage [shape=box, label="应用伤害", style=filled, fillcolor="#FF6347"];
    
    IsBuff [shape=diamond, label="Buff效果?"];
    ApplyBuff [shape=box, label="应用 Buff", style=filled, fillcolor="#87CEEB"];
    
    IsHeal [shape=diamond, label="治疗效果?"];
    CalcHeal [shape=box, label="计算治疗量"];
    ApplyHeal [shape=box, label="应用治疗", style=filled, fillcolor="#90EE90"];
    
    IsOther [shape=diamond, label="其他效果?"];
    ExecuteOther [shape=box, label="执行其他效果"];
    
    NextEffect [shape=box, label="下一个效果"];
    MoreEffect [shape=diamond, label="还有效果?"];
    PlayAnim [shape=box, label="播放效果动画"];
    UpdateUI [shape=box, label="更新 UI"];
    End3 [shape=ellipse, label="使用成功", style=filled, fillcolor="#90EE90"];
    
    // 主流程
    Start -> GetCard;
    GetCard -> CheckCost;
    CheckCost -> HasResource;
    
    HasResource -> NoResource [label="否"];
    NoResource -> End1;
    
    HasResource -> DeductCost [label="是"];
    DeductCost -> GetEffect;
    GetEffect -> CheckCondition;
    CheckCondition -> ConditionMet;
    
    ConditionMet -> NoCondition [label="否"];
    NoCondition -> End2;
    
    ConditionMet -> ExecuteEffect [label="是"];
    ExecuteEffect -> Loop;
    Loop -> GetEffectType;
    
    // 效果类型判断
    GetEffectType -> IsDamage;
    IsDamage -> CalcDamage [label="是"];
    CalcDamage -> ApplyDamage;
    ApplyDamage -> NextEffect;
    
    GetEffectType -> IsBuff;
    IsBuff -> ApplyBuff [label="是"];
    ApplyBuff -> NextEffect;
    
    GetEffectType -> IsHeal;
    IsHeal -> CalcHeal [label="是"];
    CalcHeal -> ApplyHeal;
    ApplyHeal -> NextEffect;
    
    GetEffectType -> IsOther;
    IsOther -> ExecuteOther [label="是"];
    ExecuteOther -> NextEffect;
    
    NextEffect -> MoreEffect;
    MoreEffect -> Loop [label="是"];
    MoreEffect -> PlayAnim [label="否"];
    
    PlayAnim -> UpdateUI;
    UpdateUI -> End3;
}
```

## 说明

卡牌效果执行的完整流程：

1. **资源检查** - 检查是否有足够的资源使用卡牌
2. **资源扣除** - 扣除使用卡牌所需的资源
3. **条件检查** - 检查卡牌的触发条件是否满足
4. **效果执行** - 依次执行卡牌的所有效果：
   - 伤害效果：计算伤害并应用
   - Buff 效果：应用 Buff 到目标
   - 治疗效果：计算治疗量并应用
   - 其他效果：执行特殊效果
5. **动画播放** - 播放效果相关的动画
6. **UI 更新** - 更新游戏 UI 显示

## ⚙️ 高级调整

### 调整节点间距

```dot
graph [
    rankdir=TB
    splines=orthogonal
    nodesep=0.8      # 减小=更紧凑, 增大=更宽松
    ranksep=1.0      # 减小=层距离近, 增大=层距离远
]
```

### 调整线条样式

```dot
edge [
    arrowsize=1.0    # 箭头大小
    penwidth=1.2     # 线条粗细
    color="#333333"  # 线条颜色
]
```

### 调整节点颜色

```dot
node [
    style=filled
    fillcolor="#e3f2fd"  # 节点填充色
    color="#1976d2"      # 节点边框色
]
```

## ❓ 常见问题

**Q: 图表渲染失败怎么办？**  
A: 检查 DOT 代码是否有语法错误。在 edotor.net 上会自动高亮错误位置。

**Q: 导出的图片太小/太大？**  
A: 在 edotor.net 页面调整缩放级别，或下载后在图片编辑器中调整大小。

**Q: 能否修改图表颜色？**  
A: 可以。编辑 DOT 代码中的 `fillcolor` 和 `color` 参数。

**Q: splines=orthogonal 是什么？**  
A: 强制所有连接线为 90 度角（正交），提升专业感和可读性。

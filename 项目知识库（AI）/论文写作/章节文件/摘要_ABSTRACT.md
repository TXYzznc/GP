# 摘要

## 中文摘要

本论文以 Clash of Gods 回合制 RPG 游戏项目为实践平台，系统地研究了基于 GameFramework 框架的游戏系统架构设计与实现。论文重点关注如何通过模块化设计、事件驱动架构、配置表驱动等技术手段，构建一个功能完整、性能高效、易于维护和扩展的游戏系统。

论文的主要贡献包括：（1）提出了一套完整的游戏系统架构设计方案，通过事件驱动和模块化设计实现了系统间的低耦合高内聚；（2）详细介绍了战斗系统、物品系统、卡牌系统等核心系统的设计与实现，这些系统代表了现代 RPG 游戏的主要功能；（3）总结了游戏开发中的性能优化经验，包括 CPU 优化、GPU 优化、内存优化等多个方面；（4）实现了完整的热修复支持，允许在游戏运行时进行代码和资源的更新。

论文通过实际项目的验证，证明了所提出的设计方案的有效性和可行性。研究成果对游戏开发者具有重要的参考价值，可以为其他游戏项目的开发提供借鉴。

**关键词**：游戏系统架构、事件驱动、配置表驱动、性能优化、热修复、回合制 RPG

---

## ABSTRACT

This thesis systematically studies the design and implementation of game system architecture based on the GameFramework framework, using the Clash of Gods turn-based RPG game project as a practical platform. The thesis focuses on how to construct a complete, efficient, maintainable, and extensible game system through modular design, event-driven architecture, and configuration-driven approaches.

The main contributions of this thesis include: (1) proposing a comprehensive game system architecture design scheme that achieves low coupling and high cohesion between systems through event-driven and modular design; (2) detailing the design and implementation of core systems such as combat system, item system, and card system, which represent the main functions of modern RPG games; (3) summarizing performance optimization experiences in game development, including CPU optimization, GPU optimization, and memory optimization; (4) implementing complete hot-fix support that allows code and resource updates during game runtime.

Through verification in actual projects, the thesis proves the effectiveness and feasibility of the proposed design schemes. The research results have important reference value for game developers and can provide insights for the development of other game projects.

**Keywords**: Game System Architecture, Event-Driven, Configuration-Driven, Performance Optimization, Hot-Fix, Turn-Based RPG

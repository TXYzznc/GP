## 一、资源体系总览

### 1.1 资源分类

游戏中的资源分为两大类：**全局资源**（局外）和**战斗资源**（局内）

```
资源体系
├── 全局资源（局外）- 永久保存
│   ├── 金币 (Gold)
│   ├── 起源石 (Origin Stone) ⭐ 特殊资源
│   ├── 卡牌碎片 (Card Fragment)
│   └── 科技点 (Tech Point)
│
└── 战斗资源（局内）- 单局临时
    ├── 灵力 (Spirit Power)
    ├── 圣水 (Holy Water) ⭐ 特殊资源
    └── 灵石 (Spirit Stone) ⭐ 可在探索时获得
```

## 三、资源流转关系

### 3.1 全局资源流转

```
战斗胜利
    ↓
获得金币 + 起源石(仅困难/BOSS) + 卡牌碎片
    ↓
基地
    ├→ 金币 → 购买圣水 → 战斗
    ├→ 起源石 → 商店购买高级物品（皮肤、高级卡牌等）
    ├→ 卡牌碎片 → 合成/升级卡牌
    └→ 科技点 → 升级科技树 → 解锁新功能
```

### 3.2 战斗资源流转

```
进入战斗
    ↓
初始灵力: 50 + 科技加成
初始圣水: 携带量（从全局消耗）
初始灵石: 0（探索中可能获得）
    ↓
每回合
    ├→ 灵力 +10（可被Buff加速）
    ├→ 释放策略卡 -灵力
    ├→ 击杀敌人 +局内金币 +经验
    └→ 拾取宝箱 +局内金币 +经验 +灵石(可能)
    ↓
战斗结束
    ├→ 胜利: 局内金币 → 全局金币
    ├→ 胜利: 经验 → 棋子升级
    ├→ 胜利: 灵石 → 保留到下一局（可用于棋子进阶）
    └→ 失败: 损失50%局内金币，灵石保留
```

## 四、策略卡刷新机制

### 4.1 刷新规则

**基础刷新**：
- 初始手牌：3张（可通过科技提升到4-5张）
- 自动刷新：每回合自动补充到满手
- 刷新来源：从卡池中随机抽取

**主动刷新**：
- **条件**：消耗灵力
- **代价**：
  - 第1次刷新：20灵力
  - 第2次刷新：30灵力
  - 第3次刷新：40灵力
  - 第4次及以后：50灵力（上限）
- **效果**：丢弃当前手牌，重新抽取相同数量的卡牌
- **限制**：每回合最多刷新3次

**特殊刷新**：
- **商人刷新**：消耗局内金币，刷新商人的卡牌库
- **强化术士刷新**：消耗HP，获得临时Buff（可能包含卡牌刷新）

### 4.2 刷新策略

| 场景 | 推荐 | 代价 | 收益 |
|------|------|------|------|
| 手牌全是高耗卡 | ✓ | 20灵力 | 获得低耗卡，保持灵力流动 |
| 需要特定卡牌 | ✓ | 30灵力 | 增加获得概率 |
| 灵力充足(>100) | ✓ | 40灵力 | 寻找最优卡牌组合 |
| 灵力不足(<30) | ✗ | 浪费 | 应该等待自动补充 |
| BOSS战前 | ✓ | 50灵力 | 精心准备最强卡组 |

## 五、战斗结果与资源结算

### 5.1 胜利结算

**基础奖励**：
```
金币 = 敌人等级 × 10 + 击杀数 × 5
灵石 = 敌人稀有度 × 2（仅精英/BOSS）
卡牌碎片 = 敌人类型相关（污神类型）
经验 = 敌人等级 × 8 + 难度系数
```

**额外奖励**：
- 宝箱掉落：+局内金币 +经验 +随机物品
- 事件奖励：+金币 +灵石 +卡牌碎片
- 完美胜利（无损）：+20%金币 +10%经验

**示例**：
```
击败普通敌人（等级10）
├→ 金币: 100 + 5 = 105
├→ 经验: 80
└→ 掉落宝箱: +50金币 +20经验

击败精英敌人（等级15）
├→ 金币: 150 + 5 = 155
├→ 灵石: 4
├→ 卡牌碎片: 2
├→ 经验: 120
└→ 掉落宝箱: +100金币 +50经验

击败BOSS（等级20）
├→ 金币: 200 + 5 = 205
├→ 灵石: 8
├→ 卡牌碎片: 5
├→ 经验: 160
└→ 掉落宝箱: +200金币 +100经验
```

### 5.2 失败结算

**资源损失**：
```
全局金币损失 = 50% × 本局获得的金币
灵石损失 = 0（不损失）
卡牌碎片损失 = 0（不损失）
经验损失 = 0（不损失）
```

**保底奖励**：
```
基础保底 = 本局获得金币的10%
棋子存活奖励 = 存活棋子数 × 10金币
```

**示例**：
```
战斗中获得: 300金币 + 50经验
战斗失败
├→ 损失: 150金币（50%）
├→ 保底: 30金币（10%）
├→ 棋子奖励: 20金币（2个棋子存活）
└→ 最终获得: 50金币 + 50经验
```

### 5.3 撤离结算

**主动撤离**（在BOSS前撤离）：
```
获得 = 本局获得资源的100%
无额外惩罚
```

**强制撤离**（棋子全灭）：
```
获得 = 本局获得资源的50%
保底 = 本局获得资源的10%
最终 = 60%
```

## 六、资源珍贵度等级

### 6.1 珍贵度定义

| 等级 | 符号 | 资源 | 获取难度 | 用途重要性 |
|------|------|------|---------|-----------|
| 极高 | ⭐⭐⭐⭐ | 起源石、圣水 | 困难 | 商店/战斗必需 |
| 高 | ⭐⭐⭐ | 灵力、灵石(战斗) | 中等 | 战斗/进阶关键 |
| 中 | ⭐⭐ | 卡牌碎片、科技点 | 容易 | 长期养成 |
| 低 | ⭐ | 金币 | 很容易 | 基础消耗 |

### 6.2 资源价值换算

以**金币**为基准单位（1 = 1金币）：

```
1 起源石 = 500 金币（充值获得，无法用金币购买）
1 灵石(战斗) = 100 金币（战斗中的相对价值）
1 卡牌碎片 = 50 金币
1 科技点 = 200 金币
1 圣水 = 500 金币（战斗中无法用金币购买）
1 灵力 = 2 金币（战斗中的相对价值）
```

## 七、资源管理最佳实践

### 7.1 全局资源管理

**金币**：
- 优先用于购买圣水（战斗必需）
- 其次用于科技升级（长期收益）
- 不要囤积，及时消耗

**起源石**：
- 保留用于商店购买高级物品（皮肤、高级卡牌等）
- 充值获得，极其珍贵
- 不建议用于购买普通物品

**卡牌碎片**：
- 优先合成缺少的角色
- 其次升级常用角色
- 不要分散投资

**科技点**：
- 优先升级战斗相关科技（灵力恢复、手牌数量）
- 其次升级资源获取科技（金币倍数、经验倍数）
- 最后升级便利性科技（背包容量）

### 7.2 战斗资源管理

**灵力**：
- 前期保守使用，积累灵力
- 中期根据局势灵活释放卡牌
- 后期BOSS战前主动刷新，精心准备
- 不要浪费在低价值卡牌上

**圣水**：
- 仅用于关键时刻（BOSS战、困难敌人）
- 不要在普通敌人上浪费
- 每局携带量有限，要精打细算

**灵石**：
- 在探索中优先拾取
- 用于棋子进阶（提升棋子星级）
- 战斗结束后保留，可跨局使用
- 不要浪费，进阶需要大量灵石

## 十、实现建议

### 10.1 代码架构

```csharp
// 全局资源管理
public class GlobalResourceManager
{
    public int Gold { get; set; }
    public int SpiritStone { get; set; }
    public int CardFragment { get; set; }
    public int TechPoint { get; set; }
    
    public void AddResource(ResourceType type, int amount);
    public bool ConsumeResource(ResourceType type, int amount);
}

// 战斗资源管理
public class BattleResourceManager
{
    public int SpiritPower { get; set; }
    public int HolyWater { get; set; }
    public int BattleGold { get; set; }
    public int Experience { get; set; }
    
    public void RefreshCards(int refreshCount);
    public void SettleRewards(BattleResult result);
}

// 资源结算
public class ResourceSettlement
{
    public void SettleVictory(BattleContext context);
    public void SettleDefeat(BattleContext context);
    public void SettleWithdraw(BattleContext context);
}
```

## 十一、平衡建议

### 11.1 难度调整

**简单模式**：
- 灵力初始值：60
- 灵力恢复：+12/回合
- 奖励倍数：×0.8

**普通模式**：
- 灵力初始值：50
- 灵力恢复：+10/回合
- 奖励倍数：×1.0

**困难模式**：
- 灵力初始值：40
- 灵力恢复：+8/回合
- 奖励倍数：×1.5

### 11.2 进度检查点

- **第1周**：玩家应该积累 5000 金币
- **第2周**：玩家应该有 2-3 个角色
- **第4周**：玩家应该能稳定通过困难副本

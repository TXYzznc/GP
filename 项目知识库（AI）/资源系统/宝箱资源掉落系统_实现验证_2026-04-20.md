## 📋 实现清单

### ✅ 配置表修改
- **文件**: `Assets/AAAGame/DataTable/TreasureBoxTable.txt`
- **新增字段** (6个):
  - `CoinsProbability` (double): 金币掉落概率
  - `MinCoins` (int): 最小金币数量
  - `MaxCoins` (int): 最大金币数量
  - `MagicaStoneProbability` (double): 灵石掉落概率
  - `MinMagicaStone` (int): 最小灵石数量（保底）
  - `MaxMagicaStone` (int): 最大灵石数量（上限）

- **配置示例**:
  ```
  ID=1 (可疑的箱子):
    - CoinsProbability: 0.5 (50% 概率)
    - MinCoins: 200, MaxCoins: 500
    - MagicaStoneProbability: 0.2 (20% 概率)
    - MinMagicaStone: 0, MaxMagicaStone: 2
  
  ID=4 (神秘古董箱):
    - CoinsProbability: 1.0 (100% 概率)
    - MinCoins: 10000, MaxCoins: 100000
    - MagicaStoneProbability: 0.8 (80% 概率)
    - MinMagicaStone: 1, MaxMagicaStone: 5
  ```

### ✅ 代码实现
- **文件**: `Assets/AAAGame/Scripts/Game/Interact/Items/TreasureChestInteractable.cs`
- **方法**: `GenerateTreasureItems()`

#### 金币掉落逻辑
```csharp
// 处理金币掉落
if (Random.value < treasureBoxRow.CoinsProbability)
{
    int coins = Random.Range(treasureBoxRow.MinCoins, treasureBoxRow.MaxCoins + 1);
    PlayerAccountDataManager.Instance?.AddGold(coins);
    DebugEx.Success("TreasureChest", $"获得金币: +{coins}");
}
```

**特点**:
- 根据概率判断是否掉落金币
- 金币数量在 [MinCoins, MaxCoins] 范围内随机
- 直接添加到 PlayerAccountDataManager（全局资源）
- 使用 DebugEx 输出成功日志

#### 灵石掉落逻辑（连续抽奖）
```csharp
// 处理灵石掉落（连续抽奖机制）
int totalMagicaStone = treasureBoxRow.MinMagicaStone; // 先加上保底数量

// 连续抽奖：每次都有概率获得灵石，直到抽出不是灵石
while (Random.value < treasureBoxRow.MagicaStoneProbability)
{
    totalMagicaStone++;
    
    // 检查是否达到上限
    if (totalMagicaStone >= treasureBoxRow.MaxMagicaStone)
    {
        totalMagicaStone = treasureBoxRow.MaxMagicaStone;
        break;
    }
}

if (totalMagicaStone > 0)
{
    PlayerRuntimeDataManager.Instance?.AddSpiritStone(totalMagicaStone);
    DebugEx.Success("TreasureChest", $"获得灵石: +{totalMagicaStone}");
}
```

**特点**:
- 保底机制：先获得 MinMagicaStone 数量
- 连续抽奖：每次有 MagicaStoneProbability 概率继续获得灵石
- 上限保护：不会超过 MaxMagicaStone
- 直接添加到 PlayerRuntimeDataManager（战斗资源）
- 使用 DebugEx 输出成功日志

### ✅ 资源管理器支持
- **PlayerAccountDataManager**: 
  - ✅ `AddGold(int amount)` - 添加全局金币
  - ✅ 金币自动保存到存档

- **PlayerRuntimeDataManager**:
  - ✅ `AddSpiritStone(int amount)` - 添加灵石
  - ✅ `ConsumeSpiritStone(int amount)` - 消耗灵石
  - ✅ `SetSpiritStone(int amount)` - 设置灵石
  - ✅ 灵石仅在战斗中存在（不保存到存档）

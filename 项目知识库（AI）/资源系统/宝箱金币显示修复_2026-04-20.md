# 宝箱金币显示修复

**完成时间**: 2026-04-20  
**状态**: ✅ 修复完成，编译通过

---

## 🐛 问题诊断

**现象**:
- 宝箱掉落金币成功（日志显示 +498 金币）
- 但 CurrencyUI 没有更新，画面中没有显示金币图标
- 金币数值没有增加

**根本原因**:
- `GenerateTreasureItems()` 方法中，金币被成功添加到 `PlayerAccountDataManager`
- 但没有触发 `CurrencyUI` 的刷新
- `CurrencyUI` 只在特定事件时刷新（进入局内、战斗结束等）
- 宝箱掉落金币时没有这些事件，所以 UI 没有更新

---

## ✅ 修复方案

### 1. 修改 CurrencyUI.cs
**文件**: `Assets/AAAGame/Scripts/UI/CurrencyUI.cs`

将 `RefreshCurrency()` 方法从 `private` 改为 `public`，允许外部调用：

```csharp
// 修改前
private void RefreshCurrency()

// 修改后
public void RefreshCurrency()
```

### 2. 修改 TreasureChestInteractable.cs
**文件**: `Assets/AAAGame/Scripts/Game/Interact/Items/TreasureChestInteractable.cs`

在 `GenerateTreasureItems()` 方法中，金币掉落后立即刷新 CurrencyUI：

```csharp
// 处理金币掉落
int totalCoins = 0;
if (Random.value < treasureBoxRow.CoinsProbability)
{
    totalCoins = Random.Range(treasureBoxRow.MinCoins, treasureBoxRow.MaxCoins + 1);
    PlayerAccountDataManager.Instance?.AddGold(totalCoins);

    // 刷新 CurrencyUI 显示
    string currencyUIAssetName = GF.UI.GetUIFormAssetName(UIViews.CurrencyUI);
    if (!string.IsNullOrEmpty(currencyUIAssetName))
    {
        var currencyUIForm = GF.UI.GetUIForm(currencyUIAssetName);
        var currencyUI = currencyUIForm?.Logic as CurrencyUI;
        if (currencyUI != null)
        {
            currencyUI.RefreshCurrency();
            DebugEx.Log("TreasureChest", "[GenerateTreasureItems] CurrencyUI 已刷新");
        }
    }
}
```

## 📝 修复细节

### 获取 CurrencyUI 的正确方式

```csharp
// ❌ 错误方式（会编译错误）
var currencyUI = GF.UI.GetUIForm(typeof(CurrencyUI)) as CurrencyUI;

// ✅ 正确方式
string currencyUIAssetName = GF.UI.GetUIFormAssetName(UIViews.CurrencyUI);
if (!string.IsNullOrEmpty(currencyUIAssetName))
{
    var currencyUIForm = GF.UI.GetUIForm(currencyUIAssetName);
    var currencyUI = currencyUIForm?.Logic as CurrencyUI;
}
```

### 关键步骤

1. **获取资源名称**: `GF.UI.GetUIFormAssetName(UIViews.CurrencyUI)`
2. **获取 UIForm**: `GF.UI.GetUIForm(assetName)`
3. **获取逻辑类**: `uiForm?.Logic as CurrencyUI`
4. **调用刷新**: `currencyUI.RefreshCurrency()`

## 🔑 关键知识点

### GameFramework UI 系统

- **UIViews**: 枚举类，定义所有 UI 的标识符
- **GetUIFormAssetName()**: 根据 UIViews 获取 UI 的资源名称
- **GetUIForm()**: 根据资源名称获取已打开的 UIForm 实例
- **Logic**: UIForm 的逻辑类（如 CurrencyUI）

### 为什么需要手动刷新

- CurrencyUI 通过事件系统自动刷新（进入局内、战斗结束等）
- 宝箱掉落金币是特殊场景，不会触发这些事件
- 需要手动调用 `RefreshCurrency()` 来更新 UI

## 使用方式

### 1. 启动 TCP 服务器

使用现有的 TCP 服务器：

```bash
# 方式 1：使用批处理文件（推荐）
cd .kiro/skills/txt-to-xlsx-converter/Tool/Server
start_tcp_server.bat

# 方式 2：直接运行 Python 脚本
cd .kiro/skills/txt-to-xlsx-converter/Tool/Server
python mcp_server_tcp.py
```

服务器将在 `localhost:9000` 启动。

### 2. 转换配置表

使用现有的转换脚本：

```bash
# 方式 1：使用现有的转换脚本（推荐）
cd .kiro/skills/txt-to-xlsx-converter/Tool/Core
python txt_converter.py -f "配置表文件路径"

# 方式 2：交互模式
python txt_converter.py

# 方式 3：批量转换
python txt_converter.py --batch
```

### 3. 转换函数调用

在 SKILL 中直接调用现有的转换函数：

```python
# 导入现有的转换模块
import sys
import os
sys.path.append('.kiro/skills/txt-to-xlsx-converter/Tool/Core')
from txt_converter import convert_txt_file

# 转换单个文件
success = convert_txt_file("MCP工作区/配置表/ItemTable.txt", "MCP工作区/配置表")

if success:
    print("✅ 转换成功")
else:
    print("❌ 转换失败")
```

## 现有工具说明

### TCP 服务器 (mcp_server_tcp.py)
- **功能**: 提供 TCP 接口的转换服务
- **端口**: localhost:9000
- **协议**: JSON-RPC over TCP
- **工具**: convert_txt_to_xlsx, convert_directory_to_xlsx, validate_txt_format

### 转换客户端 (txt_converter.py)
- **功能**: 命令行转换工具，连接 TCP 服务器
- **模式**: 单文件转换、批量转换、交互模式
- **路径处理**: 自动处理相对路径和 MCP 工作区路径

### 核心转换器 (txt_to_xlsx_converter.py)
- **功能**: 实际的 TXT 到 XLSX 转换逻辑
- **格式**: 支持 DataTableProcessor 兼容格式
- **特性**: 自动文本格式化、数组列处理

## 使用示例

### 在 SKILL 中的标准用法

```python
# 1. 启动服务器（如果未运行）
listProcesses()  # 检查是否已有服务器进程

if not server_running:
    controlPwshProcess(
        action="start",
        command="start_tcp_server.bat",
        cwd=".kiro/skills/txt-to-xlsx-converter/Tool/Server"
    )

# 2. 生成 TXT 配置表
txt_content = """#	ItemTable
#	ID		Name	Type	Price
#	int		string	int	float
#	ID编号	备注	物品名称	类型	价格
	1		金币	1	100.0
	2		银币	1	10.0"""

fsWrite("MCP工作区/配置表/ItemTable.txt", txt_content)

# 3. 转换为 XLSX
executePwsh("python .kiro/skills/txt-to-xlsx-converter/Tool/Core/txt_converter.py -f 'MCP工作区/配置表/ItemTable.txt'")
```

### 批量转换示例

```python
# 批量转换所有配置表
executePwsh("python .kiro/skills/txt-to-xlsx-converter/Tool/Core/txt_converter.py --batch")
```

## 注意事项

1. **服务器启动** - 使用前必须先启动 TCP 服务器
2. **端口占用** - 确保 9000 端口未被占用
3. **路径处理** - 转换脚本会自动处理 MCP 工作区路径
4. **编码问题** - 确保 TXT 文件使用 UTF-8 编码
5. **服务器管理** - 可使用 stop_server.bat 停止服务器

## 从 MCP 迁移指南

1. **禁用 MCP 配置** - 在 mcp.json 中设置 `"disabled": true`
2. **启动 TCP 服务器** - 运行 `start_tcp_server.bat`
3. **使用现有脚本** - 直接调用 `txt_converter.py`
4. **测试功能** - 确保转换功能正常工作

这样就完全利用了现有的完善实现，避免了重复开发，同时摆脱了 MCP 配置的复杂性。
